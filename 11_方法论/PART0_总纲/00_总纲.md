# 小说创作五阶段总纲

```
灵感捕捉 → 构思定要素 → 设定与人物 → 大纲结构 → 初稿 → 修改 → 定稿发布
    ↓            ↓            ↓            ↓         ↓       ↓        ↓
  阶段0        阶段1        阶段2        阶段3     阶段4   阶段5    阶段6
```

---

## 阶段0：灵感捕捉与筛选

**目标**：从大量灵感中筛选出"值得写"的核心概念

**产出**：一句话故事（包含主角+目标+阻碍+代价）

**检查清单**：
- [ ] 有强烈冲突（人vs人 / 人vs命运 / 人vs自己）
- [ ] 有"反差"（身份与性格 / 目标与阻碍 / 表面与真相）
- [ ] 有情绪价值（让你自己生气/心疼/兴奋/害怕）
- [ ] 能用一句话讲清楚

---

## 阶段1：构思核心三要素

**目标**：确定故事骨架的三个支柱

**产出**：主角一句话 / 唯一主目标 / 最大阻碍

**公式**：`有缺陷的主角 + 清晰目标 + 层层阻碍 = 故事骨架`

**检查清单**：
- [ ] 主角：身份 + 性格 + 执念/短板，一句话能说清
- [ ] 目标：具体（不是"变强"而是"三个月内赢得冠军"）、贯穿全书
- [ ] 阻碍：反派/自身缺陷/社会环境，三者至少占一

---

## 阶段2：设定与人物

**目标**：建立故事世界的"够用"设定和人物弧光

**产出**：极简世界观 + 主角/关键配角小传

**检查清单**：
- [ ] 世界观：时代 + 核心规则 + 主要场景，1-2页以内
- [ ] 主角：欲望 vs 恐惧冲突 → 故事引擎
- [ ] 主角：有缺陷，故事结束时有成长/领悟
- [ ] 配角 ≤ 5人，每人有动机
- [ ] 反派：有立场、有苦衷，不纯粹邪恶

---

## 阶段3：大纲与结构

**目标**：搭好骨架，防止写崩

**产出**：章纲（每章1-2句）/ 场景表

**推荐结构**：三幕式 / 8节点大纲

**检查清单**：
- [ ] 开端（1-20%）：日常被打破，主角被迫上路
- [ ] 发展（20%-70%）：打怪升级 × 层层阻碍 × 接近低谷
- [ ] 高潮（70%-90%）：终极对决，直面最大阻碍
- [ ] 结局（90%-100%）：目标达成/遗憾，伏笔回收
- [ ] 每章结尾有"翻页钩子"

---

## 阶段4：写初稿

**目标**：先完成，再完美

**心态**：允许自己写"烂"，卡住就跳

**节奏**：日均500-1000字，稳定比爆发更靠谱

**检查清单**：
- [ ] 按大纲顺序写，不东一榔头西一棒子
- [ ] 每日定额，不回读修改，一路写到底
- [ ] 卡住就用 `//TODO:` 标记，跳过继续

---

## 阶段5：修改打磨

**目标**：从"乱麻"到"有型"到"好看"

### 第一轮：宏观结构（动大手术）
- [ ] 主角目标是否清晰？情节是否推进目标？
- [ ] 有无"废章节"？能删则删
- [ ] 人物行为是否符合性格？有无"降智/变性格"？
- [ ] 有无重大逻辑漏洞/时间线错误？

### 第二轮：中观节奏
- [ ] 每章结尾是否有"翻页冲动"？
- [ ] 信息释放节奏是否合理（不太久困惑，不过早剧透）？
- [ ] 删除不影响主线的冗余场景

### 第三轮：微观语言
- [ ] 剔除副词（"愤怒地说"→ 用动作和对话表现愤怒）
- [ ] 精简形容词，多用名词和动词
- [ ] 视角一致性（不中途跳头）
- [ ] 朗读一遍，标记拗口处

---

## 阶段6：定稿发布

**最终校对**：
- [ ] 错别字、标点、格式
- [ ] 人名、地名、专有名词统一
- [ ] 章节编号完整

**发布渠道**：
- 网文平台（起点/晋江）：连载，考验更新频率
- 传统出版：完整书稿+简介+作者简介
- 自出版（Kindle/豆瓣阅读）：完全自主

---

## 新手极简启动路径（7天）

| 天数 | 任务 |
|------|------|
| Day 1 | 写出核心冲动 + 一句话梗概 |
| Day 2 | 设定主角（欲望/缺陷/秘密）+ 反派 |
| Day 3 | 画出三幕式大纲（开端/发展/高潮/结局） |
| Day 4 | 写开篇前2000字（不回头改） |
| Day 5-7 | 每天写1000字，完成第一幕 |
