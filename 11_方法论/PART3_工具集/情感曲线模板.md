## 情感曲线检查

每章扩写后自问：本章情感曲线完整吗？

```
本章情感曲线检查：
- 开头（0-200字）：什么情绪？（悬念/压力/好奇/紧张）
- 发展（200-2000字）：情绪如何变化？（期待↑/紧张↑/疑惑/希望）
- 高潮（收尾前200字）：情绪高点是什么？（爽/哭/震惊/愤怒/感动）
- 结尾：是否留钩子让读者点下一章？

合格情感曲线的最低标准：
├─ 有开头情绪（不是平铺直叙）
├─ 有发展起伏（不是一口气平到底）
└─ 有收尾余韵或钩子（不是「无事发生」式结束）
```

**如果缺少任何一个要素，本章是「无效章节」，需要修改。**

## S6「点击下一章」快速判断

评审时额外问一个问题：「如果我是读者，我会点下一章吗？」

| 判断 | S6加分 | 情形 |
|------|--------|------|
| 会 | +1分 | 章末钩子强，强烈想看后续 |
| 可能会 | 不加分 | 有基本吸引力，但不强烈 |
| 不会 | -1分 | 平收、无聊、节奏拖沓 |

## 信息收获感（每章必问）

- 读者读完本章后，是否感觉「知道了新东西」、「了解了新信息」、「解开了疑惑」？
- 还是感觉「什么都没推进」、「白读了」？
- 每章至少要有**1个信息增量**（新信息/新理解/新悬念解答），让读者有「收获感」
- 信息收获感与记忆点的区别：记忆点侧重「情绪/场面」，信息收获感侧重「认知/情报」

## 记忆点检查（名场面）

读者读完某章后能记住什么？如果什么都不记得，这一章是「无效章节」。

**记忆点类型**：

| 类型 | 例子 | 效果 |
|------|------|------|
| **爆点台词** | "我命由我不由天"、"苏琳，我会回来的" | 读者记住并传播 |
| **震撼场面** | 主角逆转、牺牲、重大发现 | 视觉化记忆 |
| **情感共鸣** | 触动人心的对白/描写 | 情感连接 |
| **悬念设置** | 强烈钩子让读者无法停止 | 追读动力 |

**如果无法回答「读者会记住什么」，本章需要修改。**

## S4情感共鸣扩写指南

基于专家A第一卷评审及P1精修实践发现：S4是最高失分维度，需要系统化补充指导。

### S4失分高频场景与扩写方法

| 失分场景 | 扩写方法 | 示例 |
|---------|---------|------|
| 情感流于口号 | 增加微表情/心理活动/触景生情 | ❌ "他很悲伤" → ✅ "他低着头，指节攥紧泛白" |
| 配角互动单薄 | 增加对话细节/过往回忆/未说出口的牵挂 | 增加"我记得你小时候..."类对话 |
| 情绪爆发缺铺垫 | 增加闪回/前置暗示/生理反应 | 情绪爆发前增加心跳加速、手抖等生理反应 |
| 环境与情绪脱节 | 增加触景生情、环境烘托 | 雨夜送别、暖灯夜谈等具象化场景 |
| 情感表达平铺直叙 | 增加对比反衬、欲言又止 | "话到嘴边又咽了回去"比直接说出来更有力 |

### 新增情感描写模式

#### 1. 内心独白深化类
通过主角对过往的闪回或内心独白，强化情感深度和动机。

| 场景 | 示例 |
|------|------|
| 复仇意志 | "十五年来，他一直在寻找那个杀害父母的凶手...而现在，星辰系统给了他希望。" |
| 父母回忆 | "他还记得那个夜晚——火光、尖叫、以及父亲最后将他推进地下室时说的话..." |
| 心理独白 | "这就是废土。这就是他将要面对的人生。" |

#### 2. 心理挣扎可视化类
将内心挣扎转化为可感知的生理/行为反应。

| 示例 |
|------|
| "但他的手在微微发抖。不只是因为毒液的侵蚀。而是因为他刚才做的事情——杀人。" |
| "他的胃里一阵翻涌。但他不能吐。在这个吃人的世界里，脆弱只是一瞬间的事情。" |
| "他的手指握紧了匕首的柄，指节泛白。" |

#### 3. 欲言又止类
通过沉默或中断的对话，增加情感张力。

| 示例 |
|------|
| "她没有回答，只是将手攥得更紧。" |
| "苏琳微微一笑。'我知道。'她说，'所以我会一直等他。'" |
| "她的声音很轻，带着一丝难以察觉的忧虑。'小心雷城。'" |

### 情感共鸣检查清单

扩写后自检S4：
- [ ] 本章是否有具象化的情绪描写（非口号式）？
- [ ] 主角/配角的情感是否有细节支撑（微表情、动作、心理）？
- [ ] 情感发展是否有合理的触发事件或回忆？
- [ ] 环境描写是否服务于情绪烘托？
- [ ] 读者是否能代入这个情感场景？
- [ ] 是否有内心独白或过往回忆深化情感动机？
- [ ] 是否有可感知的生理/行为反应展示内心挣扎？

### 情感描写的「三感」原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **画面感** | 让读者能「看见」这个场景 | 描写动作、表情、环境细节 |
| **带入感** | 让读者能「感受」这个情绪 | 触景生情、生理反应、心理活动 |
| **余韵感** | 让读者读完后能「回味」 | 欲言又止、留白、对比反衬 |
