# 情感色彩曲线工具

> **版本**：v1.0
> **来源**：动画制作流程的Color Script（色彩脚本）
> **日期**：2026-05-19

---

## 一、工具说明

动画制作中有一个重要工具叫"色彩脚本"（Color Script）——把关键情节浓缩成一连串色块，用色彩来讲述故事的情绪变化。

小说创作可以借鉴这个工具：**用色彩标注每章的核心情绪，可视化整个作品的情感节奏**。

---

## 二、使用时机

- **规划期**：为每卷制作情感色彩曲线，用于规划阶段的节奏设计
- **修改期**：检查已完成的章节是否遵循情感色彩曲线，发现问题

---

## 三、操作步骤

### Step 1：列出章节的核心情绪

对于每一章，用一个词标注核心情绪：

| 情绪类别 | 色彩词示例 |
|---|---|
| 压抑/绝望 | 深蓝、灰黑 |
| 迷茫/低沉 | 蓝灰、雾灰 |
| 希望/温暖 | 暖黄、浅金 |
| 危机/紧张 | 橙红、深红 |
| 高潮/爆发 | 金色、火红 |
| 平缓/日常 | 浅绿、米白 |

### Step 2：绘制情感色彩曲线

用文字绘制从第1章到最后1章的色彩轨迹：

```
卷一·情感色彩曲线：

第1章  深蓝（压抑）  ————————————————
第2章  蓝灰（迷茫）     —
第3章  暖黄（希望）        ———
第5章  橙红（危机）              —————
第6章  深红（高潮）                 ———
第8章  灰（低谷）                      ————
第10章 金色（升华）                        ————————

规律：深蓝 → 暖黄（抑到扬）
      橙红 → 灰（爆发后必有小憩）
      灰 → 金（绝望后必有人性升华）
```

### Step 3：检查问题

| 问题 | 判断标准 | 解决方案 |
|---|---|---|
| 连续三章以上同色 | 情绪单调，读者疲劳 | 在中间插入"呼吸章节"（日常/温情/幽默） |
| 缺少对比色 | 缺少张力，剧情平 | 在色彩低谷后插入一个小高潮（再跌） |
| 色彩突变 | 情绪跳跃太快 | 用过渡章节逐步转换 |
| 缺少高潮色 | 全程平缓，没有巅峰 | 找出最重要的3-5个节点，用最强色彩标注 |

## 五、示例

### 示例：玄幻升级文（30章/卷）

```
卷一·情感色彩曲线：

第1章   深蓝（穿越+压制）        ——————————
第3章   灰蓝（困境）                —
第5章   暖黄（奇遇+希望）              ———
第8章   橙红（第一次战斗）                  —————
第10章  深红（小高潮：越级击败）                 ————
第12章  蓝灰（危机：强敌出现）                      —
第15章  深蓝（低谷：师父受伤）                        ———————
第18章  暖黄（转机：发现规则漏洞）                        —
第20章  橙红（决战开始）                              —————
第22章  深红（绝境）                                    ———
第25章  金色（爆发：突破）                               ———
第28章  深蓝（代价）                                         —
第30章  金色（升华+收尾）                                      ————————

规律总结：
1. 每10章一个大节奏：5章一个小抑扬，10章一个大抑扬
2. 高潮后必有小憩（打斗后接日常）
3. 绝望后必有人性温暖（低谷中有温情配角）
4. 金色高潮只出现在最重要的节点（开头奇遇、结尾升华）

问题：
- 第12-15章连续灰蓝色，情绪偏低，可能让读者疲劳 → 考虑在第13章增加一个小爽点
```

## 六、与评审维度的对应

| 维度 | 情感色彩曲线相关检查点 |
|---|---|
| S4 情感共鸣 | 情绪曲线是否有效，读者是否被带动 |
| S5 节奏控制 | 情绪色彩的抑扬是否分明，是否有"呼吸空间" |
| S6 可读性 | 连续同色章节是否导致读者疲劳 |
