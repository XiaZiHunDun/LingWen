# 历史穿越模板

> **版本**：v1.0
> **适用类型**：历史穿越 / 宫斗宅斗 / 武侠江湖
> **日期**：2026-05-20

---

## 核心驱动链

```
穿越 → 适应 → 崛起 → 危机 → 抉择 → 改变历史 / 被历史改变
```

历史穿越核心是**认知落差**——现代人带着知识/价值观进入古代，与古代规则碰撞。

---

## 穿越设定模板

### 穿越方式

```
【魂穿型】（灵魂进入原住民身体）
- 优点：身份掩护，无需解释来历
- 缺点：需要继承原主记忆/性格/社会关系
- 关键挑战：模仿原主行为，不露馅

【身穿型】（肉体穿越）
- 优点：保持完整能力和知识
- 缺点：身份完全无法解释，需要伪造身份
- 关键挑战：如何在古代解释自己的来历

【原身型】（回到自己年轻时的身体）
- 优点：完全知道自己的身份和社会关系
- 缺点：知道未来的能力可能改变历史，也可能被未来记忆压垮
- 关键挑战：如何在已知历史轨迹下做出不同选择
```

### 穿越时空背景

```
【常见朝代设定】

秦朝：
- 优势知识：郡县制、统一度量衡、秦弩技术
- 风险提示：严刑峻法、焚书坑儒风险

汉朝：
- 优势知识：丝绸之路、张骞出塞、造纸术改进
- 风险提示：外戚干政、七国之乱

三国：
- 优势知识：三国志内容、名人典故
- 风险提示：军阀混战、人才争夺激烈

唐朝：
- 优势知识：科举制雏形、贞观之治、开元盛世
- 风险提示：安史之乱、藩镇割据

宋朝：
- 优势知识：活字印刷、指南针、火药改良
- 风险提示：重文轻武、靖康之耻

明朝：
- 优势知识：郑和下西洋、资本主义萌芽
- 风险提示：锦衣卫东厂、文字狱
```

## 人物弧光模板

### 穿越者成长路径

```
【阶段一：生存期】
- 目标：活下来，找到立足之地
- 挑战：适应古代生活、理解古代规则
- 关键：隐藏现代知识，建立基本身份

【阶段二：发展期】
- 目标：在古代社会中获得地位/财富/权力
- 挑战：运用现代知识但不能太超前（避免被视为妖异）
- 关键：找到古代伙伴/盟友，获得信任

【阶段三：改变期】
- 目标：改变某个历史事件/社会问题
- 挑战：历史修正力、既得利益者反扑
- 关键：改变 vs 顺应的终极抉择

【阶段四：结局】
- 留在古代：真正融入，改变历史
- 返回现代：带着经历成长
- 超越时空：成为传说/神话的一部分
```

### 配角设定

```
【古代原住民类型】

忠臣型：
- 被穿越者的现代思想吸引
- 成为穿越者的古代伙伴
- 冲突点：价值观差异

反派型：
- 传统规则维护者
- 第一个发现穿越者的异常
- 冲突点：利益冲突/立场对立

贵人型：
- 给予穿越者第一桶金/身份掩护
- 通常有自己目的（培养人才/拉拢人才）
- 冲突点：最终可能站在对立面

红颜/蓝颜型：
- 古代的爱慕对象
- 可能知道穿越者秘密（同类穿越者）
- 冲突点：感情vs理性/身份差异
```

## 宫斗宅斗模板

```
【宫斗核心公式】
宠爱 → 争嫡 → 失宠 → 翻盘 → 复仇/上位

【常见宫斗情节】

晋升路线：
1. 秀女入宫 → 最底层宫女
2. 获得赏识 → 升级为嫔妃
3. 获得宠爱 → 威胁其他嫔妃
4. 卷入阴谋 → 被陷害/反击
5. 翻盘获胜 → 成为皇后/太后

常用手段：
- 借刀杀人：利用其他嫔妃斗倒对手
- 苦肉计：让自己受害博取同情
- 离间计：破坏对手同盟
- 情报战：收买宫女/太监获取信息
- 怀柔政策：关键时刻施恩争取支持

【宅斗核心公式】
新妇入门 → 站稳脚跟 → 获得管家权 → 斗倒恶婆婆/小妾 → 掌家

宅斗要点：
- 古代宗族关系复杂
- 婆婆是天然对手（小妾可能成为盟友）
- 子女是筹码（生儿子很重要）
- 嫁妆是经济基础
```

## 权谋模板

```
【权谋思维】

基本原则：
1. 信息为王：知道别人不知道的信息
2. 利益分析：每个人都有自己的利益
3. 联盟变化：没有永远的朋友，只有永远的利益
4. 留有余地：做事不做绝，留后路

【常用权谋手段】

情报收集：
- 安插眼线/收买下人
- 截获书信/偷听对话
- 试探话语（看对方反应）

借势：
- 借贵族之势压人
- 借舆论之势造势
- 借规则之势（法律/道德/礼教）

制衡：
- 让对手互相斗
- 坐山观虎斗
- 最后出面收割

破局：
- 引入外部势力打破平衡
- 创造新规则改变游戏
- 釜底抽薪（断其根本）
```

## 历史细节模板

### 古代日常生活

```
【称呼礼仪】
- 皇室：陛下/殿下/公主/皇子
- 官员：大人/老爷/相公
- 平民：相公/娘子/员外
- 自称：臣/妾身/在下/小人

【礼节规矩】
- 跪拜礼：跪几拜、叩首几下
- 饮食礼：座次排序、谁先动筷
- 见面礼：拱手/万福/作揖
- 婚丧礼：六礼/守孝期

【日常用语】
- 时间：时辰（一个时辰=2小时）/ 更（夜间）/ 节气
- 货币：两/钱/文
- 距离：里（一里=500米）
- 称呼：爹/娘/老爷/夫人/少爷/小姐
```

### 历史事件融入

```
【如何利用历史事件】

方式一：推动历史
- 提前引入历史发明的技术
- 帮助历史人物度过关键节点
- 改变某个历史事件的时机

方式二：借势而为
- 利用已知的历史结果做决策
- 在历史大事件中找到自己的位置
- 通过历史事件获得利益

方式三：改变历史
- 某个历史事件被改变（需要充分铺垫）
- 蝴蝶效应导致新结果
- 最终发现历史依然"正义"

方式四：历史循环
- 发现穿越不是第一次
- 前人穿越者留下了痕迹
- 历史在重复，给主角选择的机会
```

## 章节钩子模板

```
【穿越文特有钩子】

身份危机：
- 她突然停下手中的动作，发现自己差点说出现代词汇……
- "你真的还是你吗？"他看着她，眼中闪过一丝疑虑……

古代规则冲击：
- 她忘了古代女子不能外出，门口已经被两个大汉拦住……
- 按照规矩，她必须嫁给那个人，但她心里只有另一个选择……

信息优势：
- 她知道这个人十年后会成为宰相，但现在他只是一个穷书生……
- 凭借历史知识，她知道自己正站在一个改变天下格局的时刻……

利益博弈：
- 婆婆突然提出一个她无法拒绝的条件，代价是……
- 管家送来一封信，信上的内容让她脸色大变……

时代冲突：
- 她想做一件事，但所有人都说她疯了……
- "你不懂，"她听到自己说，"这个世界的规则会变的。"……
```

- [ ] 历史背景考证准确（官职/礼仪/称谓/服饰）
- [ ] 现代人穿越后的心理转变自然

- [ ] 穿越设定有逻辑（穿越机制/时空规则）
- [ ] 权谋情节有层次（不只用一计到底）
- [ ] 主角金手指有边界（不是万能）
- [ ] 历史蝴蝶效应有铺垫（不是突然改变）
- [ ] 古代女性的处境有体现（不是开挂无边）
