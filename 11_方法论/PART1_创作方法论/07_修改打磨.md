# 阶段5：修改打磨

> **核心观点**：好小说是改出来的。初稿完成后，先放1-2周不看，以陌生读者眼光审视。

---

## 修改顺序：先大后小

```
宏观结构（整书/卷）→ 中观节奏（章）→ 微观语言（句/词）
```

**绝对不要**：从第一句开始逐字改，改完全文发现结构有问题。

---

## 第一轮：宏观结构修改（动大手术）

**关注**："故事好不好看、合不合理"，不管文笔。

### 检查维度

| 维度 | 检查问题 |
|------|----------|
| **结构** | 节奏是否太慢？有无废章节？高潮够不够有力？ |
| **人物** | 动机合理吗？成长弧光清晰吗？对话是否符合人设？ |
| **情节** | 逻辑自洽吗？设定有无矛盾？ |
| **视角** | 有无中途跳头？视角是否统一？ |

### 工具：反向大纲

写完初稿后，每章自己总结："这章在干嘛"，再和原大纲对比，看跑偏多少。

### 宏观修改可能需要的操作

- 整章**删除**（不影响主线就删）
- 整章**合并**（两章内容合并一章）
- 整章**重写**（某章完全跑偏）
- **调整顺序**（情节顺序不合理）

### 检查清单

- [ ] 主角目标是否始终清晰？
- [ ] 开头是否勾人？（前300字决定生死）
- [ ] 中段是否有拖沓？（删掉所有"解释性段落"）
- [ ] 结尾是否有力？（情感/主题闭环了吗？）
- [ ] 每个角色是否有独立的声音？
- [ ] 有无重大逻辑漏洞/时间线错误？

---

## 第二轮：中观节奏修改

**关注**：每章是否让读者想翻下一页。

### 检查维度

| 维度 | 检查问题 |
|------|----------|
| **翻页钩子** | 每章结尾是否有"然后呢"的冲动？ |
| **冗余场景** | 删掉某章会影响主线吗？不影响就删 |
| **信息节奏** | 读者困惑太久？还是过早剧透？ |
| **情绪曲线** | 全程一个情绪？还是起伏有致？ |

### 节奏问题常见表现

| 问题 | 现象 | 解决 |
|------|------|------|
| 太拖 | 连续3章没有实质性情节推进 | 删掉解释性段落，推进冲突 |
| 太赶 | 关键情节没展开就过了 | 加内心/对话/场景细节 |
| 平 | 全程一个节奏，无张弛 | 加入小冲突/小挫折 |
| 乱 | 场景之间跳跃无逻辑 | 加过渡场景/章节 |

### 检查清单

- [ ] 每章结尾是否有"翻页钩子"？
- [ ] 有无可以删除的冗余场景？
- [ ] 情绪节奏是否起伏有致（不是全程紧张或全程平淡）？

---

## 第三轮：微观语言修改

**关注**：文字表达精准、有韵味、可朗读。

### 检查维度

| 维度 | 检查问题 |
|------|----------|
| **副词** | 能否用动作和对话本身表现情绪？（"愤怒地说"→ 用动作） |
| **形容词** | 是否过于堆砌？精简，多用名词和动词 |
| **重复** | 同一个词/表达是否重复出现过多？ |
| **拗口** | 能否朗读出来？拗口处标记修改 |
| **对话** | 是否符合人物身份？是否推进剧情而非闲聊？ |

### 微观修改技巧

```
❌ "他慢慢地说，声音低沉而悲伤"
✅ "他没说话，只是把杯子推到一边，指尖在桌面画着圈。"

❌ "她很害怕，心跳加速"
✅ "她的手指在门把手上滑了三次才握紧。"

❌ "他走进房间，看到了一切——混乱、破坏、还有血。"
✅ "他推开门。地上有脚印。墙上有掌印。角落里，一动不动的东西他不想认。"
```

### 检查清单

- [ ] 剔除副词（"X地说"类全部检查）
- [ ] 精简形容词，多用具体名词和动词
- [ ] 朗读一遍，标记读起来拗口的地方
- [ ] 对话是否像"人话"而非书面语？
- [ ] 心理描写是否过多"我觉得..."？

---

## 第四轮：外部反馈

### 找试读读者

- **选人**：2-3位目标读者（不要无条件夸你的人）
- **问问题**：
  1. 哪里让你感到无聊/想跳过？
  2. 哪个角色你最喜欢/最不喜欢？为什么？
  3. 结尾让你感到满足、意外，还是困惑？

### 反馈处理原则

- 共性问题 → 必须改
- 个人偏好 → 坚持自己的判断
- 多人提出相同问题 → 优先改

---

## 修改期禁止

1. ❌ 大规模重写（结构问题修改，微观问题不要推倒重来）
2. ❌ 改细节改到忘了整体（先整体后细节）
3. ❌ 永远改不完（学会在某一版停下来）
