# 核心驱动链设计

> **版本**：v1.0
> **来源**：借鉴游戏制作"核心循环（Core Loop）"方法论
> **日期**：2026-05-19
> **状态**：新增

---

## 核心理念

**核心驱动链**是小说的"引擎"，类似于游戏中的 Core Loop。

它是驱动读者持续阅读的核心循环——每间隔一定篇幅，读者会获得一次明确的"满足感"。

> **核心原则**：如果核心驱动链不成立，再好的文笔内核依然是空洞的。

---

## 一、概念定义

### 1.1 什么是核心驱动链

**游戏中的 Core Loop**：
> 玩家每分钟在做什么？例如《暗黑破坏神》：打怪 → 掉宝 → 角色变强 → 打更强的怪。

**小说中的核心驱动链**：
> 读者每读一定篇幅（每章/每三章/每卷）会获得什么类型的满足感？

### 1.2 核心驱动链的构成

```
[类型] + [频率] + [满足感] = 核心驱动链

示例：
- 爽文类：每3章一次小爽、每10章一次大爽
- 悬疑类：每5章一次线索揭示、每卷一次核心反转
- 言情类：每3章一次情感拉扯、每卷一次关系突破
```

## 二、设计模板

### 2.1 标准模板

```
# [小说名] 核心驱动链设计

## 小说类型
[类型：玄幻/都市/悬疑/言情/科幻等]

## 核心驱动链
[一句话描述驱动读者持续阅读的核心模式]

## 驱动链分解

### 每[X章]的"小满足"
- [类型]满足感1：[描述]
- [类型]满足感2：[描述]
- ...

### 每[X章]的"大满足"
- [类型]满足感1：[描述]
- [类型]满足感2：[描述]
- ...

## 驱动链验证
- [ ] 驱动链是否具有递增性？（越来越强/越来越复杂）
- [ ] 每个满足感是否清晰可描述？
- [ ] 驱动链能否用一句话向读者传达？
```

### 2.2 驱动链示例

**示例1：《斗破苍穹》风格爽文**
```
小说类型：玄幻升级爽文

核心驱动链：
屈辱 → 蓄能 → 越级反杀 → 获得尊重/资源 → 更强对手出现 → 循环

每3章一次小满足：
- 被嘲讽后打脸对方
- 修炼突破获得新能力
- 获得稀有宝物/丹药

每10章一次大满足：
- 跨级战胜强敌
- 震惊全场
- 收获重要伙伴/资源
```

**示例2：《诡秘之主》风格悬疑**
```
小说类型：悬疑探索

核心驱动链：
探索 → 发现异常 → 遭遇危险 → 智慧应对 → 揭开谜底 → 新谜团浮现

每5章一次小满足：
- 发现关键线索
- 识破敌人伪装
- 获得新情报

每卷一次大满足：
- 核心谜题真相揭示
- 反派真实身份揭露
- 主角能力/地位跃升
```

**示例3：《何以笙箫默》风格言情**
```
小说类型：都市言情

核心驱动链：
错过 → 重逢 → 误会 → 拉扯 → 真相 → 坚守 → 圆满

每3章一次情感拉扯：
- 双方冷战/争吵
- 暧昧试探
- 回忆触发

每卷一次关系突破：
- 表白/接受表白
- 确立关系
- 重大误解解开
```

## 三、设计检查点

在确定核心驱动链前，必须回答以下问题：

| 问题 | 说明 |
|------|------|
| **类型定位** | 你的小说是哪种类型？读者对这类小说的核心期待是什么？ |
| **核心期待** | 读者读你的小说，最想获得什么？（爽感、悲伤感、好奇感、甜蜜感） |
| **满足频率** | 多长时间给读者一次"爽"？（每章/每三章/每卷） |
| **满足类型** | 这种满足感是什么形式的？（打脸、揭秘、发糖、解谜） |
| **递增性** | 随着故事发展，满足感是否越来越强/越来越复杂？ |
| **差异化** | 你的核心驱动链与同类型小说的差异点在哪里？ |

## 四、常见问题

### 4.1 核心驱动链模糊

**症状**：作者自己说不清"读者每读一章能得到什么"

**解决**：回到类型定位，明确读者期待

### 4.2 驱动链断裂

**症状**：前10章有明确的"爽点"，后10章变成平铺直叙

**解决**：检查中段是否有足够的阻碍和递进

### 4.3 满足感重复

**症状**：每章都是同样的"打脸"，缺乏变化

**解决**：设计多样化的满足感类型，避免重复

### 4.4 递增性不足

**症状**：主角一直赢，缺乏挑战

**解决**：确保每个"满足"后紧跟更大的挑战

## 五、与创作流程的接口

### 5.1 位置

核心驱动链设计位于**构思期末尾、规划期开头**。

### 5.2 验收标准

- [ ] 核心驱动链能用一句话描述
- [ ] 每X章的小满足类型清晰
- [ ] 每X卷的大满足类型清晰
- [ ] 驱动链具有递增性
- [ ] 与同类型小说有差异点

### 5.3 与其他节点的关联

```
核心冲动 → 类型定位 → 一句话梗概 → [核心驱动链设计] → 三幕式大纲
                                                    ↓
                                             情节骨架验证
```
