# 07_汇总仓库 · 正文汇总中心

> **版本**: v1.0
> **更新 (2026-05-27)**：补充操作说明，完善CLAUDE.md关联

## 职责

**07_汇总仓库**是正文汇总中心，负责：
- 将分章正文（03_内容仓库/04_正文/）合并为完整的卷正文/全文正文
- 生成各阶段的阶段汇总、卷汇总、全文汇总
- 汇总后的文件用于发布存档（08_已发布）

## 目录结构

```

├── run_summary.sh          # ⚠️ 主要操作入口（正文合并）
```

## 关键区分

| 仓库 | 职责 | 操作命令 |
|------|------|---------|
| **03_内容仓库** | 内容存储 + 元数据索引 | `update_index.py --all`（只更新index.json） |
| **07_汇总仓库** | 正文汇总（合并成完整文档） | `run_summary.sh merge full`（合并正文） |

## 操作命令

### 合并正文（主要操作）

```bash
# 进入汇总仓库
cd 07_汇总仓库

# 全文合并（常用）
./run_summary.sh merge full

# 卷合并
./run_summary.sh merge volume 卷1
./run_summary.sh merge volume 卷2
./run_summary.sh merge volume 卷3

# 阶段合并
./run_summary.sh merge stage 卷1_阶段1
./run_summary.sh merge stage 卷1_阶段2
# ...
```

### 验证汇总完整性

```bash
./run_summary.sh verify
```

### 列出已存在汇总

```bash
./run_summary.sh list
```

### 完整工作流

```
1. ./run_summary.sh merge stage 卷1_阶段1   # 合并阶段正文
2. ./run_summary.sh merge stage 卷1_阶段2   # ...
3. ...
4. ./run_summary.sh merge volume 卷1        # 合并卷正文
5. ./run_summary.sh merge volume 卷2        # ...
6. ./run_summary.sh merge volume 卷3        # ...
7. ./run_summary.sh merge full              # 合并全文正文
8. ./run_summary.sh verify                  # 验证完整性
```
