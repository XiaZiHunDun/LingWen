# 第7章 最低点

便利店的门铃响了。

林栀没有回头。

屏幕亮度已经调到最低，她的眼睛还是被那片空白刺得生疼。

"槐安巷37号"——零条结果。
"槐安巷 事故"——零条。
"槐安 地下坍塌 2010"——零条。

她把搜索词又改了几种组合，指尖在屏幕上敲得有些急。冷气开得很足，后颈却渗出一层细汗。

"姑娘，要点什么不？"

"不用。"

她头也没抬，声音有点哑。收银台后面的人把杂志翻了一页，没再说话。

她继续搜。"槐安 区域 旧照片"终于有结果——摄影论坛2015年帖，二十多张老城巷照，没有槐安巷。最后一张标注"槐安路与建宁街交叉口"，路牌放大再放大，只辨出最后一个字：**巷**。

她切到地图：建宁街与槐安路交叉处是"槐安居委会"小院子，北面围墙标 **槐安路87号**；87号再往北半百米，37号理应也在——地图上却是一片空白，点上去只有"暂无该位置信息"。87在、37不在，相隔不到五十米，一个地址活着，一个消失得干干净净。

"林栀？"收银台的人叫她，眼神像在确认什么，"坐好久了？""等雨停。"

她搜 **槐安路87号**——一家已注销的街道办事处，2011年4月，与8月17塌方、围挡拖大半年后对得上。再换关键词仍是零，直到 **"槐安路 老照片 巷子"** 刷出2018年求助帖：小时候听说有槐安巷，怎么都找不到？三条回复，前两条否认，第三条：**"别查了，有些地方查不出来。"** ID **周姐不姓周**，2015年注册，仅此一回。

再搜 **"周姐 槐安 论坛"**，命中2017年帖"老槐安路的朋友们进来聊聊"：1987年黑白照，路牌清晰写着 **槐安巷**，拍摄位置 **槐安路37号**——"规划红线穿过，这条巷子拆得最快。"帖下无回复。林栀保存照片，把右下角人影放到最大，辨认出旁边一行淡字：**周姐，1987年。**

线头全拧到了一起。

便利店的门铃又响了一次。"姑娘。""嗯？""雨停了。"

林栀看向窗外，路面积水映着灯光。她把手机装进口袋站起来，道了声谢，推门出去。天还是阴的，云层低压。

她再看手机——进店时是21:30，此刻 **23:47**。两个多小时像从中间挖走，她不记得搜了什么，只剩 **周姐不姓周** 和 **周姐，1987年** 缠在脑子里。

檐外风灌进来，冷得她起鸡皮疙瘩。公交站灯箱亮着却没有车，她向公交站走去，水花溅上裤脚，没低头看。
