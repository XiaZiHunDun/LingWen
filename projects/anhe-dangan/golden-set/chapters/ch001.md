# 暗河档案·第1章 丰水

水位计的第三频道在凌晨两点十七分发出了第一声杂音。护林摩托停在洞口外侧松软泥地里，引擎余温让排气管发出几声闷响，像某种大型动物在远处翻身——沈渡熄了火却没立刻下车，只盯着手腕接收器：第三频道波形本该平直，丰水期前夜地下水位通常不会过激波动，那条标定为「汊-07」的支脉早在三年前就从监测网络除名。

波形抖了一下，不是仪器故障那种抖；十二年暗河观测教会他分辨水流拍击岩壁、溶腔塌方挤压气柱，以及更接近呼吸的节律。增益旋钮往右拧半格，四个波峰便清晰起来，间隔均匀，振幅0.3毫米，不足两秒后归于死寂——这不是水。

他把耳机塞进耳朵，推开车门。山里的夜风带着松脂和湿泥的气味，洞口方向传来值守员老覃收音机里咿咿呀呀的越剧唱段，音量开得不大，混在虫鸣里几乎听不真切。

"沈工？"老覃从帆布躺椅里欠起身，手电筒的光柱扫过沈渡的胶鞋，又落到他脸上，"这个点儿上来……有任务？"

"看看水尺。"

老覃噢了一声，没多嘴。每年丰水期前夜他都要来一趟，算是老规矩了。三年前立的规矩——立规矩的人早去了省城，听说孩子都会跑了。沈渡从老覃身边过时，余光扫到洞口的木牌还是那块，漆面剥落得厉害，红底白字写着"丰水期 严禁入内"，落款处"林湄"两个字的笔画被雨水冲得只剩浅浅的凹痕。

他没停步。

值守棚里的铁柜没上锁；头灯调到最低亮度，光束扫过几层隔板——第三层，他记得清楚，三年前林湄撤得仓促，这个本子应该还在原处。一本空白观测本，A5灰蓝硬壳，右下角贴着「汊-07 2019.04-2021.06」手写标签。

观测本抽出手感沉得不似空白——沈渡翻开第一页，一路翻到第三十七页，右下角才现出一小片淡黄色水渍，形状不规则，像凝结的水珠被时间封存在纸纤维里；指腹抚过纸面微凉，带着几乎察觉不到的潮润：钟乳水。

林湄只在记录钟乳水渗漏量时才会用这种纸质观测本——市面上买不到，她自己裁的，边角没对齐，厚度也因此参差不齐。往后翻，空白一页接一页，直到封底内侧夹着一张对折宣纸：溶缝剖面铅笔素描，线条干净，标注详尽，但有几处被刻意涂成黑块，看不清原本画的是什么。

素描折回封底，整本观测本收入防水袋。

耳机里波形又动了：这一次不是四个波峰，而是三个，间隔略长，振幅却比刚才高了将近一倍；接收器屏幕上的坐标定位在自动刷新，每刷新一次，数值都往同一方向偏移——西南，斜下，深度六十一点四米，正是汊-07，那个在监测网络中「不存在」的位置。

头灯熄灭，走出值守棚时老覃仍坐在躺椅里，收音机已换台播深夜情感热线，女主持人用绵软嗓音劝解一个声称被前女友下了蛊的男听众。

"老覃。"

"嗯？"

"今晚上来过别人没有？"

老覃想了想："没有。就我一个人守到天亮。"收音机音量调小，"出什么事了？"

"没事。"摩托发动，引擎轰鸣盖过下一句——问话的理由连沈渡自己也说不清，也许只想确认汊-07的坐标偏移不是设备延迟的鬼影，也许只想确认那个节律不是从洞口附近传过去的。往山下开时，后视镜里洞口木牌在余光里晃了晃，「丰水期 严禁入内」被甩在身后，越来越小；耳机里第四声波形响了，五个波峰间隔更短、振幅更高，坐标锁定在六十一点四米深的黑暗里，纹丝不动，像是有什么东西屏住呼吸，等着他。
