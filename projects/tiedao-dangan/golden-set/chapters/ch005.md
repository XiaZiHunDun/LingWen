# 第5章 断粮

纪川是被渴醒的——不是喉咙发干那种渴，而是舌根黏在上颚、连口水都咽不下去的那种。

睁眼时头灯还亮着，电量指示灯在视线最下方一明一灭闪着红光，不知闪了多久；下意识去摸水袋，指尖触到一层冰凉的塑料膜——接缝处长时间受压，胶合剂被挤开细缝，水袋瘪下去大半，剩水全积在最底部，沉甸甸坠着。

袋口拧紧，试着往嘴里挤了两口，塑料味凉意直酸牙根——渴意不算厉害，见底却是事实。手机没信号，意料之中，竖井深处岩层厚，卫星和蜂窝都进不来；本地录音和数据卡不受影响。卫星电话摸到手里，外壳磨得发亮——上一任工务复核组留在侧室的旧货，拨了两次，信号灯都只闪两下便熄。

电话搁下，靠着坑道壁坐了一会儿。前方约三米被道砟和落石封死，堆高两米出头，石块从缝隙间往下漏，落在头盔上，发出细碎的声响——方才试着挖过，挖不动，顶板还在松动，停手时半个肩膀已埋在碎石里；那时没慌，现在也没慌，只是渴。

水袋重新别回腰上，开始往上回想：上一次喝水大概四个小时前，主坑道尽头发现一道裂缝，冷风往外渗，顺着风往里摸，走了不到二十米脚下就塌了——七十年代施工工艺问题，竖井挖到底横向开凿时没做支护，土体干缩崩解，踩塌的那块正是酥得最厉害的一块。

命大。掉下来时背包卡在中段一根锈穿的铁管上，那根管子是通风管道的残余，锈得只剩一层壳，却撑住了整个人的重量；顺着铁管滑到井底，没受伤，只是脚踝崴了一下。站起来走了两步，横向坑道还在延伸——往哪儿延伸不知道，信号没有，路也封了，水快没了。表盘指向下午四点十七分，日落大概在七点四十前后，山里入夜快，井口外气温会掉到零以下；手放下来，靠着墙，呼吸放得很慢，想到了两年零三个月前的方晓。

——

"我不走。"

纪川的声音在 **K47 工棚** 里显得很大。

方晓蹲在 **折叠桌前**，背对着他，正在往笔记本上写什么。笔尖划在纸上的声音很轻，但在那种安静里，反而清晰得像刀刻。

"我没让你走。"方晓说。

"段里在传。"

"传什么。"

"说我在 **联络坑道** 里擅自行动，没报备，违反了天窗规程。要把我调回 **南山口**。"

方晓的笔停了一下。

"谁说的。"

"老孙。"

方晓没再问"老孙是谁"。

工棚外有风声，把 **铁皮顶** 吹得一鼓一鼓的。

"那你来找我干嘛。"方晓说。

纪川没接话。

他看着方晓的背影——那件洗得发白的工作服，领口有一道旧口子，是上一期 **天窗复核** 时被 **道钉** 划的。方晓到现在都没换。

"我走可以。"纪川说，"但你得给我一个理由。"

方晓这才转过身。

他的脸在头灯的侧光里显得很瘦，颧骨的阴影很深。他看着纪川，眼神平静得近乎冷淡。

"你不需要理由。"他说。

"我需要。"

"你不需要。"方晓把笔记本合上，"你走，是因为你该走了。跟我没关系。"

"方晓——"

"纪川。"方晓打断他，"这条 **旧联络线** 底下，有东西。"

纪川愣了一下。

"什么东西。"

"我说不上来。"方晓说，"但它在那儿。"

"所以你让我走？"

"我让你走，是因为你再往下查，会出事。"

纪川盯着他。

方晓的脸在灯光下一动不动。

"你怎么知道会出事。"纪川说。

"因为上一个查的人，已经出了事。"方晓说，"你没见过他。我见过。"

工棚外，风声突然大了起来。

——

纪川从回忆里退出来。

记忆停在「已经出了事」那句上。那时他不信——编制表上没有「上一个查的人」，他在联络巷里单独行动过三次，每次都囫囵回来。方晓不说为什么，他问不出来，最后被调回南山口。方晓没送他；营地里只剩工棚和车辙。

两年零三个月后，他在录音里又听见方晓的声音，连咳嗽的习惯都一样。

——

水袋又漏了一点，往高处挪了挪，让仅剩的水尽量积到袋口附近，随后猫着腰朝坑道深处走了几步——井底这一段比上面窄，坑壁上有七十年代气动凿岩机留下的凿痕，排列整齐；空气里混着铁锈味和渗水的咸。

侧耳听，坑道深处有很轻、很规律的闷响：咚，咚，咚，间隔大约两秒，不是滴水，也不是落沙。手电照过去，拐弯处一根测风铃管在气流里轻轻晃，管头撞在混凝土环上，像被天窗日从井口灌下来的气流推成摆锤。

拐弯后面是二号侧室入口，门楣铝牌上写着：**轨旁位移监测点**。便携轨检仪的自动锤还在工作，电池未耗尽，锤头每隔两秒敲击基座一次；自动锤关掉，坑道里一下子安静得能听见自己的呼吸。侧室门半掩，喇叭里传出方晓两年前的声线，正在念 K47+247 里程复核读数——监测点里循环播放的旧录音。

播放掐断，数据卡收进内兜。卫星电话依然没有信号，水袋只剩小半袋；天窗日禁令下，旧线带外不会有搜救队进来——得靠自己把录音和日志带出去。
