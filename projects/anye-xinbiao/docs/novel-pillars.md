# 《暗夜信标》创作支柱

## 支柱

1. **因果清晰** — 信号、坐标、人物动机均有前置；反转来自信息差，不是作者强行降神。
2. **沈柯的执拗** — 他追查是因为欠父亲一个答案，不是因为「主角光环」。
3. **科技写实** — 射电、电磁屏蔽、地下设施用可理解语言写，不堆伪术语。
4. **悬念服务主题** — 每章钩子指向「听见 vs 装作没听见」。

## 反支柱

- 不是外星接触爽文
- 不是多线群像
- 不是为续作留扣
