# 第2章 第一条线索

台风白鹿的尾翼还在阵地上空打转，数据分析楼却像一块被风掏空的壳，只剩机房里的白噪音。

沈柯和顾岚从侧门进来时，门禁灯闪了两下才识别通过——值班系统宕机后的后遗症。走廊里飘着海水和臭氧混合的气味，像刚被雷劈过的电缆。

"进三号室。"顾岚刷卡，"别碰主控台，只用离线终端。"

三号室是阵列数据备份间，墙上一排旧终端，平时只有季度审计才开。沈柯把防水背包里的频谱仪接上，顾岚插自己的便携盘。两台屏幕同时亮起，脉冲波形叠在一起，严丝合缝。

沈柯手指在键盘上飞快地敲。解码脚本是他入职时写的，没人审查过——维护工程师总得留点趁手的工具。

十六进制字符每四个一组滚过屏幕——

`TIMESTAMP: 2004.07.15 / 23:41:07`
`COORD: B3-Δ-07`
`SOURCE: NIGHT-BEACON / LEAK`

沈柯的后背僵住了。

2004年7月15日23时41分07秒。事故简报写的是23时40分供电中断。七秒。六十七秒若算上整点——总之，信标脉冲标记的时间，和父亲坠海那一夜，叠在同一分钟里。

"B3-Δ-07。"他在便签上画了个圆，"地下三层，第七区段。"

顾岚站在他身后半步，没坐。她的影子落在屏幕上，盖住半个波形。

"你解码出来了。"她说。不是祝贺，是陈述。

"顾工，这坐标指向——"

"禁区。"顾岚接话，"你我知道。规程也知道。"她伸手，"原始频谱片段和日志，交给我。我按流程上报瀚海能源派驻组。"

瀚海能源。沈柯第一次在这栋楼里听见这个名字，像一枚钉子敲进木板。

"上报可以。"沈柯把U盘在掌心转半圈，"副本我按规程登记在终端。您经手盖章就行。"

顾岚的耳钉在应急灯下闪了一下。"阵里规程第十二条，涉密原始数据不得过私人存储。"

"所以我只在站内终端做备份登记。"沈柯拔下U盘——明面上那一枚，银色，站里配发的——放进顾岚掌心。

顾岚收进外套内袋，指节在金属壳上敲了一下。"二十年前的事了，沈柯。B3的结论你父亲的事，都有定论。你的工作是维护天线，不是重开旧案。"

"结论我看过。"沈柯声音很平，"失足落水。可这条信标叫了二十年，今晚突然变强。顾工，您也在那一夜值班。"

顾岚嘴角没有任何弧度，转身往门口走。到门边停了一瞬，没回头。

"明天上午瀚海来做例行安全检查。别让他们看见你在查B3——吊销维护资格，不是威胁，是流程。"

门合上了。

沈柯在原地坐了整整四分钟。他等顾岚折返，等她用那种慢条斯理的语气补一句"还有别的要交吗"。她没有。

04:12。窗外天幕仍是墨蓝，风势稍减。

沈柯这才把视线移向显示器背后——那里贴着一枚黑色贴片，和散热孔同色。父亲通信兵时代的习惯：关键数据永远留第二份。改装过的微型存储模块，容量不大，够装几段原始频谱。

`Removable Disk (F:)`

文件还在。TIMESTAMP、COORD、LEAK，一字不差。

他把模块揣进贴身口袋，长长吐出一口气。便签上的"B3-Δ-07"被汗洇开一角，他重新写了一遍，写得更深。

如果2004年的简报是真的，B3-Δ-07对应的是地下三层七号变电间——事故后封死，对外称辐射超标。但信标脉冲的载波、调制、帧结构，全都指向二十年前站内自用的窄带协议。

有人把它留到了现在。

或者——有人二十年前就埋好了，等它醒来。

沈柯站起来，椅子滚轮在地板上短促一响。他走向门口，刚触到门把——

走廊尽头，监控室方向传来对讲机的杂音。

"……C区七号天线馈源舱，04:05，人员两名，男性维护工，女性数据分析……侧脸已采……"

沈柯的手指停住。

不是朝三号室来。声音往A区方向去了，像有人在调阅刚才台风夜的巡检录像。

他退后两步，从三号室后门的维护通道离开。这条路过通风井入口，地图上标着"设备层·非值班禁止"，但维护工有钥匙。

通道尽头，铁栅上的锁扣锈迹斑斑。沈柯摸出十二寸扳手，卡住第一颗螺栓。

便签还在口袋里。B3-Δ-07。

笔记本里若真有父亲留下的话，只能在那下面。

他拧动扳手，金属磨金属的声音在管道里被拉长，像暗处有什么东西低吟。

**倒计时从这一声开始。**
