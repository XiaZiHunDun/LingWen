# 灵感筛选流程

> 灵感生成后，必须经过结构化筛选才能进入立项。
> 参照：10_方法论/01_灵感捕捉与筛选.md

---

## 筛选阶段

### 阶段0：捕捉灵感

**输入**：无结构的灵感碎片（画面/情绪/"如果"/关系/对白/名场面）

**输出**：灵感原始记录（一句话描述）

---

### 阶段1：评估筛选（核心）

**判断标准**（满足任一即可继续）：

| 标准 | 说明 | 检查 |
|------|------|------|
| **强烈冲突** | 人vs人/人vs命运/人vs社会/人vs自己 | [ ] 有 |
| **"反差"** | 身份vs性格 / 目标vs阻碍 / 表面vs真相 | [ ] 有 |
| **情绪价值** | 能让创作者自己生气/心疼/兴奋/害怕 | [ ] 有 |

**淘汰条件**（全部不满足则淘汰）：
- 无冲突（平淡日常）
- 无反差（脸谱人物）
- 无情绪（读者不会有情感反应）

---

### 阶段2：提炼一句话故事

**模板**：`谁（主角）+ 想要什么（目标）+ 但遇到什么阻碍（核心冲突）+ 如果失败会怎样（代价）`

**示例**：
> 一个失忆的刑警必须在72小时内找回记忆并阻止连环杀手，但所有证据都指向他自己就是凶手，如果失败，下一个死者将是他的女儿。

**检查**：
- [ ] 主角清晰（一句话能想象出这个人）
- [ ] 目标具体（不是"要幸福"而是"要找回记忆"）
- [ ] 阻碍有力（外在敌人+自身缺陷）
- [ ] 代价够大（失败的后果很严重）
- [ ] 读者想问"然后呢？"

---

### 阶段3：核心三要素提取

**产出**：`00_核心三要素.yaml`

```yaml
# 核心三要素
protagonist:
  身份: "..."
  性格: "..."
  执念/短板: "..."
  一句话: "..."

goal:
  主目标: "..."
  具体描述: "..."
  贯穿全书: true/false

obstacle:
  类型: "外在反派 / 自身缺陷 / 社会环境"
  最大阻碍: "..."
  层层阻碍:
    - "阻碍1"
    - "阻碍2"

stakes:
  如果失败: "..."
  代价描述: "..."
```

---

### 阶段4：立项判断

| 结果 | 条件 | 下一步 |
|------|------|--------|
| **通过立项** | 三要素完整 + 一句话故事清晰 + 至少一个筛选标准通过 | 进入立项流程 |
| **暂存观察** | 有亮点但要素不全 | 补充后再评估 |
| **淘汰** | 不满足任何筛选标准 | 记录后不采用 |
