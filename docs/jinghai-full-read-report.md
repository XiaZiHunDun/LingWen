## 仍可修（P2，非必须）

1. **ch010 派出所**：沈舟说老周在「**灯塔下面**」，前文是「**西边雾灯**」——若西边雾灯就是灯塔旁那盏，加半句「雾灯在灯塔西坡下」即可。  
2. **ch006**：「她在三楼」出现两次（盐渍段 + 海图段），删一处。  
3. **ch005**：黑影台词「你妹妹的朋友」缺主语，可改「我是你妹妹的朋友」。  
4. **ch005 沈舟 agency**（机器 P1）：全程「被推着走」——若扩写，加一处「他选择不回应杂讯、只跟灯语」已有一点，可再在出港前加一句主动决定。
