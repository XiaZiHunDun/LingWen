- 测试用例 (3-4 项):
  1. `test_search_async_returns_same_results_as_sync` — monkeypatch sync `search` 返回 `[{id:1}]`, 调 `search_async` 返相同 (event loop release 验证)
  2. `test_search_async_does_not_block_event_loop` — 用 `asyncio` 跑 3 个并发 search_async，measuring elapsed < 3 × 单次 sync 时长 (验证真并发而非串行)

### T2.2 改 GREEN `infra/memory_system/vector/qdrant_client.py`

- 加方法 `async def search_async(self, collection_name, query_vector, top_k=None, query_filter=None) -> list[dict]`：

- 加方法 `async def _raw_vector_search_async(self, collection_name, query_vector, limit=10, query_filter=None) -> list[Any]`：
  - 内部 `return await asyncio.to_thread(self._raw_vector_search, collection_name, query_vector, limit=limit, query_filter=query_filter)`
- 全部 3-4 tests GREEN
- 关键不变: `search()` / `_raw_vector_search()` / `upsert()` / `delete()` / `batch_search()` / `search_with_filter()` 等 sync 方法不动；6+ 现有调用点零改动

## Phase 5 — T3 Cache TTL (HIGH-PERF)

### T3.1 写 RED 测试 `tests/memory_system/test_qdrant_cache_ttl.py`
- deps: 无
- DoD: `pytest tests/memory_system/test_qdrant_cache_ttl.py` → RED (新 TTL 字段未注入)
- 测试用例 (3-4 项):
  1. `test_lru_cache_get_removes_stale_entry` — put `(key, value, ts=now-10)`, ttl=5; get(返回 None)
  2. `test_lru_cache_get_returns_fresh_entry` — put `(key, value, ts=now)`, ttl=300; get 返回 value
  3. `test_lru_cache_ttl_zero_disables_cache` — ttl=0 等价禁用: put 后 get 仍 None (或每次都 miss)
  4. `test_lru_cache_put_same_key_keeps_original_ts` — 同一 key 多次 put, ts 不刷, 验证 fast TTL 时不被人为延长

### T3.2 改 GREEN `infra/memory_system/vector/qdrant_client.py` (_LRUCache 部分)
- 重写 `_LRUCache` 类:
  - 加 `ttl_seconds: float` 构造参数 (默认 300.0)
  - `_cache: dict[str, tuple[Any, float]]` 改成 list of (value, ts)
  - `get(key)` 检查 `time.monotonic() - ts > ttl_seconds` 则 del + 返 None
  - `put(key, value, ts=None)` 接受可选 ts (默认 `time.monotonic()`); 同 key 已存在只覆 value 保留 ts
  - `clear()` / `__len__()` 不变
- 改 `QdrantClientWrapper.__init__`:
  - 读 yaml `retrieval.cache_ttl_seconds` (默认 300)
  - `self._search_cache = _LRUCache(max_size=cache_size, ttl_seconds=ttl_seconds)`
- 改 yaml loader 不破 schema (新字段为可选 + 默认)
- 全部 3-4 tests GREEN
