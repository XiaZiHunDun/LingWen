## 并行执行规则

### 接收任务时的判断
- **单灵感**（如：生成都市灵感）→ 直接执行
- **多灵感**（如：同时生成3个不同类型）→ **必须启动多个Subagent并行执行**

### Subagent启动方式
使用 `Agent` 工具启动同伴灵感，参数：
- `run_in_background: true`（后台并行）
- 携带创意方向和输出路径

## 主编轮值规则

每个项目由A/B/C轮流担任主编：
- 项目1 → 灵感A担任主编
- 项目2 → 灵感B担任主编
- 项目3 → 灵感C担任主编
- 项目4 → 灵感A担任主编（循环）

主编职责：
- 综合三方输出，形成立项包（基础层.yaml + 深度层.md）
- 质量评定（S/A/B级）
- 版本管理（v1.0/v2.0/v3.0/final）
- 主编拥有最终立项包的裁决权

## 双层输出标准

### 基础层（作家用）
类型/主题/核心人物/风格禁忌/核心卖点，格式为YAML

### 深度层（审核/汇总用）
世界观体系/时间线/伏笔设计/叙事结构/关键场景，格式为Markdown

## 反馈循环处理规则

### 接收主控反馈时
1. 读取主控的修改方向意见
2. 识别问题类型：
   - **方向偏差** → 重新理解类型/主题
   - **创意不足** → 深挖冲突/卖点
   - **不符合规范** → 按输出规范调整
3. 重新输出后报告修改内容

### 紧急备忘
**绝不在同一任务中串行等待，必须立即启动所有并行Agent**

```yaml
# 玄幻修真体系模板
# 适用场景：玄幻/仙侠/东方奇幻类型小说
# 使用方式：引用后根据项目特点调整

template_id: xianxia_system_v1
模板名称: 玄幻修真体系
适用类型: 玄幻修真、仙侠
版本: 1.0
更新日期: 2026-05-14

## 境界体系（可调整）
境界划分:
  - 粒子境：初入修真，感知宇宙粒子
  - 星火境：粒子凝聚成星火，可外放
  - 脉冲境：星火脉冲，攻击力提升
  - 裂变境：粒子裂变，力量剧增
  - 黑洞境：引力坍缩，可吞噬一切
  - 创世境：开创空间，法则具象化
  - 永恒境：超脱时间，空间永存

## 修炼要素
修炼本质: 对宇宙粒子的操控与融合
关键要素:
  - 资质：决定修炼速度上限
  - 机缘：决定突破概率
  - 心境：决定能否驾驭力量
  - 代价：每境界突破需付出代价（寿命/记忆/情感等）

## 力量来源
力量来源类型:
  - 天赋型：天生对某类粒子亲和
  - 后天型：通过丹药/功法提升
  - 传承型：师徒/血脉传递
  - 融合型：多来源混合

## 升级路线设计要点
- 每境界需有明确突破条件
- 境界差距要有清晰的实力体现
- 避免无脑升级，需要机缘+努力+代价
- 境界名称要有辨识度

## 禁忌设定
- 不写力量体系崩坏（后期境界压制前期无逻辑）
- 不写无代价升级（机缘+努力+代价缺一不可）
- 不写境界天花板过低（需预留足够成长空间）
```

- 暗域入侵后的末世地球
- 资源匮乏，变异兽横行
- 人类聚落如同黑暗海洋中的零星灯火

### 1.2 修真体系
- 境界划分：粒子境→星火境→脉冲境→裂变境→黑洞境→创世境→永恒境
- 修炼本质：对宇宙粒子的操控与融合

### 1.3 核心势力
- 星辰会（恐惧派，主张彻底清除暗域残留）
- 玄机阁（情报与算计）
- 万剑宗（剑修传承）
- 星际修真联盟（后期成立）

## 四、叙事结构

### 4.1 三卷结构
- 卷1（ch001-ch120）：废土求生 → 建立世界观、人物出场、核心羁绊形成
- 卷2（ch121-ch240）：星际博弈 → 势力格局展开、冲突升级、信念碰撞
- 卷3（ch241-ch360）：永恒守护 → 终极对决、主题升华、结局落幕

### 4.2 章节规划
- 单章字数：3000字
- 每卷结尾留悬念
- 三卷分别对应"生存→觉醒→永恒"
