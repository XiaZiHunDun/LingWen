## T2.5 — `infra/persistence/__init__.py` + `bootstrap.py` (1 commit)

**目标**: 把 8 个 storage 类的注册集中到一处 import side-effect-free module。

**GREEN**:
- `infra/persistence/bootstrap.py`:
  ```python
  from infra.cross_volume.storage import RippleStorage
  from infra.agent_system.cost_persistence import CostTrackerDB
  from infra.agent_system.budget_persistence import BudgetPersistence
  from infra.reading_power.db import ReadingPowerDB
  from infra.state.database import WorkflowDB
  from infra.agent_system.social_engine.relationship_tracker import RelationshipTracker
  from infra.persistence.paths import (
      RIPPLE_DB, COST_TRACKER_DB, WORKFLOW_DB, READING_POWER_DB, RELATIONSHIP_DB,
  )
  from infra.persistence.registry import register

  def register_all() -> None:
      register("ripple", RippleStorage, db_path=RIPPLE_DB)
      register("cost", CostTrackerDB, db_path=COST_TRACKER_DB)
      register("workflow", WorkflowDB, db_path=WORKFLOW_DB)
      register("reading", ReadingPowerDB, db_path=READING_POWER_DB)
      register("relationship", RelationshipTracker, db_path=RELATIONSHIP_DB)
      register("budget", BudgetPersistence, db_path=COST_TRACKER_DB)
  ```
- `infra/persistence/__init__.py`: `from infra.persistence.registry import register, get, reset, reset_all` 等导出
- **不在 import 时自动 register** (避免 side-effect)；dashboard/app.py 显式调 `register_all()` 一次

**验证**: `python3 -c "from infra.persistence.bootstrap import register_all; register_all(); from infra.persistence.registry import get; print(get('ripple'))"`
**Commit**: `feat(persistence): phase 15.0 T2.5 — bootstrap register_all + 6 storage bindings`
