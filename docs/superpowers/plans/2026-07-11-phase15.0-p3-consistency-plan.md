- 单例 `DatabasePool`:
  ```python
  class DatabasePool:
      _pools: dict[str, sqlite3.Connection] = {}
      @classmethod
      def get(cls, db_path: Path, schema_key: str | None = None) -> sqlite3.Connection:
          key = str(db_path)
          if key not in cls._pools:
              conn = sqlite3.connect(str(db_path))
              conn.execute("PRAGMA foreign_keys = ON")
              conn.row_factory = sqlite3.Row
              cls._pools[key] = conn
          return cls._pools[key]
      @classmethod
      def reset(cls) -> None: cls._pools.clear()
  ```
- `get_db(path, schema)` factory 包裹
