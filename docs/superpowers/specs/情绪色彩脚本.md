# 情绪色彩脚本（Emotion Color Script）

> **来源**：动画制作色彩脚本（Color Script）概念
> **作用**：用视觉化色块规划全篇情绪基调，让情绪设计更直观

---

## 一、定义与目的

情绪色彩脚本是用色块序列规划全篇情绪基调的文档。每个色块代表一个情绪段落，用色彩直观呈现情绪变化。

**核心目的**：将"情绪设计"从抽象描述转为可视化呈现，便于检查和调整。

---

## 二、色彩与情绪对应表

| 色彩 | 色块示例 | 情绪类型 | 适用场景 |
|------|----------|----------|----------|
| 沉稳蓝 | `#1E3A5F` | 低沉、困境 | 主角逆境、开篇 |
| 希望青 | `#4A90A4` | 转机、希望 | 金手指出现 |
| 紧张橙 | `#C45C3E` | 冲突、危机 | 打斗、高潮 |
| 爆发红 | `#8B0000` | 愤怒、决战 | 最终对决 |
| 温暖黄 | `#DAA520` | 温情、治愈 | 感情戏、温馨 |
| 平静绿 | `#2E8B57` | 平和、过渡 | 日常、休整 |
| 神秘紫 | `#4B0082` | 悬疑、未知 | 伏笔、谜团 |
| 灰调 | `#708090` | 消沉、绝望 | 低谷、挫折 |

## 三、格式模板

```markdown
## 情绪色彩脚本

### [卷名/阶段名]

[阶段1] ████ #色码 情绪描述 - 代表章节
[阶段2] ████ #色码 情绪描述 - 代表章节
...

### 全篇情绪曲线

[第一章]████#1E3A5F 沉稳蓝 - 困境开篇
     ████#4A90A4 希望青 - 转机出现
     ████#C45C3E 紧张橙 - 冲突爆发
     ...
```

## 四、使用场景

### 场景1：构思期规划
1. 确定各阶段情绪基调
2. 选择对应色彩
3. 验证情绪曲线是否流畅（不应突然跳跃）

### 场景2：写作时对照
1. 写作前查看当前章节的色彩标注
2. 对照检查情绪是否符合预设
3. 如需调整，记录原因

### 场景3：评审时验证
1. 检查情绪变化是否合理
2. 验证色彩与内容是否匹配
3. 识别情绪节奏问题

---

## 五、情绪色彩脚本检查清单

- [ ] 全篇情绪变化有逻辑（不是随机跳跃）
- [ ] 每个色彩对应合适的场景
- [ ] 高潮前有足够的铺垫色（情绪积累）
- [ ] 低谷后有转机色（不能一直沉）
- [ ] 色彩切换有渐进过程（避免 abrupt 跳变）
- [ ] 与目标读者群体匹配（爽文vs虐文情绪曲线不同）

## 六、示例

```markdown
## 《修仙逆袭录》情绪色彩脚本

[Ch4-6] ████ #4A90A4 希望青 - 偶获传承，崭露头角
[Ch7-9] ████ #C45C3E 紧张橙 - 宗门大比，初露锋芒
[Ch10]   ████ #DAA520 温暖黄 - 情感互动，温馨时刻
[Ch11-13] ████ #C45C3E 紧张橙 - 强敌出现，危机降临
[Ch14-15] ████ #4A90A4 希望青 - 绝境突破，反杀成功

### 情绪曲线可视化

Ch1-3: ████████ (低沉)
Ch4-6: ████ (上升)
Ch7-9: ████████████ (紧张)
Ch10:  ████ (舒缓)
Ch11-13: ████████████████ (危机)
Ch14-15: ████████ (爆发)
```

## 七、与情感曲线模板的区别

| 维度 | 情感曲线模板 | 情绪色彩脚本 |
|------|--------------|--------------|
| 视角 | 情节/事件维度 | 情绪/感受维度 |
| 表达 | 文字描述 | 色块可视化 |
| 粒度 | 章节级 | 阶段/卷级 |
| 侧重点 | 爽点节奏 | 情绪基调变化 |
