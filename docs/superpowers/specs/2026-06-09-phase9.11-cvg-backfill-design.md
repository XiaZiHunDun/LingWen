## Design

### 1. Extraction Rules YAML Schema

**File**: `infra/cross_volume/extraction_rules.yaml` (NEW, ~60L)

```yaml
# Phase 9.11: 4-dim rule-based extraction config
# 主公后续改规则: 0 改 Python, 仅改本 YAML

character:
  name_pattern: "[\\u4e00-\\u9fff]{2,4}(?:[·•][\\u4e00-\\u9fff]{2,4})?"
  # ↑ 2-4 中文名, 可含 ·• (e.g. "苏晨·凡")

  alias_map:
    "星尘": "星尘剑客"
    "尘": "星尘剑客"
    "林凡": "林凡"
    "凡": "林凡"
  # ↑ 跨章别名映射 (人工 curation), alias 视作 canonical name 出现

  blacklist:
    - "那个"
    - "什么"
    - "怎么"
    - "这里"
    - "那里"
    - "如此"
    - "这般"
    - "此时"
    - "此刻"
  # ↑ 排除通用代词, 避免误抽

  occurrence_threshold: 2
  # ↑ 同一 character name 出现 ≥ 2 章 才入 node (避免一次性出场被误入)

foreshadow:
  keywords:
    - "伏笔"
    - "暗示"
    - "铺垫"
    - "暗藏"
    - "留下"
    - "埋下"
    - "未曾"
  # ↑ 大纲里出现这些词 → 伏笔信号

  pattern: "(?P<content>[\\u4e00-\\u9fff]{6,30}?(?:伏笔|暗示|铺垫|暗藏|留下|埋下))"
  # ↑ 6-30 字内含上述 keyword

  chapter_window: 5
  # ↑ 5 章内同一关键词聚合成 1 node (跨章展开)

setting:
  name_pattern: "[\\u4e00-\\u9fff]{2,6}(?:学院|山门|大陆|帝国|秘境|洞府|遗迹|海域|城|宗|派|观|谷)"
  # ↑ 2-6 字 + 名词后缀 (设定/地点)

  blacklist:
    - "这个"
    - "那个"
    - "此时"
    - "此刻"

  occurrence_threshold: 1
  # ↑ 设定一次性出场即可入 (跟 character 不同, 设定稳定不消失)

plot_point:
  pattern: "^##\\s+(.+)$"
  # ↑ 大纲 .md 的 ## 章节标题抽取

  min_length: 8
  max_length: 50
  # ↑ 标题长度过滤, 避免 ## 简介 / ## 备注 等

  blacklist:
    - "^## 简介$"
    - "^## 备注$"
    - "^## 元数据$"
```

**Schema invariants**:
- 4 维每维 ≥ 1 字段 (`name_pattern` / `pattern` / `keywords`), 缺字段启动 `ValueError`
- `alias_map` 双向唯一 (alias → canonical, 不允许多 alias 映射同一 canonical 后又被覆盖)
- YAML load 失败 → `yaml.YAMLError` propagate
- 0 强制 schema validation (留 `jsonschema` 留 Phase 9.12+ 完整化)

### 2. 4 Extractor Class

**File structure**:
```
infra/cross_volume/extractors/
├── __init__.py            (~10L, 4 extractor 入口)
├── character_extractor.py (~80L, CharacterExtractor)
├── foreshadow_extractor.py(~70L, ForeshadowExtractor)
├── setting_extractor.py   (~70L, SettingExtractor)
└── plot_point_extractor.py(~80L, PlotPointExtractor)
```

**Common interface** (per Extractor):
```python
class BaseExtractor(Protocol):
    def __init__(self, rules: dict, volume: int) -> None: ...
    def extract(self, chapter_path: Path) -> list[ReferenceNode]: ...
    # ↑ 输入 1 章 .md, 返 0+ ReferenceNode (已去重 + payload 累积)
```

**CharacterExtractor** (~80L):
- `name_pattern` 走 `re.findall` 扫全章 .md
- `alias_map` 双向: 匹配 `alias` → 替换为 `canonical name` (写入 `name` 字段)
- `blacklist` 过滤
- `occurrence_threshold` ≥ N 章 触发入 node
- 跨章聚合: Backfiller 传累积 `chapter_appearances: list[int]`, 每次出现追加 chapter_num
- payload 字段: `{name, role: "unknown", first_appearance_volume, first_appearance_chapter, chapter_appearances}`

**ForeshadowExtractor** (~70L):
- 扫大纲 .md 的 `##` 标题 + 正文 keyword
- `keywords` 集合匹配, 出现 keyword 抽取 `pattern` named group → `content`
- `chapter_window: 5` 聚合: 跨章同名 `content` 前缀 (e.g. "古剑碎片" 6 字) 聚合成 1 node
- payload 字段: `{content, planted_volume, planted_chapter, chapter_appearances}`

**SettingExtractor** (~70L):
- 跟 CharacterExtractor 类似, 但 `name_pattern` 不同 (名词后缀匹配)
- `blacklist` 过滤
- `occurrence_threshold: 1` (跟 character 不同, 设定一次入)
- payload 字段: `{name, type: "unknown", first_intro_volume, first_intro_chapter, chapter_appearances}`

**PlotPointExtractor** (~80L):
- 仅扫大纲 .md (正文 chapter_milestone 拆不出)
- `pattern: ^##\\s+(.+)$` 抽 `## 标题`
- `min_length` / `max_length` 过滤
- `blacklist` 过滤
- 1 ## 标题 = 1 node (跟 character 不同, plot_point 按章拆 1 个不聚合)
- payload 字段: `{title, type: "unknown", volume, chapter, impact_level: "unknown"}`

**Key invariants**:
- 4 Extractor 走 `BaseExtractor` Protocol, 同接口 (`__init__(rules, volume)` + `extract(chapter_path) -> list[ReferenceNode]`)
- Extractor 0 写库 (per Extract pattern), 仅产 ReferenceNode list; Backfiller 负责图注入 + 写库
- Extractor 0 维护状态, 跨章聚合由 Backfiller 维护
- `created_by` 字段 = `f"backfill:{dim}_extractor"` (跟 Phase 9.10 `created_by="manual"` 区分)

### 3. Backfiller 编排

**File**: `infra/cross_volume/backfill.py` (~120L)

```python
class Backfiller:
    def __init__(
        self,
        rules_path: Path = Path("infra/cross_volume/extraction_rules.yaml"),
        corpus_root: Path = Path("novel-factory/03_内容仓库/04_正文"),
        graph: CrossVolumeReferenceGraph | None = None,
    ) -> None:
        self._rules = yaml.safe_load(rules_path.read_text(encoding="utf-8"))
        self._corpus_root = corpus_root
        self._graph = graph or CrossVolumeReferenceGraph(
            RippleStorage(Path("infra/.state/ripple.db"))
        )
        # 4 Extractor 实例化 (Phase 9.11 lazy init, 0 立即扫)
        self._extractors: dict[DimensionT, BaseExtractor] = {
            "character": CharacterExtractor(self._rules["character"], volume=1),
            "foreshadow": ForeshadowExtractor(self._rules["foreshadow"], volume=1),
            "setting": SettingExtractor(self._rules["setting"], volume=1),
            "plot_point": PlotPointExtractor(self._rules["plot_point"], volume=1),
        }

    def run(
        self,
        dry_run: bool = True,
        volume_filter: int | None = None,
    ) -> BackfillStats:
        # 1. 4 维并行扫 (顺序: 4 extractors 各自串行, 内部 re.findall 0 IO)
        all_nodes: dict[DimensionT, dict[str, ReferenceNode]] = {
            "character": {},
            "foreshadow": {},
            "setting": {},
            "plot_point": {},
        }
        for chapter_path in self._iter_chapters(volume_filter):
            chapter_num = parse_chapter_num(chapter_path)  # ch001.md → 1
            volume_num = parse_volume_num(chapter_path)    # ch001-ch120 → vol 1
            for dim, extractor in self._extractors.items():
                for node in extractor.extract(chapter_path):
                    self._merge_node(all_nodes[dim], node, chapter_num)
                    # ↑ 跨章聚合: 同 id (e.g. "character:李青云") → 合并 chapter_appearances

        # 2. --dry-run 默认 (主公决策 Q4): 0 写库, 仅 print 统计
        if dry_run:
            return BackfillStats.from_nodes(all_nodes, dry_run=True)

        # 3. --execute: storage.append_nodes_atomic × 1 (1 atomic_batch wrap N inserts)
        all_node_list = [
            n for dim_nodes in all_nodes.values() for n in dim_nodes.values()
        ]
        self._graph._storage.append_nodes_atomic(all_node_list)
        return BackfillStats.from_nodes(all_nodes, dry_run=False)

    def _iter_chapters(self, volume_filter: int | None) -> Iterator[Path]:
        # 扫 03_内容仓库/04_正文/ch{001..359}.md + ch{001..359}_大纲.md
        # volume_filter: 1/2/3 抽样式, 0 = 全 359 章
        ...

    def _merge_node(
        self,
        bucket: dict[str, ReferenceNode],
        new_node: ReferenceNode,
        chapter_num: int,
    ) -> None:
        # 跨章聚合: 同 id → 合并 payload.chapter_appearances (append chapter_num)
        if new_node.id in bucket:
            existing = bucket[new_node.id]
            merged = dataclasses.replace(
                existing,
                payload={
                    **existing.payload,
                    "chapter_appearances": list(
                        set(existing.payload["chapter_appearances"] + [chapter_num])
                    ),
                },
            )
            bucket[new_node.id] = merged
        else:
            bucket[new_node.id] = dataclasses.replace(
                new_node,
                payload={
                    **new_node.payload,
                    "chapter_appearances": [chapter_num],
                },
            )


@dataclass(frozen=True)
class BackfillStats:
    character_count: int
    foreshadow_count: int
    setting_count: int
    plot_point_count: int
    total_count: int
    elapsed_s: float
    dry_run: bool

    def summary(self) -> str:
        return (
            f"[{'DRY-RUN' if self.dry_run else 'EXECUTED'}] "
            f"character={self.character_count} "
            f"foreshadow={self.foreshadow_count} "
            f"setting={self.setting_count} "
            f"plot_point={self.plot_point_count} "
            f"total={self.total_count} "
            f"elapsed={self.elapsed_s:.2f}s"
        )
```

**Key invariants**:
- Backfiller 0 mutation of input `graph`, 仅 `add_node` (append-only)
- `_iter_chapters` 走 `pathlib.Path.glob`, 跨章顺序按 `sorted(chapter_num)`
- 跨章聚合通过 `id` (e.g. `character:李青云`), Extractor 产 node 时 `id` = `{dim}:{canonical_name}` (deterministic, 0 uuid4)
- 聚合后 `chapter_appearances` 排序 + 去重 (避免同一章出现 2 次)
- `--dry-run` 跟 `--execute` 0 共享 state (各自从 corpus 扫, idempotent)
- 4 维 0 cross-dim 干扰 (per-dim bucket)

### 6. Error Handling + Edge Cases

| 场景 | 行为 |
|------|------|
| YAML 缺失 | `FileNotFoundError` propagate (CLI user 路径错) |
| YAML 缺字段 (e.g. `character.name_pattern`) | `KeyError` propagate (config schema 错) |
| `re.error` 错 regex | `re.error` propagate, CLI exit 1 |

```python
def append_nodes_atomic(self, nodes: list[ReferenceNode]) -> None:
    """Phase 9.11: 1 atomic_batch wrap N inserts, 0 partial commit.
    复用 Phase 9.10 atomic_batch context manager, 仅封装 N × INSERT.
    """
    with self.atomic_batch() as conn:
        for node in nodes:

                    "INSERT INTO reference_nodes VALUES (?,?,?,?,?,?,?,?,?)",

                     node.title, node.description,
                     json.dumps(node.payload, ensure_ascii=False),
                     node.created_at.isoformat(), node.created_by),

    # atomic_batch 自动 commit (0 exception) 或 rollback (exception)
```
