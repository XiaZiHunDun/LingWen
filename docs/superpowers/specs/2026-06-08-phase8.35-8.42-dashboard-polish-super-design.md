### Phase 8.36 — class 镜像同步 lint

**问题**: 33 testid 已有镜像 class (Phase 8.13+8.31+8.32 加 testid 时手动加 class), 0 lint 强制同步, 加新 testid 忘加 class 或 class 拼错 → devtool 调试找不到 → drift 风险.

**设计**:
- 新 ESLint local rule `dashboard/frontend/eslint-rules/testid-class-sync.js` (NEW, ~80L), 镜像 Phase 8.33 `no-class-selector-in-test.js` pattern:
  ```js
  // eslint-rules/testid-class-sync.js — Phase 8.36
  // 规则: .vue 元素加 data-testid="x" 必加 class 含 "x" (kebab-case 镜像)
  //      加 class="...x..." 必加 data-testid="x"
  // 形态: kebab-case 镜像, 跟 Phase 8.13+8.31+8.32 既有 class 命名 1:1
  // 例外: 顶层 component root (e.g. <template><App/>) 0 强制 (component 0 class 自定义)
  module.exports = {
    meta: { type: 'problem', docs: { description: '...' }, schema: [] },
    create(context) {
      return context.parserServices.defineTemplateBodyVisitor({
        // VAttribute visitor
        'VAttribute[key.name="data-testid"]'(node) {
          const testid = node.value.value
          // 找同级 VAttribute key.name === 'class'
          const parent = node.parent
          const classAttr = parent.attributes.find(a => a.key.name === 'class')
          if (!classAttr) {
            context.report({ node, message: `data-testid="${testid}" 必须加 class 镜像` })
            return
          }
          if (!classAttr.value.value.includes(testid)) {
            context.report({ node, message: `class 必须包含 "${testid}"` })
          }
        },
        'VAttribute[key.name="class"]'(node) {
          const cls = node.value.value
          // 找同级 VAttribute key.name === 'data-testid'
          const parent = node.parent
          const testidAttr = parent.attributes.find(a => a.key.name === 'data-testid')
          if (testidAttr) return  // 上面已 check
          // 否则提取 class 中 testid-pattern (kebab-case 单 token)
          const tokens = cls.split(/\s+/)
          for (const t of tokens) {
            if (/^[a-z][a-z0-9-]*$/.test(t)) {
              context.report({ node, message: `class="${t}" 必加 data-testid="${t}"` })
            }
          }
        },
      })
    },
  }
  ```
