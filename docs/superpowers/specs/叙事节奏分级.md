# 叙事节奏分级

> **来源**：动画制作"一拍一/二/三"概念
> **作用**：明确快/中/慢节奏的适用场景和切换标准，让节奏控制可执行

---

## 一、节奏分级定义

### P1 快节奏

**特征**：
- 短句为主，动作密集
- 冲突持续升级
- 信息释放快
- 几乎没有冗余描写

**适用场景**：
| 场景类型 | 示例 |
|----------|------|
| 打斗 | 主角对战、争夺宝物 |
| 危机 | 逃生、追杀、限时任务 |
| 高潮 | 决战、揭露真相、突破境界 |
| 爽点 | 逆袭、打脸、装逼成功 |

**写作要点**：
- 减少内心独白
- 多用"然后"、"接着"推进
- 句子控制在10-15字以内

---

### P2 中节奏

**特征**：
- 叙述与动作平衡
- 有描写但不过度
- 情节稳步推进
- 偶尔有小冲突

**适用场景**：
| 场景类型 | 示例 |
|----------|------|
| 日常推进 | 修炼、收集、准备 |
| 情节发展 | 接受任务、结交盟友 |
| 情感铺垫 | 关系深化、内心挣扎 |
| 过渡 | 上一个高潮到下一个 |

**写作要点**：
- 每200字至少一个小冲突
- 描写与对话比例约3:7
- 保持"翻页钩子"

---

### P3 慢节奏

**特征**：
- 描写细腻
- 内心独白丰富
- 氛围感强
- 信息释放缓慢

**适用场景**：
| 场景类型 | 示例 |
|----------|------|
| 日常 | 用餐、休息、闲聊 |
| 感情 | 告白、分别、治愈 |
| 氛围 | 夜景、风景、心理转折 |
| 伏笔 | 神秘暗示、细节铺设 |

**写作要点**：
- 可以用长句渲染
- 内心独白占比重
- 注重感官描写

---

## 二、节奏切换规则

### 黄金法则

```
连续快节奏(P1)不超过3章 → 必须插入1章慢节奏(P3)呼吸
连续慢节奏(P3)不超过2章 → 必须插入1章快节奏(P1)激活
```

### 每卷节奏要求

| 卷类型 | P1数量 | P2数量 | P3数量 | 说明 |
|--------|--------|--------|--------|------|
| 冲突卷 | 5+ | 3-4 | 1-2 | 打斗密集 |
| 过渡卷 | 2-3 | 4-5 | 2-3 | 日常为主 |
| 高潮卷 | 6+ | 2-3 | 1 | 决战前夕 |

### 章节内节奏变化

一个章节内可以有节奏变化：
```
开头（P1快）→ 中间（P2中）→ 结尾（P1快）
```

## 三、节奏检测清单

### 单章检查
- [ ] 开头是否在正确节奏上（参考上下文）
- [ ] 全章是否有起伏（不能平铺直叙）
- [ ] 结尾是否留有钩子
- [ ] 字数是否符合该节奏类型的标准

### 连续章节检查
- [ ] 是否连续3章以上都是P1（违反呼吸规则）
- [ ] 是否连续2章以上都是P3（节奏过慢）
- [ ] 节奏切换是否渐进（避免P1直接跳P3）
- [ ] 高潮后是否有合理缓冲

### 全篇检查
- [ ] 每卷节奏分布是否合理
- [ ] 爽点间隔是否合适（3章一小爽，10章一大爽）
- [ ] 是否有节奏拖沓的段落（可删减）

## 四、节奏类型字数参考

| 类型 | 每章字数范围 | 说明 |
|------|--------------|------|
| P1快节奏 | 2000-3000 | 精简有力 |
| P2中节奏 | 2500-3500 | 标准范围 |
| P3慢节奏 | 3000-4000 | 细腻渲染 |

## 五、节奏问题诊断

| 症状 | 问题 | 解决方案 |
|------|------|----------|
| 读者反馈"无聊" | 节奏太慢P3过多 | 增加冲突，缩短描写 |
| 读者反馈"太赶" | 节奏太快P1过多 | 增加铺垫，延长过渡 |
| 读者反馈"跳脱" | 节奏忽快忽慢 | 建立渐进切换机制 |
| 写不下去 | 当前节奏不适合内容 | 调整节奏类型标注 |

## 六、节奏与情绪的关系

| 目标情绪 | 推荐节奏 | 示例 |
|----------|----------|------|
| 紧张 | P1 | 打斗、追杀 |
| 兴奋 | P1→P2 | 装逼成功 |
| 温馨 | P3 | 感情戏 |
| 压抑 | P3 | 困境描写 |
| 爆发 | P1 | 高潮对决 |
