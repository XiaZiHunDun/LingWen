## 像给朋友说

这本书**最适合当「第二本样章候选」**：不是静海那种盐雾散文，而是**都市悬疑 + 档案恐怖**——告示、编号、地图搜不到的巷名，读感像凌晨三点刷论坛。

ch9–10 的 **37 路公交 / 1998.3.17 / 寻人启事 / 底片负像** 收束漂亮，「她要找的是她自己」这句能留住人。

需要继续修的是：**短句堆叠**（尤其 ch6–8）和 **林栀被动段**（档案馆之前略「被推着走」，ch8 抉择后好转）。
