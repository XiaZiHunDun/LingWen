# 灵感部门提示词模板索引

> 版本：1.0 | 更新：2026-05-19

## 概述

灵感部门提示词模板位于 `.claude/prompts/inspiration/` 目录，为灵感生成提供标准化指导。

---

## 模板列表

| 模板文件 | 用途 | Token估算 |
|----------|------|-----------|
| `type-analysis-prompt.md` | 类型分析与卖点提炼 | ~1500-2000 |
| `worldbuilding-prompt.md` | 完整世界观设计 | ~2000-3000 |
| `character-inspiration-prompt.md` | 角色灵感生成 | ~1800-2500 |

---

## 灵感部门核心职责

灵感部门负责生成立项文件（基础层+深度层）：

| 层级 | 内容 | 负责人 |
|------|------|--------|
| **基础层.yaml** | 类型、主题、核心冲突、卖点、受众 | 灵感A |
| **深度层.md** | 世界观、时间线、伏笔、叙事结构 | 灵感B/C |

---

## 输出规范

### 基础层.yaml 格式
```yaml
type: 玄幻
theme: 星际废土
core_conflict: 生存vs守护
selling_points:
  - 废土生存
  - 星际探险
audience: 16-25岁男性
```

### 深度层.md 结构
```markdown
## 世界观
...

## 时间线
...

## 伏笔布局
...
```

---

## 质量标准

1. **类型定位清晰**：类型融合合理，核心元素明确
2. **卖点独特**：有差异化优势，不是泛泛而谈
3. **世界观自洽**：力量体系平衡，社会结构合理
4. **角色有深度**：性格有层次，弧光设计合理
5. **伏笔丰富**：有深度挖掘空间
