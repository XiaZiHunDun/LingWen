## 三、评分标准详解

### 3.1 完整性（1-5分）

| 分数 | 标准 |
|------|------|
| 1 | 缺少角色、场景、情节等核心要素 |
| 2 | 缺少2项以上核心要素 |
| 3 | 核心要素齐全，但深度不够 |
| 4 | 要素齐全，有一定深度 |
| 5 | 要素齐全，深度充分，包含伏笔/势力等 |

### 3.2 可执行性（1-5分）

| 分数 | 标准 |
|------|------|
| 1 | 指导模糊，无法据此创作 |
| 2 | 指导模糊，需大量补充 |
| 3 | 指导基本可用，需部分补充 |
| 4 | 指导明确，稍作调整即可 |
| 5 | 指导非常具体，直接可用 |

### 3.3 上下文关联（1-5分）

| 分数 | 标准 |
|------|------|
| 1 | 与大纲/设定完全脱节 |
| 2 | 与大纲/设定部分关联 |
| 3 | 基本关联，但有矛盾 |
| 4 | 较好关联，一致性高 |
| 5 | 完全关联，无矛盾，自动同步 |

### 3.4 效率提升（1-5分）

| 分数 | 标准 |
|------|------|
| 1 | 比无提示词更慢 |
| 2 | 与无提示词相当 |
| 3 | 略有提升（<20%）|
| 4 | 明显提升（20-50%）|
| 5 | 大幅提升（>50%）|

## 四、问题分类索引

### 4.1 提示词本身问题

| 问题类型 | 典型表现 | 解决方案 |
|----------|----------|----------|
| 长度不足 | 输出质量差 | 扩展提示词到200+行 |
| 结构缺失 | 输出混乱 | 增加结构化模板 |
| 示例不足 | 理解偏差 | 添加更多示例 |
| 约束不清 | 边界模糊 | 明确禁止/允许事项 |

### 4.2 上下文问题

| 问题类型 | 典型表现 | 解决方案 |
|----------|----------|----------|
| 大纲不同步 | 情节矛盾 | 自动同步机制 |
| 设定漂移 | 角色形象变 | 建立角色状态表 |
| 伏笔断裂 | 伏笔无回收 | 伏笔追踪表 |

### 4.3 执行问题

| 问题类型 | 典型表现 | 解决方案 |
|----------|----------|----------|
| Token超限 | 被截断 | 压缩/分段 |
| 耗时过长 | >60秒 | 优化结构 |
| 重复生成 | 内容重复 | 增加去重提示 |
