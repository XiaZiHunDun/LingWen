# 质量检查维度详解 v1.1（2026-05-19）

## 11 维质量检查

| 维度 | 类型 | 说明 |
|------|------|------|
| 命名一致性 | 规则 | 文件名chXXX.md与章节内标题一致 |
| 内容完整性 | 规则 | **本章完**标记、字数>=500 |
| 章节重复 | 规则 | 跨章节相似度>80%预警 |
| 人物状态 | 规则 | 性别/生死/形态前后矛盾 |
| 时间线 | 规则 | "年前"与"瞬间"同时出现等错误 |
| 情节关联度 | 规则 | 每段落与前后章节的关联程度 |
| 伏笔回收率 | 规则 | 首次出现元素在后续N章内回收 |
| 场景逻辑 | 规则 | 场景转换合理性、孤岛章节检测 |
| 情感节奏 | 规则 | 情绪波动是否合理 |
| 对话风格 | 规则 | 角色对话字数异常检测 |
| 人物弧光 | **LLM** | 角色弧光完整性（通过Agent调用） |

## 情感审核专项（与评审维度体系 S1-S8 对齐）

| 维度 | 检查内容 |
|------|---------|
| S1 剧情完整性 | 章内目标—障碍—结果；卷内因果 |
| S2 逻辑自洽 | 时间线、战力、势力行为 |
| S3 文笔风格 | 语言、对话、描写统一 |
| S4 情感共鸣 | 情绪曲线有效性 |
| S5 节奏控制 | 信息投放缓急、战斗/文戏比例 |
| S6 可读性 | 读者是否一口气读完 |
| S7 主角魅力 | 读者是否想看主角成功 |
| S8 人物弧光 | 主角是否发生可感知的变化；弧光完整性 |

## 质量等级

| 等级 | 条件 | 处理 |
|------|------|------|
| S级 | 8项全部通过（S1-S8） | 直接进入下一环节 |
| A级 | 7项通过 + 1项警告 | 修复后复查 |
| B级 | 5-6项通过 + 2-3项警告 或 有1-2项不通过 | 返回修改 |
| 不合格 | >=3项不通过 | 打回重做 |
