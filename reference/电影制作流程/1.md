# 完整电影制作全流程（从0到上映，通俗易懂版）
## 一、前期筹备（定方向·最核心）
1. **创意立项**
    出故事灵感、确定题材、风格、受众、影片时长、预算
2. **剧本创作**
    故事大纲→人物小传→分集梗概→完整剧本→反复打磨定稿
3. **组建主创团队**
    导演、编剧、制片人、摄影指导、灯光、美术、录音、剪辑、服化道
4. **勘景选景**
    实地踩点确定拍摄场地、室内外景、实景/棚拍
5. **选角定演员**
    试戏、定主角配角、群演、确定档期
6. **制作分镜脚本**
    导演画分镜，确定每一幕镜头角度、运镜、画面构图
7. **筹备道具服装化妆**
    置景搭建、服装定制、道具采购、造型定妆

## 二、中期拍摄（实拍执行）
1. **开机仪式** 正式开机
2. **现场拍摄流程**
    场务布置场地→灯光布光→机位架设→演员走位彩排→正式开拍
3. **拍摄顺序**
    不分剧情顺序，**同场景集中拍完**，节省时间成本
4. **现场把控**
    导演控情绪节奏、摄影控画面、收音同步收录现场台词
5. **每日收工**
    整理素材、核对当日戏份是否拍完、复盘次日拍摄计划

## 三、后期制作（成片灵魂）
1. **素材整理**
    筛选可用原片、剔除废片、分类归档
2. **粗剪**
    按剧本顺序拼接画面，理顺完整剧情，定整体时长
3. **精剪**
    调整节奏、删减多余戏份、卡点情绪、优化转场
4. **调色**
    定影片整体色调、氛围感、明暗质感、统一画面风格
5. **音效配音**
    现场收音降噪、补配台词、加环境音、动作音效
6. **配乐BGM**
    搭配主题曲、插曲、情绪纯音乐，烘托氛围
7. **特效包装**
    加字幕、片头片尾、字幕条、画面特效、打斗/奇幻特效
8. **审片修改**
    内部观影提意见，反复修改漏洞、节奏、穿帮镜头

## 四、终审发行（收尾上线）
1. **送审备案** 完成影视审核、取得播放资质
2. **定版本** 院线版、网络版、精简版
3. **宣发推广**
    制作预告片、海报、短视频引流、媒体宣传
4. **上线播出**
    院线排片、平台上线、全网发行
5. **数据复盘** 票房/播放量统计、口碑总结

# 极简4大步速记
**前期策划写剧本→中期实景拍摄→后期剪辑调色配音→审核宣发上映**
