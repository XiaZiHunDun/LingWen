电影制作是一套高度工业化的协作流程，通常按**前期筹备（Pre-production）、拍摄制作（Production）、后期制作（Post-production）、发行上映（Distribution）**四大阶段推进。以下是每个阶段的方法论与核心任务：

---

## 一、前期筹备（Pre-production）：把创意转化为可执行计划

### 1. 剧本锁定与法务
- **剧本定稿（Locked Script）**：导演、制片、编剧三方确认最终拍摄剧本，后续修改需走正式变更流程。
- **版权清理**：音乐、品牌、地点、人物原型等法律风险评估，购买必要授权。

### 2. 预算与融资
- **分账预算表（Top Sheet → Line Item Budget）**：按部门拆分（演员、剧组、设备、后期、应急金通常占10-15%）。
- **融资闭环**：确定投资方、政府补贴、预售版权、联合出品等资金到位时间。

### 3. 核心团队组建
- **关键岗位到位**：导演、制片人、摄影指导（DP）、美术指导、录音指导、剪辑指导（部分项目提前进组）。
- **选角（Casting）**：发布角色公告、试镜、回叫（Callback）、化学反应测试（Chemistry Read）、签约。

### 4. 视觉化筹备
- **堪景（Location Scouting）**：实地勘察光线、声音、电力、交通、停车、食宿，拍摄许可谈判。
- **故事板（Storyboard）与动态预演（Previz）**：关键动作戏/特效戏用3D预演确定机位和节奏。
- **分镜脚本（Shot List）**：DP与导演逐场拆解机位、镜头规格、运动方式。
- **美术设计**：概念图、场景搭建图、服装造型定妆、道具清单。

### 5. 制片计划
- **拍摄日程表（Shooting Schedule）**：按场景集中度排期（减少转场成本），标注日戏/夜戏、内外景。
- **通告单（Call Sheet）**：每日下发，精确到每个演员的到场时间、化妆时间、拍摄顺序。
- **设备清单**：摄影机、镜头组、灯光、移动设备（轨道/摇臂/无人机/稳定器）、录音设备。
- **安全评估**：特技、烟火、水下、高空等特种拍摄的保险与应急预案。

---

## 二、拍摄制作（Production）：按计划高效执行

拍摄现场遵循**"导演中心制"**，但由制片主任（UPM/Line Producer）控制进度与预算。

### 1. 每日工作流
- **早晨**：剧组按通告单到场，化妆/服装/置景同步进行。
- **走位彩排（Blocking）**：导演带演员走位置，DP确定机位和灯光方案。
- **正式拍摄**：按镜头号逐一拍摄，通常每个镜头拍2-5条（Take），导演喊"Cut"后确认是否通过（Print）。
- **转场**：完成一个场景后，迅速撤景、转场、重新布光。

### 2. 核心部门协作
| 部门 | 核心职责 |
|------|----------|
| **导演组** | 表演指导、场面调度、整体艺术把控 |
| **摄影组** | DP掌机/指导，焦点员、摄影助理、移动组执行 |
| **灯光组** | 灯光师（Gaffer）按DP意图布光，电工（Best Boy）配电 |
| **美术组** | 置景、道具、服装、化妆现场维护，连续性检查 |
| **录音组** | 吊杆话筒员（Boom Operator）、现场混音师，确保对白清晰 |
| **场记（Script Supervisor）** | 记录每场戏的镜头号、Take号、时长、服装道具连续性、台词差异，是后期剪辑的"活地图" |
| **DIT（数字影像工程师）** | 现场调色预览、素材备份（双备份原则）、元数据管理、转码代理文件（Proxy）供剪辑 |

### 3. 现场管理原则
- **连续性（Continuity）**：服装、道具、妆容、演员位置、镜头方向在多条Take和多日拍摄中保持一致。
- **轴线原则（180° Rule）**：对话戏保持空间逻辑，避免越轴导致观众迷惑。
- **抢天光/抢夜戏**：外景高度依赖自然光，必须严格按通告单时间执行。
- **安全红线**：特技、动物、儿童、烟火需额外安全员，超时工作（尤其是夜戏）需遵守工会规定。

## 三、后期制作（Post-production）：从素材到成片

### 1. 剪辑阶段（Editing）
- **素材整理**：DIT移交原始素材（Camera Original），剪辑助理同步声音、打板合板（Sync）。
- **粗剪（Assembly Cut）**：按剧本顺序拼接，通常比成片长30-50%，验证叙事逻辑。
- **精剪（Director's Cut / Fine Cut）**：导演与剪辑师压缩节奏、调整场次顺序、删减冗余，确定片长。
- **锁定画面（Picture Lock）**：剪辑定稿后不再改动时长，进入声音和特效阶段。

### 2. 声音制作
- **对白编辑**：清理杂音、补录对白（ADR, Automated Dialogue Replacement）。
- **拟音（Foley）**：脚步、衣物摩擦、道具碰撞等现场缺失的声音表演录制。
- **环境音/音效库**：构建空间感和动作音效。
- **配乐（Scoring）**：作曲家根据锁定画面写谱，乐团/电子制作录音。
- **混音（Mixing）**：对白、音乐、音效三条音轨分层平衡，输出5.1/7.1/Dolby Atmos等格式。

### 3. 视觉特效（VFX）
- **镜头拆分（VFX Breakdown）**：前期确定需特效的镜头，拍摄时做绿幕/跟踪点/参考素材。
- **特效制作**：概念设计、建模、绑定、动画、合成，按难度分A/B/C级管理。
- **数字绘景（Matte Painting）**：扩展场景边界。

### 4. 调色与完成片
- **调色（Color Grading）**：调色师在DaVinci Resolve等软件中统一影调、匹配镜头、建立视觉风格。
- **字幕与片头片尾**：演职员表、版权信息、合作方Logo。
- **母带输出**：生成DCP（数字影院包）用于院线放映，或各流媒体平台要求的编码规格。

## 四、发行与上映（Distribution）：从作品到商品

### 1. 电影节策略
- 选择A类（戛纳、威尼斯、柏林）或类型电影节首映，争取口碑和发行商关注。

### 2. 宣发物料
- 预告片、海报、剧照、幕后纪录片、社交媒体 campaign、首映礼。

### 3. 发行渠道
- **院线发行**：与发行商谈分账比例、排片量、密钥期限。
- **流媒体/电视**：Netflix、Amazon、国内优爱腾等平台版权销售。
- **海外销售**：电影节市场（如戛纳电影市场）或海外发行代理。

### 4. 回收与结算
- 票房分账、版权收入、衍生品、政府奖励，最终与投资方结算。

## 五、关键方法论总结

| 原则 | 说明 |
|------|------|
| **前置决策** | 后期70%的问题源于前期准备不足，宁可前期多花一周，不拍废镜头 |
| **双备份原则** | 素材现场双备份，云端/硬盘分离存储，避免拍摄事故 |
| **每日审片（Dailies）** | 导演、DP每日回看前日素材，及时发现技术/表演问题 |
| **制片人是刹车片** | 导演负责艺术，制片人负责预算和进度，二者制衡 |
| **锁定画面后再做声音/特效** | 剪辑未定稿前，任何声音和特效投入都可能白费 |
| **留足应急预算** | 10-15%不可预见费是行业铁律 |

## 六、不同规模项目的流程差异

- **大制片厂电影**：流程严格分阶段，每个节点有"绿灯会议"（Greenlight），部门齐全，周期2-4年。
- **独立电影**：前期与后期可能重叠（如边拍边剪），团队精简，一人多岗，预算驱动创意。
- **短片/学生作品**：可压缩前期流程，但拍摄期仍需专业分工，后期是提升质感的关键。
