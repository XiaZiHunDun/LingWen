# 动画完整制作全流程（零基础易懂·2D/3D通用）
## 一、前期策划（定方向）
1. **创意立项**
确定题材、风格、时长、受众、故事内核、制作预算
2. **故事设定**
世界观、背景设定、时代、剧情走向
3. **人物设定**
人设图、三视图、表情集、动作习惯、性格文案
4. **场景设定**
场景原画、氛围图、建筑布局、色调风格
5. **编写剧本**
故事大纲→分集梗概→完整动画台词剧本
6. **制作文字分镜**
用文字写清：镜头、画面内容、台词、时长、情绪

## 二、中期美术&动态筹备
1. **绘制画面分镜**
画出每一个镜头构图、运镜、机位、转场，确定镜头顺序
2. **制作动态故事板（动分）**
简单拼画面+卡点时长，定整片节奏
3. **配音预配**
先录台词初稿，按声音卡动作时长（动画必走）
4. **原画设计**
- 2D：关键动作原画、动作规律、透视、肢体动态
- 3D：模型建模、UV拆分、贴图上色、骨骼绑定

## 三、正式动画制作（核心干活）
### 2D动画流程
1. **加中间帧** 补齐关键动作，让动作流畅
2. **动画分层** 人物、前景、背景、特效分层
3. **上色描边** 线稿定稿、平涂上色、阴影明暗
4. **动态合成** 镜头推拉、平移、旋转、镜头切换

### 3D动画流程
1. **K帧调动作** 骨骼摆姿势、调整走路/打斗/表情动作
2. **打灯光** 主光、辅光、氛围光，塑造立体感
3. **搭建镜头** 架设摄影机，做运镜、推拉摇移
4. **渲染出片** 批量渲染每一帧画面

## 四、后期合成（成片收尾）
1. **剪辑拼接** 按分镜顺序整合所有动画片段
2. **音效处理**
环境音、动作音效、打斗音、脚步声、风声
3. **配乐搭配**
匹配剧情情绪插入BGM、主题曲、插曲
4. **字幕包装**
加台词字幕、片头片尾、LOGO、字幕特效
5. **画面调色**
统一色调、氛围感、明暗、滤镜风格
6. **特效添加**
光影、粒子、火焰、水雾、闪电、法术特效

## 五、终审输出
1. 全片自查：动作卡顿、穿帮、时长偏差、口型对不上
2. 修改调整，反复审片
3. 导出高清成片，定格式发布

# 最简五步速记
**前期剧本人设→分镜规划→原画/建模做动作→剪辑音效配乐→调色特效出成片**

# 三类动画区别
1. **手绘2D**：重原画、线条、逐帧动作，最费人工
2. **二维矢量动画**：简易MG动画，快节奏短视频常用
3. **三维动画**：重建模绑定渲染，画面立体，工期最长

# 行业通用工时参考
- 1分钟普通动画：3-7天
- 1分钟精细动画：10-20天
