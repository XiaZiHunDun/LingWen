动画制作是一套**高度依赖前期设计**的工业化流程，核心原则是"**前期决定后期，设计决定成本**"。与真人电影不同，动画的每一个像素都是"无中生有"，因此前期美术锁定和中期的资产/作画管理尤为关键。

以下是按**前期策划、中期制作、后期合成**三大阶段拆解的方法流程：

---

## 一、前期策划（Pre-production）：锁定80%的最终效果

### 1. 创意与剧本
- **世界观 Bible（设定集）**：动画必须比真人电影更早建立完整的视觉圣经，包含时代背景、建筑风格、色彩规则、生物/机械设计逻辑。这是所有部门的工作宪章。
- **剧本特殊性**：动画剧本需标注"视觉奇观点"（如魔法释放、变形、大规模破坏），这些是预算消耗点，需在剧本阶段与制片协商。

### 2. 美术设计
- **角色设计**：三视图（正面/侧面/背面）、表情谱、口型表、配色方案、材质指定。
- **场景设计**：关键场景概念图、气氛图（Color Script）、场景布局图（Layout Guide）。
- **道具/载具设计**：功能性道具的拆解图，确保中期建模/作画有据可依。
- **色彩指定（Color Design）**：主色调板、角色与场景的配色关系、情绪色温变化（如悲伤场景降饱和）。

### 3. 分镜与动态预演
- **故事板（Storyboard）**：按镜头绘制，标注机位运动、角色走位、特效提示、时间节奏。
- **动态分镜（Animatic）**：将静态故事板配上临时音轨（Temp Track）和简易动画，输出为视频。这是动画的"第一版粗剪"，用于验证节奏和片长，**此阶段锁定后尽量不推翻**。
- **Layout（构图设计）**：在故事板基础上精确设计每一帧的景别、透视、角色与背景的空间关系。2D动画中Layout是原画师的施工蓝图；3D动画中Layout是摄像机的初始机位和角色走位参考。

---

## 二、中期制作（Production）：资产生产与表演

中期是动画与真人电影差异最大的阶段，需按**2D动画**和**3D动画**分别说明：

### A. 2D动画（传统/无纸）流程

| 环节 | 核心任务 | 交付物 |
|------|----------|--------|
| **设计稿** | 在Layout基础上细化角色动作关键姿态、背景线稿 | 精确定位的角色+背景线稿 |
| **原画（Key Animation）** | 绘制动作的关键帧（Key Pose），标注时间轨（Exposure Sheet / X-Sheet） | 关键帧画稿+律表 |
| **中间画（In-between）** | 按原画指示补全过渡帧，使动作流畅 | 完整动画序列 |
| **扫描/线摄** | 纸质稿扫描或数字绘制直接导入 | 数字线稿 |
| **上色（Paint）** | 按色彩指定填充色块，现代多用无纸流程（TVPaint、Retas、Toon Boom） | 彩色序列帧 |
| **背景（Background）** | 美术组绘制与角色分层独立的背景层 | 高分辨率背景图 |

**关键控制点**：
- **律表（X-Sheet）**：原画师的时间圣经，精确标注每一帧的作画内容、对口型、特效提示，是连接原画、中间画、上色、摄影的枢纽。
- **一拍一/一拍二/一拍三**：每秒24帧中，每张画稿停留1/2/3帧，决定动作流畅度与成本（一拍一最烧钱，常用于高速动作；一拍三常用于日常对话）。

### B. 3D动画（CGI）流程

| 环节 | 核心任务 | 交付物 |
|------|----------|--------|
| **建模（Modeling）** | 按三视图建角色/场景/道具的3D模型 | 多边形模型（高模/低模） |
| **绑定（Rigging）** | 为模型建立骨骼系统、控制器、表情Blendshape、布料/毛发动力学系统 | 可操控的绑定文件 |
| **材质贴图（Surfacing）** | UV展开、绘制纹理、调整材质属性（金属度/粗糙度/次表面散射） | 材质球+贴图集 |
| **Layout（3D）** | 在三维场景中摆放角色、摄像机、初步走位 | 带摄像机的3D场景文件 |
| **动画Blocking** | 像2D原画一样，先摆关键Pose（Stepped模式），验证表演和节奏 | 关键帧动画 |
| **动画Spline/Polish** | 将关键帧过渡为曲线运动，细化手指、表情、跟随动作（Follow-through） | 最终动画曲线 |
| **模拟解算（Simulation）** | 布料、毛发、流体、刚体破碎等物理模拟 | 缓存文件（Alembic/USD） |
| **灯光（Lighting）** | 按气氛图打主光/补光/轮廓光，匹配场景影调 | 渲染层文件 |

**关键控制点**：
- **资产审核（Asset Approval）**：模型、绑定、材质必须通过导演和各部门负责人签字，才能进入动画和灯光环节。一旦进入下游，修改成本指数级上升。
- **USD/ABC管线**：大型项目使用Universal Scene Description或Alembic作为跨软件数据交换格式，确保Maya（动画）、Houdini（特效）、Katana/Clarisse（灯光）无缝衔接。

---

## 三、后期合成与输出（Post-production）：把所有元素熔为一炉

### 1. 2D动画后期
- **摄影（Compositing）**：将角色层、背景层、特效层按律表合成，添加模糊、景深、光效、粒子。传统赛璐珞时代由摄影台完成，现代由After Effects、TVPaint等软件完成。
- **特效（SFX）**：手绘特效（爆炸、魔法、烟雾）通常由专门特效师完成，风格需与角色作画统一。
- **输出**：按广播规格（如1920×1080）或电影规格（如2K/4K）输出序列帧。

### 2. 3D动画后期
- **渲染（Rendering）**：按灯光层输出多通道（Beauty/Diffuse/Specular/SSS/Depth/AO等），常用RenderMan、Arnold、V-Ray、Redshift。
- **合成（Compositing）**：在Nuke、After Effects中合并渲染层，调色、加景深、运动模糊、体积光、大气雾。
- **剪辑（Editing）**：将合成后的镜头按动态分镜的节奏精剪，调整转场。
- **声音与配乐**：同真人电影流程（ADR、拟音、配乐、混音），但动画的ADR比例更高（因为前期录音的临时音轨可能被替换）。

### 3. 最终母带
- **色彩校正（Color Correction）**：统一全片色调，匹配不同渲染农场的输出差异。
- **字幕与片尾**：动画片的片尾通常更长（原画师名单按镜头列出）。
- **DCP/流媒体输出**：院线动画需DCP包，流媒体需按平台要求编码（如Netflix的IMF包）。

## 四、不同类型动画的流程差异

| 类型 | 核心差异 |
|------|----------|
| **院线2D动画（如吉卜力/新海诚）** | 作画密度极高，一拍一比例大，背景精细如油画，周期3-5年 |
| **TV系列2D动画** | 分镜→Layout→原画→中间画并行推进，多集同时制作，作画监督统一画风，周期紧张时外包（中割/背景海外发包） |
| **3D动画电影（如皮克斯/迪士尼）** | 资产复用率高，但单镜头渲染成本高，灯光和毛发是瓶颈，周期4-6年 |
| **3D TV动画/番剧** | 模型和绑定标准化，动作捕捉（Mocap）辅助，渲染采用游戏引擎实时管线（Unreal Engine）以提速 |
| **定格动画（Stop Motion）** | 前期需实体偶型制作、场景微缩模型；中期按帧拍摄，日均产量极低（2-5秒/天），无"中间画"概念 |
| **动态图形（Motion Graphics）** | 流程极短，无角色绑定，直接在After Effects/Cinema 4D中设计动画，周期以天/周计 |

## 五、动画制作的关键方法论

| 原则 | 说明 |
|------|------|
| **前期锁定是生命线** | 动画的修改成本远高于真人电影。动态分镜（Animatic）阶段锁定节奏，Layout阶段锁定构图，资产审核后不改设计 |
| **并行管线管理** | 2D动画中，第1集进入上色时，第3集可能还在分镜；3D动画中，灯光与动画可部分并行（Previz灯光） |
| **作画监督/动画总监制** | 2D动画的灵魂岗位，负责统一所有原画师的风格，修正角色走形（Off-model） |
| **渲染农场管理** | 3D动画的渲染是算力密集型任务，需提前测试单帧渲染时长，合理分配农场资源 |
| **版本控制** | 动画文件（.ma/.blend/.tvp）迭代极快，必须使用Perforce、ShotGrid等版本管理系统 |

## 七、与真人电影的核心对比

| 维度 | 真人电影 | 动画 |
|------|----------|------|
| **表演来源** | 演员现场发挥 | 动画师逐帧/逐曲线创造，或动作捕捉采集 |
| **场景获取** | 实景拍摄或搭景 | 完全从无到有建造（2D绘制/3D建模） |
| **修改成本** | 可补拍、可剪辑挽救 | 修改意味着重画/重渲染，成本极高 |
| **时间弹性** | 实拍受天气/档期限制 | 不受自然条件限制，但受算力/人力限制 |
| **重复利用** | 素材一次性 | 模型/角色库可跨项目复用（如TV动画系列） |

---

**核心心法**：动画是**"用时间换控制"**的艺术。真人电影的惊喜常来自现场的"意外"，而动画的震撼则来自**前期毫厘不爽的设计**和**中期 millions of decisions 的精确执行**。
