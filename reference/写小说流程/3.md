写小说可以拆成一条清晰的“流水线”：  
**灵感 → 构思 → 设定 → 大纲 → 初稿 → 系统性修改 → 定稿/发布**。  
你不必每一步都做得很重，但最好心里有这个链条，才不会写着写着就崩。
---
## 一、整体流程一图看懂
```mermaid
flowchart LR
  A[1. 灵感与点子] --> B[2. 构思与核心三要素]
  B --> C[3. 世界与人物设定]
  C --> D[4. 故事结构/大纲]
  D --> E[5. 写初稿]
  E --> F[6. 大结构修改]
  F --> G[7. 行文与细节润色]
  G --> H[8. 定稿/连载/发布]
```
下面按阶段讲，每一步给你可操作的做法和避坑点。
---
## 二、阶段 1：灵感与点子——从“想写”到“值得写”
目标：不是所有点子都值得写成长篇，先筛选出**有故事**的那个。
1. 记录灵感  
   - 随时记：一句话设定、一个人物、一个名场面、一句对白都行。  
   - 很多写作教程都强调：先有零散灵感，再筛选、组合成故事核心。
2. 筛选点子的标准（任选其一即可动笔）  
   - 有强烈冲突：人vs人、人vs社会、人vs命运、人vs自己。  
   - 有“反差”：身份与性格、目标与阻碍、表面与真相。  
   - 有“情绪价值”：能让你自己都生气、心疼、兴奋、害怕。
3. 从点子到“一句话故事”  
   例：  
   - 一个懦弱的社畜，被卷入杀人游戏，必须在三天内找出真凶，否则自己顶罪。  
   这样一句话里，就有：**主角 + 目标 + 阻碍 + 倒计时**，后面所有设定都围绕它展开。
---
## 三、阶段 2：构思——确定三大核心（主角、目标、阻碍）
很多教程都会先让你定三件事：**主角、核心目标、最大阻碍**。
1. 主角一句话  
   身份 + 性格 + 执念/短板  
   例：  
   - 退伍军人，创伤后应激障碍，极度害怕被抛弃。  
   - 乡村医生，固执善良，只要认定救谁就死磕到底。
2. 核心目标（只能一个主目标）  
   - 要具体：不是“变强”，而是“在比赛前三个月练成绝招赢得冠军”。  
   - 要贯穿全书：所有情节都为这个目标服务。
3. 最大阻碍  
   - 反派 / 对手组织  
   - 自身性格缺陷（懦弱、偏执、贪婪）  
   - 社会/环境限制（制度、阶级、灾难）  
核心公式：  
**有缺陷的主角 + 清晰目标 + 层层阻碍 = 故事骨架**
---
## 四、阶段 3：设定——只做“够用”的设定，别堆世界观
很多新手一上来就写几十页设定表，很容易写崩。建议只做“极简设定”：
1. 世界/背景（只记关键）  
   - 时代：现代 / 古代 / 未来 / 架空  
   - 核心规则：有没有魔法/科技/异能/制度限制  
   - 故事主要发生地：1–2 个重点场景即可
2. 主角设定  
   - 年龄、职业、身份  
   - 性格关键词（2–3 个）  
   - 软肋/秘密：他最在乎什么、最怕什么、隐藏什么
3. 关键配角  
   - 1 个帮手型  
   - 1 个对手/反派型  
   - 暂时不要超过 5 个主要人物，否则自己先乱。
4. 视角与基调  
   - 视角：第一人称 / 第三人称有限 / 全知  
   - 基调：轻松搞笑 / 压抑现实 / 热血爽文 / 治愈
---
## 五、阶段 4：大纲与结构——先搭骨架，再添肉
很多写作教程都会强调：**先有结构，再写初稿**，能大幅减少烂尾。
### 1. 选一个简单结构
新手推荐**三幕式 / 四段式**：
1. 开端（日常被打破）  
   - 主角日常 → 突发事件 → 被迫上路 / 卷入事件  
2. 发展（闯关+成长）  
   - 尝试解决问题 → 遭遇失败  
   - 遇到帮手 / 反派施压  
   - 情况越来越糟，接近低谷
3. 高潮（终极对决 / 考验）  
   - 主角直面最大阻碍  
   - 做出关键选择 / 牺牲 / 觉醒
4. 结局（收尾+蜕变）  
   - 目标达成 / 遗憾 / 留白  
   - 人物完成成长，伏笔回收
也可以用“8 节点大纲”：
1. 主角日常  
2. 突发意外  
3. 第一次尝试失败  
4. 遇帮手，反派加压  
5. 跌入最低谷  
6. 醒悟，决心反击  
7. 终极对决  
8. 结局收尾，人物蜕变  
### 2. 写大纲的实用方式
- **章纲**：每章 1–2 句话，只写“这章发生什么事”。  
- **场景表**：每个场景写：  
  - 视点人物  
  - 场景目标  
  - 冲突/阻碍  
  - 结局（更好/更糟）
---
## 六、阶段 5：写初稿——先完成，再完美
几乎所有专业作者都会说：**第一稿只求写完，不求写好**。
1. 心态  
   - 允许自己写“烂”：文笔差、逻辑漏洞都先不管。  
   - 卡住就跳：写不下去的情节，先空着或简单标注，继续写后面。
2. 节奏  
   - 设定一个可坚持的日目标：如 500–1000 字，比“爆更 5000 然后弃文”靠谱。  
   - 尽量在同一个时间段写，形成习惯。
3. 写作顺序  
   - 建议按大纲顺序写，别东一榔头西一棒子，否则结构容易乱。  
   - 如果灵感特别强烈，也可以先写最兴奋的场景，再前后补。
---
## 七、阶段 6：大结构修改——从“乱麻”到“有型”
很多写作教程会把修改拆成：**先大结构，再小细节**。
1. 第一轮：看“故事”  
   - 主角目标是否清晰？  
   - 情节是否推进目标？有无“废章节”？  
   - 人物行为是否符合性格？有无突然“降智 / 变性格”？  
   - 有无重大逻辑漏洞 / 时间线错误？
2. 常用工具  
   - **反向大纲**：写完后，每章/每场景自己总结“这章在干嘛”，再和原大纲对比，看跑偏多少。  
   - **角色弧线检查**：主角、反派、重要配角是否有成长/变化？  
   - **删减与合并**：能删的情节果断删，人物能合并就合并。
3. 不要在这一轮纠结文笔  
   - 只解决“故事好不好看、合不合理”，  
   - 行文粗糙、对话尴尬，都留给下一轮。
---
## 八、阶段 7：行文与细节润色——让小说“好看”
1. 第二轮：看“读者体验”  
   - 开头是否足够抓人？前 3 章/ 前 1 万字决定读者去留。  
   - 节奏：是否有长段无聊说教、无冲突日常？  
   - 情绪：有没有“名场面”“高光时刻”“虐点/爽点”？
2. 第三轮：看“语言与细节”  
   - 场景：环境、五感细节是否足够？  
   - 对白：是否符合人物身份？是否太“书面语”？  
   - 心理：有没有过多直白“我觉得”，可以转为动作/细节暗示。
3. 第四轮：校对  
   - 错别字、标点、格式、时间线/人名硬伤。  
   - 最好能打印出来读，或让朗读软件读给你听，耳朵对语病很敏感。
---
## 九、阶段 8：定稿与发布——写完不是结束
1. 定稿前  
   - 再通读一遍，只做“最后的小修小补”，不要再大改结构。  
   - 确认：人名、地名、专有名词统一；章节编号、分段完整。
2. 发布与连载  
   - 网文平台：按平台风格拆章、控制每章字数与断章点。  
   - 实体/自出版：排版、封面、简介、样章等，又是一套工程。
3. 心态  
   - 接受“作品永远可以再改”，但**必须学会在某一个版本停下来**。  
   - 把更多经验留给下一本书，比在一本上死磕更健康。
---
## 十、给你一个可直接照做的“新手流程”
如果你现在就想开始，可以按这个极简版走一遍：
1. 今天：10 分钟  
   - 写下：主角 + 目标 + 最大阻碍（各一句话）。  
2. 明天：15 分钟  
   - 定题材与文风。  
   - 用“三段式/四段式”写一个极简大纲（开端–发展–高潮–结局）。  
3. 后天起：每天 500–800 字  
   - 按大纲写第一章，只求写完，不管文笔。  
4. 写完前 3 章：  
   - 检查：主角目标是否清晰？冲突是否足够？  
5. 写完全书：  
   - 做一次“大结构修改” + 一次“润色修改”，就可以先当“完成版”。
