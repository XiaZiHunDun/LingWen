# 写小说完整流程+实操方法（新手直接照做）
## 一、前期定盘（打底，不写就动笔必崩）
1. **定题材+赛道**
   确定类型：玄幻/都市/言情/悬疑/科幻等，锁定受众、文风、爽点风格。
2. **立核心三要素**
   - 主角：性格、目标、执念、弱点
   - 核心冲突：人和人/人和世界/自我对抗
   - 故事主旨：想表达什么（热血、救赎、逆袭、治愈）
3. **提炼一句话大纲**
   例：落魄少年意外获得奇遇，一路逆袭登顶，守护身边之人。
4. **设定世界观**
   势力、等级、规则、地域、战力体系、货币、习俗，精简够用即可，别堆废设定。

## 二、人物塑造（故事灵魂）
1. 主角：人设+成长线+前后反差
2. 配角：盟友、对手、反派、路人，每人有动机
3. 反派：不纯粹坏，有立场、有苦衷、有目标
4. 人物关系网：梳理爱恨、敌对、师徒、亲友

## 三、搭建大纲（最关键）
### 1. 经典三段式结构
- **开端（1-20%）**：出场、背景、困境、获得金手指/机遇、立下目标
- **发展（20%-70%）**：打怪升级、遇贵人、结仇敌、闯关历练、积累实力、埋下伏笔
- **结局（70%-100%）**：终极矛盾爆发、决战、恩怨了结、圆满/遗憾收尾

### 2. 章节细纲写法
每卷定**卷名+核心事件**，每章写：
本章目标+出场人物+剧情走向+情绪落点+留钩子

### 3. 必备写作节奏
- 3章一小爽，10章一大转折
- 张弛有度：打斗/冲突→日常/铺垫→再爆发
- 全程埋伏笔，前后呼应

## 四、开篇写法（决定留人）
1. 开篇前三原则
   - 快速出场主角
   - 立刻抛出困境/危机
   - 直接给希望/转机
2. 开篇避雷
   不堆砌设定、不长篇回忆、不多人乱出场、节奏别慢

## 五、正文写作实操方法
1. **视角统一**
   首选**第三人称限知视角**（跟着主角视角走），少乱切视角。
2. **叙事公式**
   环境铺垫→人物动作→对话互动→心理活动→事件推进→留下悬念
3. **对话写法**
   人话自然，符合身份，**对话推动剧情，不闲聊**
4. **描写分寸**
   外貌极简，动作精准，心理点到为止，侧重剧情流动
5. **情绪带动**
   主角情绪带动读者情绪：委屈、愤怒、激动、震惊、释然

## 六、伏笔与埋线技巧
1. 前期随手埋小细节，中后期回收
2. 人名、物品、一句随口话都可当伏笔
3. 主线不动，支线穿插不跑偏

## 七、日常码字流程（高效稳更）
1. 先看细纲，确定本章写什么
2. 先写剧情主干，再补细节描写
3. 写完通读一遍：删废话、顺逻辑、改生硬对话
4. 固定字数，稳定节奏

## 八、中期卡文解决办法
1. 回看大纲，确定当前阶段目标
2. 给主角制造麻烦、敌人、意外变故
3. 让配角搞事，推动剧情
4. 暂停细写，先把剧情骨架写完

## 九、收尾完结流程
1. 收拢所有伏笔、恩怨、支线
2. 完成主角最初目标与成长蜕变
3. 交代主要人物最终归宿
4. 结尾留余味，不仓促烂尾

## 十、进阶提升写法
1. 多看同品类爆款，拆解剧情节奏
2. 积累场景词、动作词、情绪词库
3. 控制水字数，每一段都有用
4. 统一文风，全程风格不变
