# 写剧本完整流程+实操方法（通用影视/短剧/短视频剧本通用）
## 一、前期定方向（打底）
1. **确定题材+受众**
    定类型：甜宠、悬疑、现实、古风、搞笑、恐怖
    定人群：学生、上班族、中老年，决定台词风格
2. **提炼核心主题**
    一句话主旨（友情/成长/逆袭/救赎），全程不跑偏
3. **写一句话故事梗概**
    谁+遇到什么麻烦+做了什么+最后结果

## 二、搭建故事骨架（最重要）
### 1. 人物设定
- 主角：身份、性格、执念、弱点、目标
- 配角：作用（助攻/阻碍/反衬）
- 反派：动机合理，不纯粹坏
### 2. 经典三段式结构
1. **开端（1/4）**：出场、背景、立下目标、出现冲突苗头
2. **中段（2/4）**：矛盾升级、接连受挫、陷入低谷
3. **结局（1/4）**：决战/破局、解决矛盾、收尾升华

### 3. 五幕剧标准流程（专业编剧用）
1. 铺垫：介绍人物日常
2. 冲突触发：意外事件打破平静
3. 对抗升级：层层阻碍，压力拉满
4. 最低谷：主角濒临放弃
5. 结局反转+圆满/留白

## 三、细化剧情（分场大纲）
1. 拆分**场次**：按场景划分（家里、街头、办公室）
2. 每场写清4要素
    - 场景：时间+地点
    - 人物：出场角色
    - 事件：本场发生什么
    - 作用：推动剧情/塑造人物/埋伏笔
3. 埋伏笔+设反转
    前面留细节，后面圆回来

## 四、正式写剧本正文（格式标准）
### 标准剧本格式
1. **场头**：外景/内景+时间+地点
2. **动作描写**：人物神态、动作、环境（简洁不啰嗦）
3. **人物名**：居中加粗
4. **台词**：口语自然，贴合人设，少书面语
5. **情绪标注**：（低声、愤怒、苦笑）

### 写台词技巧
- 人话优先，拒绝华丽空话
- 每人说话有专属语气
- 对话藏情绪、藏目的、藏秘密

## 五、初稿修改打磨
1. 删废戏：无关剧情、凑时长片段全删
2. 提速节奏：短剧30秒一个小冲突，长剧5分钟一个看点
3. 逻辑自查：动机合理、行为合理、结局闭环
4. 精简台词：一句能说清不说三句

## 六、最终定稿流程
1. 通读顺一遍，卡壳处改写
2. 标注镜头建议（远近景、特写、转场）
3. 控制时长：**1分钟剧本≈220字**
4. 核对主题，确保不跑偏
