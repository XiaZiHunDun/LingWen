## 四、高频问题模式

### 1. 多人协作问题
```
问题类型: 章节编号/文件名不匹配
原因: 多作家并行写作时缺乏统一的编号校验
预防: 每个批次写作前校验编号连续性，写完后自动检查
```

### 2. 角色状态漂移
```
问题类型: 小九/星月等核心角色形态/性别不一致
原因: 长篇写作中角色形象在多人脑中不一致
预防: 建立角色状态表，每章写前确认，当前状态可视化
```

### 3. 伏笔断裂
```
问题类型: 伏笔埋下后无回收
原因: 缺乏全局伏笔追踪表
预防: 使用伏笔追踪表，标注伏笔章节和预期回收章节
```

### 4. 时间线矛盾
```
问题类型: 时间流速/时间点前后矛盾
原因: 设定的时间线与实际写作内容脱节
预防: 建立时间线主表，关键时间节点标注
```
