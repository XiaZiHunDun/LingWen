# 第10章 暗河

丰水解除的第七天，沈渡把复制好的录音交给林湄家属。

老人戴着老花镜，听完频道 3 里最后一句「要是暗河改道，去旧坐标找竖井」，沉默很久。

「他在井里等了你两年？」

「他在井里把坐标留下来了。」沈渡把工牌放在桌上，「测绘图也留下来了。」

老人没哭，只问：「疼吗？」

沈渡想起日志里那句「伤口不深，能处理」，没回答。

---

省水文站公告栏贴了通告：三号井区域列入禁入测绘，**井口 GPS 会随暗河掩埋而偏移**，不再按旧图定位。

沈渡最后一次调频道 3。

只有风的底噪。

他把对讲机还给老覃，锁进补给站抽屉。

「以后还进不进暗河？」老覃问。

「不进。」沈渡说，「记录送到了。」

喀斯特谷的地下风还在。

暗河在深处缓慢改道，像整片含水层在呼吸。

但频道 3 终于安静了。

沈渡拦了回城的班车，没再回头。

**本章完**
