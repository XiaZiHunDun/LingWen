# 暗河档案·第1章 丰水

水位计的第三频道在凌晨两点十七分发出了第一声杂音。

沈渡把护林摩托停在洞口外侧松软的泥地里，熄了火，却没有立刻下车。引擎的余温让排气管发出几声闷响，像某种大型动物在远处翻身。他盯着手腕上的接收器——屏幕上，第三频道的波形本该是平直的，地下水位在丰水期前夜通常不会有过激的波动，那条标定为"汊-07"的支脉早在三年前就被从监测网络中除名。

波形抖了一下。

不是仪器故障的那种抖。沈渡做了十二年暗河观测，分得清水流拍击岩壁、溶腔塌方挤压气柱、以及——某种更接近于呼吸的节律。他把增益旋钮往右拧了半格，波形变得清晰：四个波峰，间隔均匀，振幅0.3毫米，持续时间不足两秒，归于死寂。

这不是水。

他把耳机塞进耳朵，推开车门。山里的夜风带着松脂和湿泥的气味，洞口方向传来值守员老覃收音机里咿咿呀呀的越剧唱段，音量开得不大，混在虫鸣里几乎听不真切。

"沈工？"老覃从帆布躺椅里欠起身，手电筒的光柱扫过沈渡的胶鞋，又落到他脸上，"这个点儿上来……有任务？"

"看看水尺。"

老覃噢了一声，没多嘴。每年丰水期前夜他都要来一趟，算是老规矩了。三年前立的规矩——立规矩的人早去了省城，听说孩子都会跑了。沈渡从老覃身边过时，余光扫到洞口的木牌还是那块，漆面剥落得厉害，红底白字写着"丰水期 严禁入内"，落款处"林湄"两个字的笔画被雨水冲得只剩浅浅的凹痕。

他没停步。

值守棚里的铁柜没上锁。沈渡推开柜门，发出一声涩哑的金属摩擦声。他把头灯调到最低亮度，光束扫过几层隔板。第三层——他记得清楚，三年前林湄撤得仓促，这个本子应该还在原处。

一本空白观测本。A5大小，灰蓝色硬壳封面，右下角贴着手写标签："汊-07 2019.04-2021.06"。

沈渡把它抽出来。本子沉甸甸的，比空白本应有的分量重一些。他翻开第一页——空白。第二页，空白。一直翻到第三十七页，右下角有一小片淡黄色的水渍，形状不规则，像某种凝结的水珠被时间封存在纸纤维里。他用指腹摸了一下，触感微凉，带着一丝几乎察觉不到的潮润。

钟乳水。

林湄只在记录钟乳水渗漏量时才会用这种纸质观测本——市面上买不到，她自己裁的，边角没对齐，本子的厚度也因此参差不齐。沈渡继续往后翻，空白，一页接一页的空白。一直翻到封底内侧，夹着一张对折的宣纸，展开来是一幅铅笔素描——溶缝的剖面图，线条干净，标注详尽，但有几个位置被刻意涂成了黑块，看不清原本画的是什么。

他把素描折回去，重新塞进封底，把整本观测本塞进防水袋里。

耳机里，波形又动了。

这一次不是四个波峰，是三个，间隔略长，振幅却比刚才高了将近一倍。沈渡低头看向接收器屏幕——坐标定位在自动刷新，每刷新一次，数值都往同一个方向偏移：西南，斜下，深度六十一点四米。

那是汊-07的位置。

那个在监测网络中"不存在"的位置。

沈渡关掉头灯，走出值守棚。老覃还坐在躺椅里，收音机已经换了台，正在播一档深夜情感热线，女主持人用绵软的嗓音劝解一个声称被前女友下了蛊的男听众。

"老覃。"

"嗯？"

"今晚上来过别人没有？"

老覃想了想："没有。就我一个人守到天亮。"他把收音机音量调小，"出什么事了？"

"没事。"沈渡发动摩托，引擎的轰鸣盖过了他的下一句。他自己也说不清为什么要问这一句——也许只是想确认，汊-07的坐标偏移不是某种设备延迟造成的鬼影，也许只是想确认，那个节律不是从洞口附近传过去的。

摩托往山下开的时候，沈渡从后视镜里看了一眼洞口。木牌在头灯余光里晃了晃，"丰水期 严禁入内"那几个字被甩在身后，越来越小。

耳机里，第四声波形响了。

这一次是五个波峰，间隔更短，振幅更高。沈渡把车速放慢，盯着屏幕——坐标没有再偏移，锁定在那个深度，纹丝不动。

像是有什么东西在六十一点四米深的黑暗里，屏住呼吸，等着他。
