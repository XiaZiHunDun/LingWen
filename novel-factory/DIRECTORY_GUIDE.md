## 核心目录详解

### `.skills/` — Agent角色池配置

**用途**：定义44个Agent角色的配置（YAML格式）

```
.skills/
├── writer-dept/      # 作家角色池（writer-a ~ writer-j）
├── reviewer-dept/    # 审核员角色池（reviewer-a ~ reviewer-k）
├── reader-dept/      # 读者角色池（reader-a ~ reader-t）
├── inspiration-dept/ # 灵感Agent配置
├── summary-dept/     # 汇总Agent配置
├── consistency-check/ # 一致性检测配置
├── publish/          # 发布流程配置
└── _global/          # 全局配置
```

**说明**：这是实际使用的角色池配置，通过`.skills/SKILL.md`统领。02/04/05物理目录为遗留结构。
