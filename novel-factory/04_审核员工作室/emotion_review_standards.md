# S4 情感共鸣 · 审核标准细则

> **维度**: S4 情感共鸣
> **版本**: v1.0
> **适用阶段**: STEP_16-18 审核期
> **人工审核**: S4.3、S4.5 需人工判断；其余标准可程序化检测

---

## S4.1 情绪转变合理性

**标准**: 角色的情绪状态变化必须由事件逻辑驱动（每次重大情绪转变至少1个因果事件）

**检测规则**:
- 识别章节中所有重大情绪转变（如：恐惧→愤怒、悲伤→决绝、绝望→希望）
- 情绪转变前必须存在可验证的因果事件
- 因果事件类型：对话、动作、环境变化、他人行为、信息获取
- 禁止：无任何铺垫的情绪突变（如"他突然愤怒"前面没有引发愤怒的事件）

**评估等级**:
| 等级 | 要求 |
|------|------|
| S级 | 每个情绪转变都有清晰的因果事件 |
| A级 | 90%+情绪转变有因果事件 |
| B级 | 70%+情绪转变有因果事件 |
| 不合格 | 低于70%或有明显突兀的情绪跳跃 |

**示例**:
```
✅ 合例：父亲倒下（事件）→ 林夜恐惧颤抖（情绪）→ 母亲捂住他的嘴（事件）→ 林夜压抑恐惧（情绪）
❌ 违规：林夜突然愤怒（无任何引发愤怒的事件）
```

## S4.2 情感层次

**标准**: 关键场景必须呈现至少2层情感（表面情绪 + 深层真实情绪）

**检测规则**:
- 识别章节中的关键场景（冲突、高潮、转折点）
- 每个关键场景必须同时包含：
  - **表面情绪**: 可观察的外在表现（表情、动作、语言）
  - **深层情绪**: 内在真实感受（心理活动、隐藏动机、压抑情感）
- 层数统计：2层=合格，3层=优秀，1层=不合格

**评估等级**:
| 等级 | 要求 |
|------|------|
| S级 | 关键场景全部有3层以上情感 |
| A级 | 关键场景全部有2层情感 |
| B级 | 70%关键场景有2层情感 |
| 不合格 | 低于70%或关键场景只有单层情感 |

**示例**:
```
✅ 合例（2层情感）:
  表面：母亲死死抱住林夜，用手捂住他的嘴（动作）
  深层：林夜感受到母亲的手"冰凉而颤抖，像一片风中的枯叶"（真实情绪）

✅ 合例（3层情感）:
  表面：林夜想爬起来冲向门口（外在行为）
  表面：母亲"死死抱住他，用身体将他压在角落里"（外在行为）
  深层："他想爬起来冲向门口，但母亲死死抱住了他"（内心冲突）
```

## S4.3 情感共鸣触发点

**标准**: 每章必须有至少2个可识别的目标读者共情触发点

**检测方式**: 人工审核（需理解目标读者画像）

**触发点类型**:
- **牺牲型**: 为他人付出（亲子、伴侣、朋友）
- **奋斗型**: 逆境中坚持目标
- **正义型**: 面对不公挺身而出
- **成长型**: 克服恐惧或弱点
- **关系型**: 羁绊、离别、和解
- **反击型**: 被压迫后的觉醒或复仇

**评估要点**:
1. 触发点是否与目标读者画像匹配
2. 触发点是否有足够的情感铺垫
3. 触发点是否达到足够的情感强度

**示例**:
```
✅ 有效触发点：
  - 父亲用身体挡住门口与变异兽战斗 → 牺牲型
  - 母亲用生命争取时间让林夜躲藏 → 牺牲型
  - 林夜用手挖坑埋葬父母 → 奋斗型 + 亲情羁绊

❌ 无效触发点：
  - 突然的情绪爆发没有铺垫
  - 与目标读者无关的价值观触发
```

## S4.4 情绪节奏

**标准**: 不允许连续3章以上单一情绪色调（纯负面/纯正面）无变化

**检测规则**:
- 统计章节情绪色调（正面/中性/负面）
- 检查当前章节与前2章的情绪色调
- 连续3章同色调则触发预警
- 允许的情绪变化周期：至少每3章有一次情绪转变

**评估等级**:
| 等级 | 要求 |
|------|------|
| S级 | 每章都有情绪变化或每2章最多连续同色调 |
| A级 | 最多3章连续同色调 |
| B级 | 最多4章连续同色调 |
| 不合格 | 超过4章连续同色调 |

**色调定义**:
- **正面**: 希望、喜悦、温暖、成就感、爱情正面描写
- **负面**: 恐惧、悲伤、绝望、愤怒、痛苦、仇恨
- **中性**: 平静、专注、回忆、日常描写

**示例**:
```
✅ 合例（3章节奏）:
  ch001: 负面（恐惧、悲伤）→ ch002: 中性（回忆）→ ch003: 正面（决绝、复仇）

❌ 违规（连续4章同色调）:
  ch001: 负面 → ch002: 负面 → ch003: 负面 → ch004: 负面（违规）
```

## S4.5 伏笔情感回收

**标准**: 任何伏笔性情感铺垫必须在±15章内回响/回收

**检测方式**: 人工审核（需追踪伏笔系统）

**回收判定**:
- **完全回收**: 伏笔情感得到明确回应，读者情感得到满足
- **部分回收**: 伏笔有回应但不够充分
- **未回收**: 超出15章窗口或完全无回应
- **过度提前**: 情感回收发生在伏笔建立之前（时间线矛盾）

**示例**:
```
✅ 合例：
  ch001: 父亲的眼神"像是在说活下去"（伏笔）
  ch015: 林夜在废墟中想起父亲的遗言，找到了活下去的理由（回收）

❌ 违规：
  ch001: 父亲的眼神"像是在说活下去"（伏笔）
  ch060: 林夜仍未想起或回应这个伏笔（超出15章窗口）

⚠️ 注意：
  伏笔回收时间窗口 = 伏笔建立章 ± 15章
  例如 ch001 建立的伏笔必须在 ch016-ch046 之间回收
```

## 审核流程

```
1. S4.1 + S4.2 + S4.4 → 程序化检测 → 输出violations列表
2. S4.3 → 人工审核 → 审核员判断
3. S4.5 → 人工审核 → 核对伏笔追踪系统
4. 综合评级 → 输出最终审核意见
```
