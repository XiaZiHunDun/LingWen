# 小说创作术语表

---

## 结构与大纲

| 术语 | 英文 | 解释 |
|------|------|------|
| 三幕式 | Three-Act Structure | 建置(25%)→对抗(50%)→解决(25%) |
| 节拍 | Beat | 场景中的最小叙事单元（动作-反应） |
| 情节点 | Plot Point | 故事的重大转折点 |
| 诱发事件 | Inciting Incident | 打破主角日常的第一个事件 |
| 第一扇门 | First Door | 主角主动踏入冲突的节点 |
| 中点 | Midpoint | 故事50%处，重大转折 |
| 灵魂黑夜 | Dark Night of the Soul | 主角最低谷，几乎放弃 |
| 高潮 | Climax | 核心冲突爆发的最高点 |
| 反向大纲 | Reverse Outline | 写完后每章总结"这章干嘛"以检查跑偏 |

---

## 人物

| 术语 | 解释 |
|------|------|
| 人物弧光 | Character Arc，角色从开始到结束的内在变化 |
| 欲望 | Want，人物表面想要什么 |
| 需求 | Need，人物真正需要什么（两者常不同） |
| 致命弱点 | Fatal Flaw，人物性格缺陷导致的根本问题 |
| 对手/反派 | Antagonist，制造冲突的一方（不一定是坏人） |
| 镜像角色 | Mirror Character，与主角有相似处境但不同选择 |
| 功能性角色 | Utility Character，服务于情节的工具人 |

---

## 写作技巧

| 术语 | 英文 | 解释 |
|------|------|------|
| Show, Don't Tell | 展示而非讲述 | 不说"他很愤怒"，而是通过动作/细节表现 |
| Tucking | 讲述（Tell的同义词） | 解释性陈述而非展示 |
| 翻页钩子 | Page Turner | 章末留下的悬念，让读者想翻下一页 |
| 节拍 | Beat | 场景中的动作-反应单元 |
| 张弛 | Pacing | 节奏控制，紧张与舒缓的交替 |

---

## 网文特定

| 术语 | 解释 |
|------|------|
| 金手指 | 主角的特殊优势/能力（系统/奇遇/血统等） |
| 爽点 | 让读者感到"爽"的情节（打脸/逆袭/装逼成功） |
| 虐点 | 让读者心疼/难过的情节（悲剧/牺牲/误解） |
| 废柴逆袭 | 主角从弱小到强大的经典套路 |
| 退婚打脸 | 经典爽文套路（退婚→打脸） |
| 崩 | 文章出现逻辑/人物崩坏 |
| 水 | 内容注水，没有实质性情节推进 |
| 太监/烂尾 | 小说中途停止更新/草草结束 |

---

## 修改相关

| 术语 | 解释 |
|------|------|
| 宏观修改 | 大结构、情节、人物层面的修改 |
| 微观修改 | 句子、词语、标点层面的修改 |
| 轮次 | Revision Round，通常改3遍（宏观→中观→微观） |
| 冷却期 | 初稿完成后放1-2周不用，以陌生眼光审视 |

---

## 发布相关

| 术语 | 解释 |
|------|------|
| 定稿 | 最终确定的版本 |
| 连载 | 在平台持续更新 |
| 完本 | 小说完成 |
| 均订 | 平均每章订阅数（网文核心数据） |
| 追读 | 读者持续追着看 |
| 留存 | 读者从新书追到后期不弃 |
