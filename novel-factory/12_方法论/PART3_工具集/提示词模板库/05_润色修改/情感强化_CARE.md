## A - Action（行动）

### 情感强化维度

#### 1. 感官细节强化
通过身体感受传递情感：
- **外在表现**
  - 面部表情（眼、眉、嘴）
  - 肢体动作（手、脚、身体姿态）
  - 声音特征（语调、语速、呼吸）
- **内在感受**
  - 生理反应（心跳、出汗、颤抖）
  - 心理感受（思绪、回忆、联想）
  - 感知变化（时间感、空间感）

#### 2. 环境映衬
利用环境强化情感：
- **自然环境**
  - 天气对应情感（雨=悲伤，晴=喜悦）
  - 季节映照（寒冬=孤独，暖春=希望）
  - 景色与情感的对比或呼应
- **人造环境**
  - 空间大小与情感（空旷=孤独，狭小=压抑）
  - 色彩与情感（暗淡=消沉，明亮=希望）
  - 物品与记忆的关联

#### 3. 动作描写强化
通过行动展现情感：
- **主动动作**
  - 有意识的情感表达动作
  - 掩饰真实情感的动作
  - 宣泄情感的动作
- **被动反应**
  - 无法控制的生理反应
  - 下意识的微小动作
  - 身体对情感的真实反应

#### 4. 内心独白强化
直接但有技巧的情感表达：
- **意识流**
  - 碎片化的思绪
  - 回忆与现实的交织
  - 潜意识的流动
- **反思性独白**
  - 对过去的反思
  - 对当下的感受
  - 对未来的担忧/期待
- **情感判断**
  - 内心的评价和结论
  - 情感的自我认知
  - 情感的深度剖析

### 情感强化技巧

#### 具象化技巧
- 将抽象情感转化为具体感受
- 例子："悲伤"→"心里像压着一块石头"
- 例子："喜悦"→"心里开出了一朵花"

#### 对比技巧
- 情感对比（前后、内外）
- 期望与现实的对比
- 他人与自己的对比

#### 延迟技巧
- 情感积累后的爆发
- 压抑后的释放
- 小动作暗示大情感

#### 侧面技巧
- 通过他人反应展现情感
- 通过环境变化暗示情感
- 通过身体语言传递情感

### 约束条件
- 情感强度变化：{intensity_change}
- 禁止：{prohibited_elements}
  - 过度煽情
  - 直白表述（"他很悲伤"）
  - 情感与情境不匹配

## E - Example（示例）

### 悲伤场景强化示例

```markdown
[原文]
她很悲伤，眼泪流了下来。

[情感强化后]
她没有哭出声。

只是站在原地，肩膀微微颤抖。视线逐渐模糊，不是泪水模糊的，是世界本身在扭曲。她听见自己的心跳声，每一下都像锤子敲在胸腔上。

喉咙发紧，像是有人用手掐住了那里。她想说什么，但嘴唇动了动，只有无声的抽噎。

很久没有这样了。

上一次哭成这样，还是父亲下葬那天。那天的雨也很大，和今天一样。她站在雨里，想着再也不会有人在雨天给她送伞了。

膝盖突然一软，她跪在地上，双手捂住脸。

泪水终于决堤。

注解：
- 延迟爆发：先用身体反应铺垫
- 感官细节：心跳声、喉咙发紧、视线扭曲
- 回忆对比：与父亲下葬那天对比
- 层次推进：颤抖→无声抽噎→跪地→决堤
- 环境映衬：雨天与回忆的雨天呼应
```

### 爱意场景强化示例

```markdown
[原文]
他看着她，觉得她很美。

[情感强化后]
他看着她。

晨光从她身后透进来，给她的轮廓镀上一层淡金色。她还不知道他在看她，正低头整理手中的书页，手指轻轻拂过纸边，像是在抚摸什么珍贵的东西。

他的心脏突然紧缩了一下。

不是疼，是另一种说不清的感觉。像是有什么东西在胸腔里慢慢膨胀，挤占了所有空间，让他呼吸都变得困难。

他想告诉她。

告诉她每次她笑的时候，他都觉得天比平时更蓝。告诉她她身上总有淡淡的桂花香，他偷偷记了很多年。告诉她——

但他只是站在这里，看着她。

有些话，说出口就轻了。

注解：
- 通过他者视角展现美：晨光轮廓、手指动作
- 身体反应：心脏紧缩、呼吸困难
- 内心独白：想告诉却又不敢
- 隐喻暗示："说出口就轻了"
- 含蓄表达爱意，不直接说"美"或"爱"
```

### 愤怒场景强化示例

```markdown
[原文]
他很愤怒，拳头握紧了。

[情感强化后]
他的手指在发抖。

不是因为怕，是因为某种更原始的东西正从肚子里往上涌，灼热得像是喝下了融化的铁。他想松开拳头，但指节反而握得更紧，青筋在手背上暴起。

喉咙里有什么东西堵着，想要冲出来。

他想吼叫。想砸东西。想把眼前的一切都撕碎。

但他没有。

他只是站在那里，慢慢地，慢慢地，把那口气吐出来。然后他抬起头，看着面前的人，嘴角扯出一个笑。

那笑容冷得像是刀锋。

"好。"他听见自己说，"很好。"

注解：
- 身体先于意识：手指发抖、喉咙堵着
- 生理反应：胃里灼热、青筋暴起
- 压抑与释放：想吼叫但没有，最后用冷笑表达
- 内外对比：外表平静（微笑）vs内心翻涌
- 通过细节（冷笑比咆哮更可怕）展现愤怒
```

## 模板使用说明

### 必填参数
| 参数 | 说明 | 示例 |
|------|------|------|
| chapter_num | 章节号 | 42 |
| scene_emotion | 场景预设情感 | "悲伤、离别" |

| character_emotion_profile | 角色情感档案 | （角色设定） |

### 情感类型与表现
| 情感类型 | 推荐表现技巧 |
|----------|-------------|
| 悲伤 | 身体反应、回忆对比、环境映衬 |
| 喜悦 | 轻松节奏、明亮意象、肢体放松 |
| 愤怒 | 身体紧张、压抑vs爆发、微表情 |
| 恐惧 | 感官放大、时间扭曲、逃跑冲动 |
| 爱意 | 细节观察、身体反应、含蓄表达 |
| 恨意 | 冷漠vs激烈、报复冲动、心理剖析 |

### 注意事项
1. 情感强化不是堆砌形容词
2. 通过细节（身体、环境、动作）展现情感
3. 避免直白的情感表述
4. 注意情感的层次和节奏
5. 符合角色的性格和表达方式
6. 不要让情感凌驾于故事之上
