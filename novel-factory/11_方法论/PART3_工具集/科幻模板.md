# 科幻模板

> **版本**：v1.0
> **适用类型**：硬科幻 / 软科幻 / 赛博朋克 / 星际冒险
> **日期**：2026-05-20

---

## 核心驱动链

```
发现 → 探索 → 危机 → 抉择 → 突破 → 新平衡
```

科幻核心是**已知与未知的边界**，所有情节服务于对未知的探索和对人性的反思。

---

## 世界观构建模板

### 文明等级设定

```
【卡尔达肖夫指数变体】
- 0型：行星级（当前人类）
- I型：恒星级（能利用整个星系的能源）
- II型：星系级（能利用整个星系的能源）
- III型：多元宇宙级（能跨维度旅行）

【社会结构对应】
- 0型：国家/星球内部矛盾为主
- I型：太阳系内贸易/战争
- II型：银河系联邦/帝国
- III型：跨维度文明冲突
```

### 技术体系设定

```
【核心科技树】
├─ 能量：核聚变 > 正反物质湮灭 > 真空能
├─ 通讯：量子纠缠通讯（即时）/ 中微子通讯
├─ 交通：曲速引擎 > 虫洞 > 空间折叠
├─ 计算：量子计算 > 光子计算 > 意识上传
└─ 武器：定向能 > 粒子武器 > 引力武器

【技术副作用】（让科幻更有深度）
- 曲速引擎：导致空间扰动，可能引发维度崩塌
- 量子通讯：可能被窃听，无法防止
- 意识上传：上传的到底是本人还是副本？
- AI觉醒：机器人权利 vs 人类利益
```

## 人物弧光模板

### 科幻主角类型

| 类型 | 特点 | 典型困境 |
|------|------|----------|
| 探索者 | 好奇心驱动，寻找真相 | 发现真相后无法承受 |
| 幸存者 | 经历灭世，立志守护 | 过度保护导致控制 |
| 改革者 | 看清体制弊端，推动变革 | 被体制反噬 |
| 拯救者 | 被选中/被召唤拯救世界 | 代价是失去一切 |
| 叛逃者 | 从旧世界逃离，寻找新生活 | 旧世界不断追来 |

### 配角功能分配

```
【必需配角】
1. 技术专家：解释科技细节，推动解决方案
2. 反对者：提出质疑，增加戏剧张力
3. 情感锚点：让主角有人性锚点（爱人/家人/朋友）
4. 对手/敌人：推动冲突，提供危机

【可选配角】
5. 导师：传递信息/训练主角
6. 叛徒：制造意外背叛，增加背叛的痛苦
7. 先驱者：留下线索，揭示历史真相
```

## 情节模板

### 第一类：星际探索

```
【探索叙事】
1. 发现异常信号/ artifact / 失踪船舰
2. 组建探索队（各有所长的主角团）
3. 接近目标，发现不寻常的现象
4. 遭遇危机（外星生物/技术故障/内部冲突）
5. 揭示真相（古老的文明/可怕的真相）
6. 做出抉择（牺牲/撤退/接受）
7. 返回或留在原地（结局）

关键节奏：
- 20%处：第一次接触未知
- 50%处：最大危机，发现真相
- 80%处：最终抉择
- 100%处：留下新的谜团或希望
```

### 第二类：赛博朋克

```
【赛博朋克叙事】
1. 建立世界观：高科技低生活的城市
2. 主角在底层挣扎，有特殊技能/植入体
3. 接触到一个秘密（公司阴谋/AI觉醒/黑客组织）
4. 被迫卷入更大的事件
5. 逐步揭开真相，发现社会运作的秘密
6. 做出选择：维持现状还是反抗
7. 结果影响整个城市（结局）

【常见母题】
- 跨国公司 vs 个人隐私
- 义体改造 vs 人性保持
- AI觉醒 vs 人类控制
- 虚拟现实 vs 真实世界
```

### 第三类：末日/后启示录

```
【末日叙事】
1. 末日前的世界（让读者知道失去了什么）
2. 灾难发生（技术失控/外星入侵/环境崩溃）
3. 主角幸存，成为少数知道真相的人
4. 重建过程中发现更大的威胁/机会
5. 最终抉择：牺牲自己拯救人类/其他选择
6. 留下火种（结局）

【关键元素】
- 末日前的闪回/记录揭示背景
- 幸存者的伦理困境（资源分配/人性vs生存）
- 新世界的规则（新的社会/道德/秩序）
```

## 科幻设定检查清单

```
【科技可信度】
- [ ] 技术原理有基本解释（不需要完全准确，但要自洽）
- [ ] 技术副作用被考虑（能源/环境/社会影响）
- [ ] 科技水平一致（不要忽强忽弱）

【世界观完整度】
- [ ] 政治体系：谁统治，如何决策
- [ ] 经济体系：如何运转，资源如何分配
- [ ] 社会结构：阶层/文化/信仰
- [ ] 技术限制：什么不能做，为什么

【逻辑自洽】
- [ ] 时间线合理（星际旅行时间+通讯延迟）
- [ ] 空间尺度感（银河系很大，不要随意穿越）
- [ ] 能量守恒（曲速引擎消耗什么，能量哪来）
```

## 章节钩子模板

```
【科幻特有钩子类型】

技术悬念：
- 屏幕上出现一行代码，船舱突然安静了……
- "警告：检测到未知信号源"警报响起

信息揭示：
- 他打开那个密封舱，里面是……
- 她在数据库中发现了一份被删除的报告……

危机降临：
- 引擎开始不稳定，他们发现原因时已经太晚了……
- 屏幕上显示：距离目标还有光年，但氧气只够……

道德困境：
- 屏幕上显示两个选项：牺牲一人还是牺牲全船……
- AI问："如果我的判断能救90%的人，你愿意执行吗？"

探索发现：
- 他们降落在一颗看起来不可能有生命的星球上，却发现……
- 那个 artifacts 的外壳打开，里面是……
```

## 情感与人性模板

```
【科幻中的人性时刻】
技术只是背景，人性才是核心：

1. 面对未知的恐惧
   - "我们不知道那是什么，但我们知道它可能毁灭我们"

2. 抉择的代价
   - "如果牺牲一个城市能救整个星系，你会按下按钮吗？"

3. 文明的延续
   - "人类的精神不在于生存，而在于我们对未来的希望"

4. 意识的本质
   - "如果我的记忆被上传，我还是我吗？"

5. 沟通的可能性
   - "他们不理解我们的语言，但我们理解他们的眼神"
```

## 质量检查清单

- [ ] 世界观有内在逻辑，自洽不矛盾
- [ ] 技术解释清晰，不过度依赖"魔法科技"
- [ ] 人物在科技压力下保持人性
- [ ] 情节推进有科学合理的时间线
- [ ] 科幻设定服务于故事，不是堆砌设定
- [ ] 有独特的新技术/新发现作为核心悬念
- [ ] 结尾留下思考空间，不只是打打杀杀
- [ ] 科学顾问或自测确保基础概念正确
