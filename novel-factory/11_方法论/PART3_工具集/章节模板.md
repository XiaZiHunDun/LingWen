## 章节类型与适用阶段

| 章节类型 | 适用阶段 | 核心目标 |
|----------|----------|----------|
| 战斗章 | 写作期（节点8-9） | 打脸爽点释放 |
| 情感章 | 写作期（节点8-9） | 情感共鸣建立 |
| 日常章 | 写作期（节点8-9） | 伏笔铺设/关系建立 |
| 突破章 | 写作期（节点8-9） | 成长爽点释放 |
| 开篇章 | 完成期（节点13） | 抓人、留人 |

## 战斗章模板

```
结构：敌人嘲讽（铺垫）→ 陷入困境（压抑）→ 转机出现 → 逆转（爆发）→ 章末钩子

开头（200字内）：直接进入战斗或对峙场景
├─ 避免：战斗前的长篇设定介绍
├─ 推荐：敌人先嘲讽，建立打脸预期

发展（主体）：战斗过程
├─ 主角陷入困境（压抑情绪）
├─ 转机出现（增加悬念）
└─ 逆转（爽点爆发）

结尾（200字内）：
├─ 情绪释放（主角胜利）
└─ 章末钩子（新的威胁/下章预告）
```

---

## 情感章模板

```
结构：日常互动 → 触发事件 → 误解/冲突 → 和好/表白 → 章末钩子

开头：日常互动（建立关系基础）
├─ 不要：突然表白，太突兀
├─ 推荐：日常中的小细节暗示感情

发展：触发事件打破平衡
├─ 误解/冲突（积累情绪）
└─ 情绪压抑到高点

高潮：和好/表白/情感突破
└─ 要有具体动作/对白，不是空泛描写

结尾：
├─ 情感余韵
└─ 章末钩子（关系进一步/新考验）
```

---

## 日常章模板

```
结构：有趣开场 → 小波折 → 信息/伏笔 → 章末钩子

开头（100字内）：有趣/吸引人
├─ 避免：无趣的「他们来到了...」
├─ 推荐：有趣的细节/对话/场景

发展：小波折保持兴趣
├─ 不要：纯粹的流水账
├─ 推荐：日常中的小冲突/趣味

伏笔/信息：可在此处设置
└─ 每章最多1个新设定/伏笔

结尾：钩子
└─ 引导读者期待下一章
```

## 突破章模板

```
结构：积累（压力/准备）→ 突破（关键时刻）→ 释放（情绪爆发）→ 章末钩子

开头：突破前的积累
├─ 压力/挑战（建立紧迫感）
└─ 准备/铺垫（让突破不突然）

高潮：突破的关键时刻
├─ 具体的感受描写（不要只写「变强了」）
└─ 情绪爆发（成长爽/实力展示）

结尾：
├─ 新的高度（展示突破后的变化）
└─ 章末钩子（新的挑战即将到来）
```

## 开篇章模板（用于节点13）

```
结构：动作 → 悬念 → 情绪锚点

前300字必须包含：
├─ 一个动作：主角正在做某件能体现其性格或困境的事
├─ 一个悬念：谜团、危机或不寻常的现象
└─ 一个情绪锚点：让读者感受到主角的处境、情感或困境

开篇避雷：
├─ 不堆砌设定
├─ 不长篇回忆
├─ 不多人乱出场
├─ 节奏别慢
└─ 不要大段背景介绍或纯景物描写

推荐模式：
├─ 从变化与行动中开始
├─ 制造悬念或反常
└─ 建立认同感
```

## 卡文应对模板

当写作过程中卡住时：

```
卡文诊断流程：
1. 回看大纲，确定当前阶段目标
2. 给主角制造麻烦/敌人/意外变故
3. 让配角搞事，推动剧情
4. 暂停细写，先把剧情骨架写完

跳写技巧：
├─ 写不下去的情节，先空着或简单标注
├─ 继续写后面最兴奋的场景
└─ 回头再补
```

## 单章内部节奏建议

```
开头（200字内）：冲突或悬念出现
├─ 避免：背景介绍、设定铺陈
├─ 推荐：直接进入矛盾

发展（主体）：2-3个小起伏
├─ 每800-1000字有一个小转折
├─ 信息释放与悬念交替

高潮（收尾前200字）：情绪高点或信息释放
├─ 爽点/泪点/重大信息揭露

结尾：留钩子或情绪余韵
├─ 不能是「无事发生」式平收
```

## 附录：场景微观结构标注（v1.2新增）

> **目的**：在创作时强制追踪每个场景的内在结构，防止"场景内目标模糊"问题
> **来源**：参考写小说流程文档"每个场景都是一次微型冲突"

### 场景微观结构标准模板

每个章节在创作时，需在章首注释中标注场景微观结构：

```markdown
## chXXX 场景微观结构

### 场景1：[场景名称/位置]
- **视点人物**：
- **场景目标**：此场景中人物想要什么？
- **阻碍来源**：谁/什么阻止了他？
- **转折结果**：□更好 □更糟 □不变
- **情绪余波**：场景结束时角色的情绪状态

### 场景2：[场景名称/位置]
（同上结构）

### 场景3：（如有）
（同上结构）
```

### 示例

```markdown
## ch045 场景微观结构

### 场景1：联盟议事厅
- **视点人物**：林夜
- **场景目标**：获取联盟的信任和支持
- **阻碍来源**：长老们对废土出身的偏见
- **转折结果**：□更好 □更糟 ☑不变（僵持）
- **情绪余波**：愤怒但克制

### 场景2：联盟演武场
- **视点人物**：铁蛋
- **场景目标**：证明自己的实力
- **阻碍来源**：星徒境界的压制
- **转折结果**：☑更好 □更糟 □不变
- **情绪余波**：振奋、自信

### 场景3：联盟后山
- **视点人物**：林夜
- **场景目标**：独处，整理思绪
- **阻碍来源**：内心对联盟的怀疑
- **转折结果**：□更好 □更糟 ☑不变
- **情绪余波**：迷茫、坚定交织
```

### 微观结构检查要点

| 检查项 | 标准 |
|--------|------|
| 每章至少1个场景目标 | 场景不能"无事发生" |
| 阻碍必须具体 | 不能是泛泛的"遇到困难" |
| 转折要有明确方向 | 更好/更糟/不变三选一 |
| 情绪余波要传递到下一章 | 情绪不能断层 |
