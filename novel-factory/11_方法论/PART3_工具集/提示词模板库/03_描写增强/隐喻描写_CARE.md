## A - Action（行动）

### 隐喻构建方法

#### 1. 本体-喻体选择
- 本体（要描写的事物）：{subject}
- 喻体（用来比喻的事物）：{vehicle}
- 连接方式：{mapping_type}
  - 相似性映射：形状、颜色、质感等相似
  - 情感映射：将情感投射到自然/物体
  - 因果映射：将原因/结果具象化

#### 2. 映射层次
- 表层映射（直接相似）：{surface_mapping}
- 深层映射（抽象关联）：{deep_mapping}
- 情感映射（主观感受）：{emotional_mapping}

#### 3. 隐喻类型
- 根隐喻：{root_metaphor}
  - 核心概念的核心比喻
  - 贯穿全文的深层隐喻
- 显性隐喻：{explicit_metaphor}
  - 直接的比喻表达
  - 格式："像……"、"如……"、"仿佛……"
- 概念隐喻：{conceptual_metaphor}
  - 系统性的隐喻映射
  - 如：时间就是金钱，生命就是旅程

## E - Example（示例）

### 核心隐喻示例

```markdown
[主题]：命运的不可抗力

[原文]
他感到自己的命运无法掌控。

[隐喻增强后]
他站在命运的洪流中，像一片落叶被裹挟着前行。没有根，没有方向，只能随着水流颠簸起伏。

有时候是漩涡，将他卷入深渊；有时候是险滩，在他以为看到希望时将他撞得支离破碎。他挣扎过，抗争过，用尽全身的力气想要游向岸边。

但水流的意志比他更强大。

每一次他以为终于抓住了什么，水流就会毫不留情地将它夺走。而那些沉入河底的，有时是挚爱，有时是理想，有时是他以为永远不会再失去的东西。

他终于明白，在这命运的河流中，他能做的不是上岸，而是学会在水中呼吸。

**注解**：
- 根隐喻：命运是一条河流
- 映射层次：
  - 表层：落叶随水流
  - 深层：挣扎与无力
  - 情感：压抑、绝望、认命
- 辅助隐喻：漩涡、险滩、岸边、河底
- 主题关联：命运的不可抗力，通过河流的特性体现
```

### 情感隐喻示例

```markdown
[主题]：失恋的空虚

[原文]
分手后，她感到心里空落落的。

[隐喻增强后]
她打开冰箱，里面空空如也。就像她的心一样。

那些曾经以为会永远保鲜的东西——他的笑容、他们一起做饭的周末、深夜相拥的温度——不知从什么时候开始，已经悄悄变质，被她一一扔进了垃圾桶。

冰箱的冷光灯照在空荡荡的隔板上，冷得刺眼。她想起第一天搬进这间公寓时，这个冰箱还是满的。那时候她觉得生活充满期待，每一格都是未来的味道。

现在呢？

现在她买了一把鲜花插在冰箱里，算是对生活最后的倔强。但花也会枯萎的，就像所有她曾经觉得会永远的东西一样。

**注解**：
- 核心隐喻：心是冰箱
- 映射：
  - 空空如也→空虚
  - 保鲜→感情的新鲜
  - 变质→感情破裂
  - 鲜花→最后的倔强
- 情感：空虚→回忆→无力感
- 新颖度：用日常物品（冰箱）承载情感，避免陈词滥调
```

### 多层次隐喻示例

```markdown
[主题]：成长的代价

[原文]
他变强了，但也失去了很多。

[隐喻增强后]
他手中的剑越来越重。

不是因为剑身变长了，而是因为每一次挥剑，都要在心里增加一块石头。那些石头是愧疚，是失去，是每一个他来不及说再见的人。

修仙之路，是一条用遗忘铺成的路。他忘记了自己曾经是个会为一只受伤的鸟流泪的孩子。忘记了第一次喜欢上一个人时，那种笨拙而真挚的悸动。忘记了朋友是什么，忘记了信任是什么，忘记了脆弱是什么。

他只记得剑。

剑锋所指，便是生死。剑落之处，便是因果。他成了别人眼中的强者，却成了自己心里的陌生人。

但这就是代价，不是吗？

想要摘星，就要学会在黑暗中行走。想要登顶，就要学会放下沿途的一切。

他继续挥剑，心里又多了一块石头。

**注解**：
- 根隐喻：成长是负重前行
- 核心隐喻：剑+石头=力量+代价
- 概念隐喻：修仙之路是遗忘之路
- 映射层次：
  - 剑重→责任
  - 石头→失去
  - 遗忘→成长的代价
- 情感：从怀旧到接受，复杂而深沉
```

### 隐喻构建检查
- [ ] 本体与喻体有可理解的联系
- [ ] 隐喻服务于情感表达
- [ ] 不生硬、不炫技
- [ ] 有层次感
- [ ] 与上下文协调

### 注意事项
1. 隐喻不是目的，是手段——服务情感和叙事
2. 避免陈词滥调（"她的眼睛像星星"）
3. 一致性：核心隐喻贯穿始终
4. 克制：不要通篇都是隐喻
5. 融合：显性隐喻和概念隐喻结合使用
