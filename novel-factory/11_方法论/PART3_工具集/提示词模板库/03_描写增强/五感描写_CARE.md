## A - Action（行动）

### 五感描写（按优先级排序）

#### 1. 视觉（必须）
- 颜色：主体色彩、次要色彩、点缀色
- 光影：光源方向、光影效果、光线质感
- 形态：轮廓、形状、大小、比例
- 动态：运动状态、速度感、节奏感
- 视觉焦点：读者注意力引导

#### 2. 听觉（强烈推荐）
- 环境音：背景声、氛围声
- 人物音：呼吸、脚步、衣物摩擦
- 特殊音：与目标相关的独特声音
- 声音层次：远/中/近、清晰/模糊

#### 3. 嗅觉（推荐）
- 主体气味：最主要的气味特征
- 气味层次：主味、辅味、残留
- 气味情感关联：勾起记忆、情绪暗示

#### 4. 触觉（场景相关时使用）
- 质地：粗糙、光滑、柔软、坚硬
- 温度：冷/热/温/凉及程度
- 湿度：干燥、湿润、潮湿
- 肌理：纹理、颗粒感、细腻度

#### 5. 味觉（场景相关时使用）
- 主体味道：酸甜苦辣咸
- 味道层次：入口、回味
- 味道与记忆/情感的关联

### 隐喻要求
- 隐喻类型：{metaphor_type}
  - 视觉隐喻：通过视觉类比其他感官
  - 感官融合：多种感官的组合隐喻
  - 情感隐喻：将情感投射到感官描写
- 隐喻来源：{metaphor_source}
  - 自然类：山水风云花鸟
  - 日常类：生活用品、食物
  - 情感类：抽象情感的具象化
- 隐喻密度：每100字{m_metaphor_count}个

### 约束条件
- 字数增量：约在原文基础上增加{increase_percentage}%
- 感官种类：至少包含{variety_min}种感官
- 禁止：{prohibited_elements}

## E - Example（示例）

### 场景描写示例

```markdown
[原文]
他走进森林，感到很害怕。

[五感增强后]
他踏入森林的瞬间，空气中弥漫的腐叶气息扑面而来，混着泥土的腥甜和某种说不清的阴冷。头顶的阳光被层层叠叠的枝叶切割成细碎的光斑，斑驳地洒在苔藓覆盖的地面上。那些光斑随着树叶的摇曳而不停颤动，像是无数只窥视的眼睛。

脚下是厚厚的落叶，每一步都发出轻微的沙沙声，混合着偶尔传来的不知名鸟叫，回荡在寂静的林间。越往深处走，光线越暗，空气也越发潮湿，带着一股霉味和野花若有若无的甜香。

他的喉咙发紧，手指不自觉地攥紧了剑柄。森林深处，有什么东西在等待着他。

**注解**：
- 视觉：光斑、阴暗对比、窥视的眼睛（隐喻）
- 听觉：沙沙声、鸟叫、寂静
- 嗅觉：腐叶味、泥土腥、阴冷、霉味、野花甜香
- 触觉：脚下落叶的触感、空气潮湿
- 情感：通过五感营造阴森紧张的氛围
- 隐喻："窥视的眼睛"暗示危险
```

### 人物描写示例

```markdown
[原文]
她很漂亮，很悲伤。

[五感增强后]
月光下，她的脸庞像是被洗褪了颜色，只剩下淡淡的轮廓。那双眼睛干涩而空洞，却仍能看见里面积蓄的泪水，像两汪即将干涸的深潭。她的嘴唇微微颤抖着，偶尔逸出一两声压抑的呜咽，破碎而绝望。

她的手放在膝盖上，指节因为用力而泛白，手背上青筋隐隐可见。整个人蜷缩在那里，像一朵被霜打过的花，脆弱得仿佛一阵风就能将她吹散。

空气中弥漫着淡淡的酒气，混着她发间残留的桂花香。那香气本该是甜腻的，此刻却透着一股化不开的苦涩。

**注解**：
- 视觉：脸庞轮廓、眼睛干涩、嘴唇颤抖、指节泛白
- 听觉：呜咽声
- 嗅觉：酒气、桂花香
- 触觉：霜打的花的脆弱感
- 隐喻："被霜打过的花"暗示命运和状态
- 情感：悲伤通过细节多维度展现
```

## 模板使用说明

### 必填参数
| 参数 | 说明 | 示例 |
|------|------|------|
| target_type | 描写对象类型 | "场景环境" |
| target_name | 描写对象名称 | "万妖山脉森林" |
| writing_style | 目标风格 | "电影感、沉重" |
| chapter_emotion | 章节情感 | "紧张、压抑" |
| original_text | 原文 | （要增强的原文） |

### 可选参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| metaphor_type | 隐喻类型 | "视觉隐喻" |
| increase_percentage | 扩写比例 | 30% |
| variety_min | 最少感官数 | 3 |

### 感官描写优先级
| 场景类型 | 优先感官 | 次要感官 |
|----------|---------|---------|
| 战斗场景 | 视觉、听觉 | 触觉 |
| 情感场景 | 视觉、嗅觉 | 味觉 |
| 日常场景 | 视觉、听觉、嗅觉 | 触觉 |
| 自然场景 | 视觉、听觉、嗅觉 | 触觉 |

### 注意事项
1. 五感不是堆砌，要服务于情感和叙事
2. 隐喻要贴切，不能生搬硬套
3. 保持与原文风格的统一
4. 感官要有层次和节奏
5. 最终落点是情感共鸣，不是感官罗列
