## E - Example（示例）

### 谈判博弈示例

```markdown
[前文]谈判桌上，对面坐着的是玄清宗的长老，而林夜代表的是刚刚兴起的青冥派。

---

[对话场景]

"林小友，你们青冥派要的这三座灵矿，可是我玄清宗三代弟子的血汗换来的。"赵长老的声音不高，但每个字都带着压迫感。

林夜端起茶杯，轻轻抿了一口，不急不慢地说："赵长老说的没错。三代弟子的血汗，所以更要好好开发利用，而不是像现在这样闲置。"

"闲置？"赵长老眉头一挑，"我玄清宗每年从这三座灵矿产出——"

"年产灵石八千枚，其中上品灵石不足百枚。"林夜接过话头，"利用率不到三成。而我青冥派有新的开采技术，可以将产量提升一倍。"

赵长老的眼睛微微眯起。他没想到这个年轻人做了这么充分的准备。

"林小友的意思是，用你们的技术，换我们的灵矿？"

"不是换。"林夜放下茶杯，直视对方，"是合作。贵宗出矿，我们出技术，收益五五分成。"

"五五？"赵长老冷笑一声，"林小友这胃口可不小啊。"

"胃口大不大，要看值不值。"林夜不为所动，"赵长老可以拒绝，那我们只好去找天剑宗谈了。"

赵长老的脸色微微一变。天剑宗与玄清宗是百年对手，如果青冥派真的和天剑宗合作——

"林小友这是在威胁我？"

"不敢。"林夜笑了笑，"只是在陈述事实。"

沉默在两人之间蔓延。

赵长老盯着林夜看了许久，突然笑了："好，好一个林夜。难怪能在这乱世中闯出名声。"

他站起身，走到窗边，背对着林夜说："四六分成，我六你四。这是我最大的让步。"

林夜想了想，点头："可以。但有一个条件——合作期限十年，期满后贵宗有优先续约权。"

"好。"赵长老转过身，伸出手，"成交。"

林夜与他击掌："成交。"

[对话场景完]

**注解**：
- 目的冲突：玄清宗想保住资源，青冥派想获取资源
- 信息战：林夜用数据说话，展示充分准备
- 潜台词：提到天剑宗是暗示筹码
- 转折点：赵长老从对抗到认可林夜的能力
- 语言风格：林夜年轻但稳重，赵长老老练但不失风度
```

### 情感对话示例

```markdown
[前文]小青和林夜站在星空下，已经沉默了很久。

---

[对话场景]

"林夜。"小青终于开口，声音很轻。

"嗯。"

"你后悔吗？"她没有看他，只是望着星空，"那天救我。"

林夜愣了一下："为什么这么问？"

"因为我是妖。"小青的声音平静得可怕，"人妖不两立。这是你们人族说的。"

"那是别人说的，不是我。"

小青转过头，第一次认真地看着他。月光下，林夜的眼神清明而坚定。

"你真的不在意吗？"

"小青。"林夜走近一步，"我在意的是你这个人，不是你的身份。"

"可是……"

"没有可是。"林夜打断她，"我知道这条路很难走。也许有一天，整个修真界都会与我们为敌。"

他伸出手，轻轻握住小青的手。

"但那又怎样？"

小青的手微微颤抖。她想说什么，却发现自己说不出话来。

"我只知道，"林夜的声音很轻，但很坚定，"如果没有你，我的修行将毫无意义。"

星光下，小青的眼泪无声地落下。

"笨蛋……"她哽咽着说，"你这个笨蛋……"

林夜笑了，张开双臂，将她拥入怀中。

星空依旧寒冷，但两人相拥的地方，却无比温暖。

[对话场景完]

**注解**：
- 表层对话：讨论人妖身份问题
- 深层情感：确认彼此的心意
- 潜台词：林夜"修行毫无意义"是深情的表达
- 转折点：从质疑到拥抱
- 收尾：用温度对比（星空寒冷vs相拥温暖）暗示情感
```

## 模板使用说明

### 必填参数
| 参数 | 说明 | 示例 |
|------|------|------|
| dialogue_type | 对话类型 | "辩论式场景" |
| speaker_1 | 说话者1 | "林夜" |
| speaker_2 | 说话者2 | "赵长老" |
| surface_goal | 表层目标 | "争取合作" |
| deep_goal | 深层目标 | "建立信任" |

### 对话类型选择
| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 对话式推进 | 自然交流，信息传递为主 | 日常、情报交换 |
| 辩论式场景 | 观点碰撞，攻防明显 | 谈判、争论、说服 |
| 审讯/谈判 | 一方主导，另一方被动 | 审讯、逼问、摊牌 |
| 情感对话 | 含蓄表达，潜台词多 | 确认心意、告别、和解 |

### 注意事项
1. 对白要简短有力，避免长篇大论
2. 每句话都要有潜台词或推进作用
3. 用动作、神态补充语言
4. 角色语言风格要统一
5. 转折点要有铺垫，不能突兀
