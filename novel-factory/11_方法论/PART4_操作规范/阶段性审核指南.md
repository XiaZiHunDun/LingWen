# 阶段性审核指南

> **版本**：v1.0
> **来源**：动画制作流程的资产审核制度
> **日期**：2026-05-19

---

## 一、工具说明

动画制作有一个核心制度：**资产审核（Asset Approval）**——模型、绑定、材质必须通过导演和各部门负责人签字，才能进入动画环节。一旦进入下游，修改成本指数级上升。

小说创作可以借鉴这个制度：**在写作期设置三个"资产关卡"，每个关卡有明确的通过标准，未通过则暂停写作，返回补充**。

---

## 二、三个资产关卡

### 关卡1：角色资产（第1章写作前）

**审核内容**：主要角色的设定卡

**通过标准**：

| 检查项 | 标准 |
|---|---|
| 三元结构 | 每个主要角色都有清晰的：欲望 + 阻碍 + 代价 |
| 致命弱点 | 每个主要角色至少有1个性格缺陷或心理创伤 |
| 秘密 | 每个主要角色至少有1个不愿被人知道的过去 |
| 角色弧线 | 主角在故事结束时发生了什么变化（可感知的变化） |
| 角色数量 | 主要角色（戏份≥1章）≤ 10个，避免人物失控 |

**未通过处理**：角色设定不完整 → 暂停写作 → 补充角色设定 → 重新审核

---

### 关卡2：世界资产（第10章写作前）

**审核内容**：核心场景、势力关系、规则文档

**通过标准**：

| 检查项 | 标准 |
|---|---|
| 核心场景 | 主要故事发生地的场景有明确描述（视觉/声音/感觉） |
| 势力关系 | 势力关系图已绘制（至少包含：主角阵营/对手阵营/中立势力） |
| 核心规则 | 世界观核心规则已文字化（≤500字），边界清晰 |
| 场景功能 | 每个核心场景都有明确的叙事功能（为什么这场戏必须在这里发生） |

**未通过处理**：世界资产不完整 → 暂停写作 → 补充世界设定 → 重新审核

---

### 关卡3：节拍资产（每10章节点）

**审核内容**：本阶段的章节完成情况

**通过标准**：

| 检查项 | 标准 |
|---|---|
| 翻页钩子 | 80%的章节结尾有明确的"翻页钩子"（留下悬念或冲突未解决） |
| 事件密度 | 无连续3章以上无事件推进（日常章节必须有功能，不能水字数） |
| 章节完整 | 每个章节都有明确的目标、阻碍、结果（或情绪变化） |
| 伏笔记录 | 已埋设的伏笔有记录，读者知道的信息和角色知道的信息分开 |

**未通过处理**：节拍问题 → 暂停下一阶段写作 → 修改问题章节 → 继续

## 三、审核流程

```
写作启动
    ↓
关卡1审核（角色资产）→ 不通过 → 补充角色 → 重审
    ↓ 通过
开始写作第1-9章
    ↓
关卡2审核（世界资产）→ 不通过 → 补充世界 → 重审
    ↓ 通过
继续写作第10-19章
    ↓
关卡3审核（第1个10章）→ 不通过 → 修改问题章节 → 重审
    ↓ 通过
继续写作第20-29章
    ↓
关卡3审核（第2个10章）→ 不通过 → 修改问题章节 → 重审
    ↓ 通过
...
```

---

## 四、与锁定门的关系

| 概念 | 位置 | 作用 |
|---|---|---|
| 锁定门 | 规划期结束，写作期开始前 | 整体检查，检查通过才能开始写作 |
| 资产关卡 | 写作期内，每10章 | 过程检查，确保写作质量不失控 |

**区别**：
- 锁定门是"规划质量"的最终检查
- 资产关卡是"写作过程"的中间质量把控

## 六、与评审维度的对应

| 维度 | 阶段性审核相关检查点 |
|---|---|
| S1 剧情完整性 | 章节是否有明确的目标-阻碍-结果 |
| S2 逻辑自洽 | 势力行为是否与设定一致 |
| S6 可读性 | 翻页钩子是否有效 |
| S7 主角魅力 | 角色的欲望和阻碍是否清晰 |
