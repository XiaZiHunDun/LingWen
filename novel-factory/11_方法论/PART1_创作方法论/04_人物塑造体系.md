# 阶段2：人物塑造体系

> **核心观点**：故事是人物推动的。不要只写标签，要探究内心。
> 好的人物不是"完美的人"，而是"有缺陷的人做出了艰难的选择"。

---

## 一、人物三角：欲望、恐惧、缺陷

```
        欲望 ←——————→ 恐惧
              ↑↓
            缺陷
```

| 要素 | 问题 | 示例 |
|------|------|------|
| **欲望** | 人物最想要什么？ | 复仇、认可、安全、爱 |
| **恐惧** | 人物最怕失去什么？ | 被抛弃、死亡、失败、孤独 |
| **缺陷** | 什么性格缺陷阻止他？ | 偏执、懦弱、愤怒、不信任 |

**欲望与恐惧的冲突 = 故事引擎**

举例：退伍军人
- 欲望：被认可，被接纳
- 恐惧：被抛弃
- 缺陷：不信任人，偏执
- 冲突：想要被接纳，但又本能地推开所有人

## 二、人物弧光（Character Arc）

**定义**：故事结束时，人物改变了什么？

### 弧光三问
1. 故事开始时，这个人相信什么？（信念）
2. 过程中，什么让他付出代价？（考验）
3. 故事结束时，他领悟/改变了什么？（成长）

### 常见弧光类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 成长型 | 从软弱到坚强 | 《功夫熊猫》阿宝 |
| 堕落型 | 从善到恶 | 《星球大战》黑武士 |
| 领悟型 | 从误解到理解 | 理解敌人的苦衷 |
| 救赎型 | 从破碎到完整 | 走出创伤阴影 |
| 循环型 | 回到原点但改变 | 离开又回到小镇 |

## 三、人物小传要素

即使书里用不到，也要想清楚：

| 要素 | 内容 |
|------|------|
| 背景 | 经历过的重大事件（塑造了他的现在） |
| 秘密 | 不愿被人知道的过去 |
| 口头禅 | 让角色更鲜活 |
| 习惯动作 | 让角色更可感知 |
| 价值观 | 他相信什么是对的？ |
| 矛盾点 | 他嘴上说的 vs 实际做的 |

## 四、配角设计原则

**原则**：每个配角都要有功能——功能性 + 独特性

| 角色类型 | 功能 | 设计要点 |
|----------|------|----------|
| 帮手型 | 帮助主角成长 | 有自己动机，不只是工具人 |
| 对手型 | 制造冲突 | 和主角目标对立，但不是纯粹邪恶 |
| 镜像型 | 反映主角 | 和主角有相似处境但不同选择 |
| 导师型 | 提供信息/教训 | 推动情节，但不能代替主角行动 |

**数量控制**：主要人物 ≤ 5人，避免自己先乱

## 五、反派设计原则

**核心**：反派要有立场、有苦衷，不纯粹邪恶

| 维度 | ❌ 坏反派 | ✅ 好反派 |
|------|----------|----------|
| 动机 | "我就是坏" | 有合理的理由（即使你不认同） |
| 视角 | 脸谱化 | 反派认为自己是正义的 |
| 能力 | 比主角弱 | 对主角构成真正的威胁 |
| 关系 | 无关 | 和主角有私人联系（师徒/亲友/镜像） |

### 好反派示例思路
- 他曾是受害者，变成了加害者
- 他的目标和主角目标在某种程度上一致
- 他让读者理解为什么他这样选择

## 六、人物关系网

梳理人物之间的关系：

```
爱（亲情/友情/爱情）
  ↑
仇恨（复仇/背叛/嫉妒）
  ↑
师徒/上下级
  ↑
路人 → 配角 → 核心配角 → 主角
```

**检查清单**：
- [ ] 每对有关系的人物，冲突在哪里？
- [ ] 主角最重要的关系是谁？为什么？
- [ ] 有没有"孤岛角色"（和其他角色没关系）？去掉或连接
