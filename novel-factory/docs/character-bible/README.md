## 概述

本目录包含《星陨纪元》核心角色的完整角色圣经（Character Bible），基于 CCGS narrative-character-sheet.md 模板创建。

每个角色圣经包含：概念、外貌、个性、动机与弧光、内心冲突、关系、对话笔记、Lore 连接、跨引用索引、审核检查点。
