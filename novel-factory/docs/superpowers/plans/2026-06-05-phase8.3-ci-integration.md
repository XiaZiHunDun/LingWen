### Step 1.1: 创建 workflow 文件

**File**: `.github/workflows/real-llm-tests.yml` (NEW)

**完整内容** (~70 行):

```yaml
# Phase 8.3: Real LLM tests CI
# 跑 Phase 8 (Anthropic) 2 个真实 LLM 测试, 抓 production 路径 regression.
# - 触发: cron (UTC 02:00 daily = 北京 10:00) + 手动 workflow_dispatch
# - Provider: Anthropic only (Phase 8 HAIKU, ~$0.025/run, $0.75/月)
# - 完整 setup 见 docs/CI_SETUP.md
# 维护: 修改 trigger 频率/provider 范围/测试 selection 在此文件

name: real-llm-tests

on:
  schedule:
    - cron: '0 2 * * *'  # UTC 02:00 daily
  workflow_dispatch:

concurrency:
  group: real-llm-tests-${{ github.ref }}
  cancel-in-progress: true

jobs:
  real-llm-tests:
    name: Real LLM tests (Anthropic, Python 3.13)
    runs-on: ubuntu-latest

    strategy:
      fail-fast: false
      matrix:
        python-version: ["3.13"]

    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Check ANTHROPIC_API_KEY secret
        # Fail-loud on missing secret — skipif 静默 SKIP 会让主公误以为成功
        run: |
          if [ -z "${{ secrets.ANTHROPIC_API_KEY }}" ]; then
            echo "::error::ANTHROPIC_API_KEY secret is not set in repo Settings → Secrets → Actions"
            exit 1
          fi

      - name: Setup Python ${{ matrix.python-version }}
        uses: actions/setup-python@v5
        with:
          python-version: ${{ matrix.python-version }}
          cache: pip
          cache-dependency-path: pyproject.toml

      - name: Install (editable + dev)
        run: |
          python -m pip install --upgrade pip
          pip install -e ".[dev]"

      - name: Run real LLM tests (Anthropic only)
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        run: |
          pytest tests/agent_system/test_novel_writing_real_llm.py \
            -k "TestNovelWritingRealLLM or TestPolishMergeSynthesisRealLLM" \
            --tb=short -v

      - name: Ruff (informational, non-blocking)
        run: |
          pip install ruff
          ruff check tests/agent_system/test_novel_writing_real_llm.py tests/agent_system/_e2e_helpers.py \
            || echo "::warning::Ruff found issues (informational only)"

      - name: Upload pytest output artifact
        if: always()
        uses: actions/upload-artifact@v4
        with:
          name: real-llm-tests-output
          path: |
            .pytest_cache/
            pytest.log
          retention-days: 14
          if-no-files-found: ignore
```

**注释说明 (3 处关键设计)**:
1. **顶部注释**: 解释 workflow 目的 + cost + 触发模式 + 引用 CI_SETUP.md
2. **Check ANTHROPIC_API_KEY**: 解释 fail-loud 设计 (vs skipif 静默)
3. **Ruff informational**: 跟 test.yml 一致, 失败不阻塞

## 风险 + 缓解

| 风险 | 缓解 |
|------|------|
| 主公没设 `ANTHROPIC_API_KEY` secret → 测试 SKIP → 误以为成功 | fail-loud check (Task 1 step), secret 缺失 → job 立刻 fail + error message |
| 真实 LLM 偶尔 flaky (Anthropic timeout) → CI 假阳性 fail | max_retries=1 (Phase 8 helper 已配), 偶发 1 retry 容忍, 持续 fail → 主公修 prompt/provider |
| 跑挂没人看 → regression 累积 | GitHub email 默认通知 (主公账号绑定), workflow run 出现在 PR check status |
| 成本失控 (spend cap 失效) | 手动 disable workflow (Task 2 step 6), 或改 cron 注释, commit |
| ANTHROPIC_API_KEY 泄漏到 git | 用 GitHub secrets 注入, workflow 文件不写明文, `.env` 在 `.gitignore` |
| ANTHROPIC_API_KEY 泄漏到 PR 公开 log | `${{ secrets.ANTHROPIC_API_KEY }}` 自动 redact, GitHub 安全机制 |
| cron 触发但 secrets 被删除 | fail-loud check 立刻 fail, 主公邮件通知 |
| 并发 (manual + cron 同时跑) | `concurrency.cancel-in-progress: true` (Task 1) |
| Python 3.13 单独跑, 3.11/3.12 兼容性 regression | test.yml matrix 仍然跑 3 version, real-llm-tests.yml 跟 test.yml 互补 |
| 跑挂找不到原因 (artifact 缺失) | upload-artifact 14d retention (Task 1), pytest.log 保留 |
