## Design

### 1. Data Model (3 frozen dataclasses)

**`ReferenceNode`** — 4 维 (character / foreshadow / setting / plot_point) 共享 schema + per-dim payload dict.

```python
DimensionT = Literal["character", "foreshadow", "setting", "plot_point"]

@dataclass(frozen=True)
class ReferenceNode:
    id: str = field(default_factory=lambda: uuid4().hex)
    dimension: DimensionT = "character"
    volume: int = 1            # 1-based
    chapter: int = 0           # 0 = volume-level
    title: str = ""            # human-readable
    description: str = ""      # ≤ 200 chars
    payload: dict[str, Any] = field(default_factory=dict)
    created_at: datetime = field(default_factory=lambda: datetime.now(timezone.utc))
    created_by: str = "manual" # Phase 11+: "llm_scanner:character"

    def __post_init__(self):
        if self.volume < 1: raise ValueError(...)
        if self.chapter < 0: raise ValueError(...)
        if len(self.description) > 200: raise ValueError(...)
```

**Payload per-dim examples** (Phase 10 minimum useful):
- `character`: `{"name": "李青云", "role": "主角", "first_appearance_volume": 1, "first_appearance_chapter": 3}`
- `foreshadow`: `{"title": "古剑碎片", "planted_volume": 1, "planted_chapter": 5, "expected_payoff_volume": 3, "status": "open"}`
- `setting`: `{"name": "凌霄宗", "type": "faction", "description": "...", "first_intro_volume": 1, "first_intro_chapter": 1}`
- `plot_point`: `{"title": "青云觉醒", "type": "revelation", "volume": 1, "chapter": 8, "impact_level": "high"}`

**`ReferenceEdge`** — from_node_id → to_node_id, 8 种 relationship_type:

```python
RelationshipT = Literal[
    "mentions", "evolves", "foreshadows", "pays_off",
    "appears_with", "causes", "conflicts_with", "supports",
]

@dataclass(frozen=True)
class ReferenceEdge:
    id: str = field(default_factory=lambda: uuid4().hex)
    from_node_id: str = ""
    to_node_id: str = ""
    relationship_type: RelationshipT = "mentions"
    weight: float = 1.0        # 0.0-1.0, manual=1.0 default
    payload: dict[str, Any] = field(default_factory=dict)
    created_at: datetime = ...
    created_by: str = "manual"

    def __post_init__(self):
        if not self.from_node_id or not self.to_node_id: raise ValueError(...)
        if self.from_node_id == self.to_node_id: raise ValueError("self-loop not allowed")
        if not 0.0 <= self.weight <= 1.0: raise ValueError(...)
```

**`CrossVolumeRipple`** — scan 触发的事件, Phase 10 `payload={}` (LLM scanner 留 Phase 11+):

```python
RippleStatusT = Literal["pending", "confirmed", "applied", "rejected", "failed"]

@dataclass(frozen=True)
class CrossVolumeRipple:
    id: str = field(default_factory=lambda: uuid4().hex)
    trigger_volume: int = 1
    trigger_chapter: int = 0
    affected_nodes: tuple[str, ...] = ()     # tuple for immutability
    affected_edges: tuple[str, ...] = ()
    proposed_actions: tuple[dict, ...] = ()
    status: RippleStatusT = "pending"
    created_at: datetime = ...
    confirmed_at: datetime | None = None
    applied_at: datetime | None = None
    payload: dict[str, Any] = field(default_factory=dict)
```

**Key invariants**:
- 3 dataclass `frozen=True` → append-only at Python level (mutation raises `FrozenInstanceError`)
- `tuple[str, ...]` for `affected_nodes` / `affected_edges` / `proposed_actions` — 真 immutable
- 0 `__hash__` override → 默认 `id` based hash OK (跟 `ThoughtNode` 一致)
- `payload: dict` 是 mutable inside frozen dataclass, caller 协议不 mutate (per docs)

### 2. CVG Container (in-memory layer)

```python
class CrossVolumeReferenceGraph:
    def __init__(self, storage: RippleStorage) -> None:
        self._storage = storage
        self._nodes: dict[str, ReferenceNode] = {}
        self._edges: dict[str, ReferenceEdge] = {}
        self._ripples: dict[str, CrossVolumeRipple] = {}
        # 4 secondary indexes for O(1) / O(k) lookup
        self._index_by_volume: dict[int, set[str]] = {}
        self._index_by_dimension: dict[DimensionT, set[str]] = {}
        self._index_by_node_edges: dict[str, set[str]] = {}  # 双向, O(k) neighbor
        self._load_from_storage()  # bulk load on startup
```

**API surface** (10 public methods):
- `add_node(node) -> None` — validate (id unique), update 3 indexes, persist via `storage.append_node`
- `get_node(node_id) -> ReferenceNode | None`
- `get_nodes_by_volume(volume: int) -> list[ReferenceNode]`
- `get_nodes_by_dimension(dim: DimensionT) -> list[ReferenceNode]`
- `add_edge(edge) -> None` — validate (id unique, FK exists in `_nodes`, no self-loop, no duplicate pair)
- `get_edge(edge_id) -> ReferenceEdge | None`
- `get_neighbors(node_id: str) -> list[ReferenceNode]` — O(k) via `_index_by_node_edges`
- `record_ripple(ripple) -> None` — validate (id unique), persist
- `get_ripple(ripple_id) -> CrossVolumeRipple | None`
- `trigger_cascade(ripple) -> list[ReferenceNode]` — **Phase 10 stub returns `[]`**, Phase 14 real BFS impl
- `__len__() -> int` — returns node count (test convenience)

### 3. RippleStorage (sqlite3 直写)

**3 tables + 4 indexes**:
```sql
CREATE TABLE reference_nodes (
    id TEXT PRIMARY KEY, dimension TEXT NOT NULL, volume INTEGER NOT NULL,
    chapter INTEGER NOT NULL, title TEXT NOT NULL, description TEXT NOT NULL,
    payload TEXT NOT NULL,           -- JSON
    created_at TEXT NOT NULL,        -- ISO 8601 UTC
    created_by TEXT NOT NULL
);
CREATE INDEX idx_nodes_volume ON reference_nodes(volume);
CREATE INDEX idx_nodes_dim ON reference_nodes(dimension);

CREATE TABLE reference_edges (
    id TEXT PRIMARY KEY,
    from_node_id TEXT NOT NULL REFERENCES reference_nodes(id),
    to_node_id TEXT NOT NULL REFERENCES reference_nodes(id),
    relationship_type TEXT NOT NULL, weight REAL NOT NULL,
    payload TEXT NOT NULL,
    created_at TEXT NOT NULL, created_by TEXT NOT NULL
);
CREATE INDEX idx_edges_from ON reference_edges(from_node_id);
CREATE INDEX idx_edges_to ON reference_edges(to_node_id);

CREATE TABLE reference_ripples (
    id TEXT PRIMARY KEY,
    trigger_volume INTEGER NOT NULL, trigger_chapter INTEGER NOT NULL,
    affected_nodes TEXT NOT NULL,     -- JSON list
    affected_edges TEXT NOT NULL, proposed_actions TEXT NOT NULL,
    status TEXT NOT NULL,
    created_at TEXT NOT NULL, confirmed_at TEXT, applied_at TEXT,
    payload TEXT NOT NULL
);
```

**PRAGMAs** (`__init__` 设置):
- `PRAGMA synchronous=FULL` — fsync on commit, Phase 13 rollback 时不丢数据
- `PRAGMA journal_mode=WAL` — 多 reader + 1 writer 并发 (Phase 12 dashboard 查询安全)
- `PRAGMA foreign_keys=ON` — FK 强制, 删节点 cascade 边 (Phase 13 audit 用)

**API surface** (7 public methods + 1 context manager):
- `append_node(node) -> None` — single INSERT autocommit
- `append_edge(edge) -> None` — single INSERT autocommit (FK enforced)
- `append_ripple(ripple) -> None` — single INSERT autocommit
- `atomic_batch()` — `@contextmanager`, `BEGIN IMMEDIATE` + `commit` / `rollback` on exception
- `load_all_nodes() -> list[ReferenceNode]` — `SELECT *`, `_row_to_node` 反序列化
- `load_all_edges() -> list[ReferenceEdge]`
- `load_all_ripples() -> list[CrossVolumeRipple]`
- `_row_to_{node,edge,ripple}(row: sqlite3.Row) -> dataclass` — JSON deserialize + datetime

**2-layer separation rationale** (跟 `CostTracker` + `CostTrackerDB` 1:1):
- In-memory dict 索引 → O(1) `get_node`, O(k) `get_neighbors`, Phase 14 cascade BFS 性能 OK
- Storage 0 业务逻辑, 只管持久化 — append-only 写入 + atomic batch 兜底
- 启动时 `_load_from_storage()` bulk load, 关闭时无状态 (DB 持久化) — multi-process 安全

### 4. Error Handling + Edge Cases

**3-layer validation defense**:
1. **Dataclass `__post_init__`** — 单 entity invariants (volume, chapter, weight, description length, no self-loop)
2. **CVG `add_*`** — 跨 entity invariants (id unique, FK exists, no duplicate edge pair)
3. **Storage** — DB-level (FK enforcement, PRIMARY KEY, JSON deserialize)

**Error type strategy** (跟 codebase 5 处 precedent 一致, 0 自定义 exception class):
- `ValueError` — 所有 validation 失败
- `sqlite3.IntegrityError` — DB-level UNIQUE/FK 违反, CVG 捕获 re-raise as `ValueError`
- `json.JSONDecodeError` — payload 反序列化失败 → `ValueError("corrupt payload in {table} {id}: {e}")`
- `sqlite3.DatabaseError` — DB 损坏 / disk full → propagate (caller 决策 recovery)
- `FileNotFoundError` — DB 缺失, `__init__` `mkdir(parents=True, exist_ok=True)` + sqlite3 connect auto-create → 0 raise

**Edge cases explicit**:
| 场景 | 行为 |
|------|------|
| Empty DB 启动 | `load_all_*` returns `[]`, CVG indexes empty, `len(cvg)==0` |
| Duplicate node id | `add_node` raise `ValueError("duplicate node id: {id}")` |
| FK 找不到 | `add_edge` raise `ValueError("from_node_id not found: {id}")` |
| 边加在已存在节点 pair | `add_edge` raise `ValueError("duplicate edge between {a} and {b}")` |
| `atomic_batch` 异常 | `ROLLBACK` → re-raise original (不 wrap) |
| `atomic_batch` 成功 | `COMMIT` → fsync (synchronous=FULL) |
| 并发写 | WAL mode: 1 writer, N readers; 写写竞争 `sqlite3.OperationalError: database is locked` → caller retry (Phase 10 single-process OK) |
| 节点删除 | **Phase 10 不支持** (append-only), 0 `delete_node` method |
| payload 不可 JSON | `json.dumps` raise `TypeError`, `add_node` 不 catch → propagate |

**Logging** (跟 codebase 一致, stdlib `logging`, 0 print):
```python
logger = logging.getLogger(__name__)
logger.debug("add_node: id=%s dim=%s vol=%d", node.id, node.dimension, node.volume)
logger.warning("validation failed: %s", e)
logger.info("ripple.db initialized at %s", db_path)
```
