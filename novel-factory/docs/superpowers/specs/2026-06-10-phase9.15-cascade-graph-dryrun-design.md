## 3. Architecture

### 3.1 Layer 1 — Backend (T1 cascade BFS + storage)

**`infra/cross_volume/reference_graph.py`** (146L → ~190L, +44L):
```python
@dataclass(frozen=True)
class CascadedRipple:
    trigger_ripple_id: str
    cascade_nodes: tuple[ReferenceNode, ...]
    cascade_edges: tuple[ReferenceEdge, ...]
    cascade_actions: tuple[dict, ...]  # proposed actions generated by BFS traversal
    depth_reached: int  # 0-3, how deep BFS actually went
    generated_at: str  # ISO 8601
    bfs_algorithm_version: str = "v1"  # future-proofing for Phase 9.16 weighted BFS

class CrossVolumeReferenceGraph:
    def trigger_cascade(self, ripple: CrossVolumeRipple) -> CascadedRipple:
        """Real BFS depth 3 + visited set, replaces Phase 9.10 stub.
        Returns CascadedRipple with all downstream nodes/edges/actions collected.
        """
        visited: set[str] = {n for n in ripple.affected_nodes}
        queue: deque[tuple[str, int]] = [(n, 1) for n in ripple.affected_nodes]
        collected_nodes: list[ReferenceNode] = []
        collected_edges: list[ReferenceEdge] = []
        collected_actions: list[dict] = []
        max_depth = 0
        
        while queue:
            node_id, depth = queue.popleft()
            if depth > 3: continue
            max_depth = max(max_depth, depth)
            for neighbor in self.get_neighbors(node_id):
                if neighbor.id in visited: continue
                visited.add(neighbor.id)
                collected_nodes.append(neighbor)
                edge = self._index_by_node_edges.get(node_id, [])
                collected_edges.extend(e for e in edge if e.source_id == node_id and e.target_id == neighbor.id)
                collected_actions.append({
                    "action": "propagate", "from": node_id, "to": neighbor.id,
                    "depth": depth, "weight": neighbor.payload.get("weight", 1.0),
                })
                if depth < 3:
                    queue.append((neighbor.id, depth + 1))
        
        return CascadedRipple(
            trigger_ripple_id=ripple.id,
            cascade_nodes=tuple(collected_nodes),
            cascade_edges=tuple(collected_edges),
            cascade_actions=tuple(collected_actions),
            depth_reached=max_depth,
            generated_at=datetime.utcnow().isoformat() + "Z",
        )
```

**`ripple_cascade` 新表 schema** (additive, 0 改既有 6 表 `ripple` / `ripple_audit` / 4 既有):
```sql
CREATE TABLE IF NOT EXISTS ripple_cascade (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    trigger_ripple_id TEXT NOT NULL,
    cascade_nodes_json TEXT NOT NULL,
    cascade_edges_json TEXT NOT NULL,
    cascade_actions_json TEXT NOT NULL,
    depth_reached INTEGER NOT NULL,
    created_at TEXT NOT NULL,
    FOREIGN KEY (trigger_ripple_id) REFERENCES ripple(id) ON DELETE CASCADE
);
CREATE INDEX IF NOT EXISTS idx_ripple_cascade_trigger ON ripple_cascade(trigger_ripple_id);
```
