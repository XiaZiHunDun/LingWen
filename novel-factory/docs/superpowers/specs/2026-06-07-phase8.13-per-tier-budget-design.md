### 1. Backend (T1)

#### 1.1 `_extract_cost_by_tier` helper

**File**: `dashboard/protocols.py` (新函数, 跟 `_extract_cost_by_scenario:316` 镜像)

```python
def _extract_cost_by_tier(controller: Any) -> dict[str, float]:
    """Phase 8.13: 跟 _extract_cost_by_scenario 同模式 — 拿 controller.cost_tracker 调 cost_by_tier().

    ModelTier str enum 自动 serialize .value ('haiku'/'sonnet'/'opus') 到 JSON key.
    旧 tier 名字 (不在 enum 范围) 走 str 透传, 不抛 ValidationError (Pydantic dict[str, float]
    接受任意 str key).

    Args:
        controller: MasterController 实例 (decoupling Protocol)

    Returns:
        dict[tier_name, cost_usd] — 空 dict if cost_tracker is None / 无该属性 / 无 records
    """
    cost_tracker = getattr(controller, "cost_tracker", None)
    if cost_tracker is None:
        return {}
    try:
        # ModelTier key auto-serialize to 'haiku'/'sonnet'/'opus' via .value
        return {tier.value: float(amt) for tier, amt in cost_tracker.cost_by_tier().items()}
    except Exception as exc:
        logger.warning("cost_tracker.cost_by_tier() failed: %s", exc)
        return {}
```

**关键设计**:
- `{tier.value: ...}` 显式拆 enum → str (Pydantic v2 + use_enum_values 行为依赖 config, 显式更稳)
- `float(amt)` 防御 `Decimal` 等非 float 类型 (DB 返 Row['total'] 可能是 Decimal)
- silent degrade 跟 `_extract_cost_by_scenario` 完全对齐

### 4. Error Handling

| Error Scenario | Behavior |
|----------------|----------|
| `controller.cost_tracker is None` | `_extract_cost_by_tier` → `{}` |
| `cost_tracker.cost_by_tier()` raises | `logger.warning(...)` + return `{}` |
| `ModelTier` key 不在 enum (历史/脏数据) | `.value` 抛 ValueError → try/except wrapper 返 `{}` (不 crash) |
| Pydantic `dict[str, float]` 收 dict with str tier 名字 | 直接 serialize, 不抛 ValidationError |
| Frontend `costByTier = {}` 切到 tier mode | `hasData=false` → 显 "暂无成本数据" 占位 (复用 Phase 8.7 UX) |
| Frontend `mode` 不是 'scenario'/'tier' | `validator` 抛 warn + Vue runtime 默认 'scenario' |

**0 silent crash** — 所有 error path 走 logger.warning + empty dict/UX fallback, 跟 Phase 8.7/8.12 完全一致.
