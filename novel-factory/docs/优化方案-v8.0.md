## 一、执行摘要

### v7.0 成果
- 工程鲁棒性：5.5 → 8.0/10
- SQLite 原子更新、flock 文件锁、断点续跑
- 伏笔回收率从 40.7% 提升

### v8.0 目标
将已实现的 memory_system 集成到主工作流，解决：
- 伏笔回收率仍需提升（当前 40.7% → 目标 80%+）
- 角色状态一致性（多章节跨场景一致性）
- 写作上下文丢失（作家写新章节时缺乏上下文）

### 核心变更
```
现有流程：
  作家写章节 → 审核员审核 → 意见仓库 → 作家修改

v8.0 流程：
  作家写章节 → 记忆系统推送上下文 → 审核员审核（含记忆校验）→ 意见仓库 → 作家修改
```

### 3.2 创建记忆服务单例

**目的**：在工作流中全局访问 MemoryGateway

```python
# novel-factory/infra/memory_service.py（新增）

"""记忆服务单例 - 全局记忆系统访问点"""
from pathlib import Path
from typing import Optional
from infra.memory_system import MemoryGateway
from infra.memory_system.vector.qdrant_client import QdrantClientWrapper
from infra.memory_system.embeddings.batch_embed import BatchEmbedder
from infra.memory_system.state.character_tracker import CharacterTracker
from infra.memory_system.state.plot_thread_tracker import PlotThreadTracker
from infra.memory_system.state.timeline_manager import TimelineManager
from infra.memory_system.state.fact_base import FactBase

_memory_gateway: Optional[MemoryGateway] = None

def get_memory_gateway() -> MemoryGateway:
    """获取记忆服务单例"""
    global _memory_gateway

    if _memory_gateway is None:
        config_path = Path(__file__).parent.parent / "config" / "memory_config.yaml"
        if not config_path.exists():
            raise RuntimeError(f"Memory config not found: {config_path}")

        # 初始化各组件
        qdrant_wrapper = QdrantClientWrapper()
        embedder = BatchEmbedder()

        # 读取配置
        import yaml
        with open(config_path) as f:
            config = yaml.safe_load(f)

        character_tracker = CharacterTracker(config)
        plot_thread_tracker = PlotThreadTracker(config)
        timeline_manager = TimelineManager(config)
        fact_base = FactBase(config)

        _memory_gateway = MemoryGateway(
            qdrant_wrapper=qdrant_wrapper,
            embedder=embedder,
            character_tracker=character_tracker,
            plot_thread_tracker=plot_thread_tracker,
            timeline_manager=timeline_manager,
            fact_base=fact_base,
        )

    return _memory_gateway
```

### 3.3 工作流集成节点

#### 节点 1：作家开始新章节（auto_push_context）

**触发时机**：`run_writer.sh launch` 时

```bash

cmd_launch() {
    local chapter=${1:-""}
    local writer=${2:-""}

    # 获取记忆上下文
    local memory_context=$(python3 -c "

context = gateway.auto_push_context(chapter_num=int('$chapter'.replace('ch','')))
print(context)
" 2>/dev/null || echo "{}")

    # 将上下文传递给作家Agent
    echo "[记忆系统] 获取上下文: $memory_context"
    # ... 原有launch逻辑
}
```

**输出**：作家在开始写作前收到：
- 角色状态摘要（当前场景中的角色位置、情绪）
- 待回收伏笔（与本章相关的伏笔）
- 最近事件（最近3章的重要事件）
- 相关段落（与本章情节类似的过往段落）

#### 节点 2：一致性检查（check_consistency）

**触发时机**：`run_review.sh` 审核时

```python
# 审核流程中新增
def review_with_memory_check(chapter_content, chapter_num):
    gateway = get_memory_gateway()

    # 1. 基础审核（现有）
    issues = run_base_review(chapter_content)

    # 2. 记忆一致性检查（新增）
    consistency = gateway.check_consistency(chapter_content, chapter_num)
    if not consistency['is_consistent']:
        issues.extend(consistency['issues'])

    # 3. 角色状态验证（新增）
    character_states = gateway.get_all_characters()
    for char, state in character_states.items():
        if char in chapter_content:
            issues.extend(validate_character_state(char, state, chapter_content))

    return issues
```

# 伏笔自动登记流程
def detect_and_register_foreshadow(chapter_num, content):
    gateway = get_memory_gateway()

    # 检测伏笔关键词
    foreshadow_patterns = [
        (r'神秘组织', 'fp_mystery_org'),
        (r'古老预言', 'fp_ancient_prophecy'),
        # ... 更多模式
    ]

    for pattern, fp_id in foreshadow_patterns:
        if re.search(pattern, content):
            metadata = {
                'title': pattern,
                'planted_chapter': chapter_num,
                'keywords': [pattern],
            }
            gateway.plant_foreshadow(fp_id, metadata)
            log_audit("INFO", "FORESHADOW_PLANT", f"{fp_id} in ch{chapter_num:03d}")

# 章节发布前更新角色状态
def update_character_states(chapter_num, content):
    gateway = get_memory_gateway()

    # 提取角色位置
    locations = extract_locations(content)
    for char, loc in locations.items():
        gateway.update_character_state(char, {
            'current_location': loc,
            'last_updated_chapter': chapter_num,
        })

    # 提取角色情绪

### B. MemoryGateway API

```python
class MemoryGateway:
    def auto_push_context(chapter_num: int) -> Dict:
        """作家写章节前调用，获取上下文"""

    def query(query: str, scope: str, top_k: int) -> List[Dict]:
        """主动查询记忆"""

    def update_character_state(character: str, state: Dict) -> None:
        """更新角色状态"""

    def get_character_state(character: str) -> Optional[Dict]:
        """获取角色状态"""

    def plant_foreshadow(fp_id: str, metadata: Dict) -> None:
        """登记伏笔"""

    def update_foreshadow(fp_id: str, event_type: str, chapter: int) -> None:
        """更新伏笔状态（activate/recycle/invalidate）"""

    def check_consistency(chapter_content: str, chapter: int) -> Dict:
        """一致性检查"""
```
