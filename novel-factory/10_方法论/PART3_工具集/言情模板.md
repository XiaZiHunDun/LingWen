## 核心驱动链

```
相遇 → 心动 → 误解 → 分离 → 重逢 → 告白 → 在一起
```

言情核心是**情感递进**，所有事件服务于角色关系发展。

## 情节模板

### 相遇模式

```
1. 命中注定型：意外邂逅 → 留下深刻印象 → 命运安排多次见面
   - 示例：撞车/泼咖啡/误认人 → 互相留下坏印象 → 再次见面惊讶

2. 职场接触型：上下级/合作伙伴/竞争对手
   - 示例：新上司竟是前男友 / 谈判对手是相亲对象

3. 社交场合型：共同朋友聚会/被迫组队/邻居
   - 示例：婚礼被迫当伴郎伴娘 / 合租成为室友
```

### 心动触发

```
【女主心动瞬间】
- 他在别人都不相信她时相信了她
- 他记住了她随口说的一句话
- 他在她脆弱时展现了从未见过的温柔
- 他为她做了一个别人做不到的小事

【男主心动瞬间】
- 她在别人面前伪装坚强，在他面前露出脆弱
- 她做了一件他没想到她会做的事（善良/勇敢/倔强）
- 她的笑容/某个表情让他心跳漏拍
- 她让他感到被需要
```

### 误解与冲突

```
常见误解来源：
1. 第三人作梗：误会/欺骗/挑拨
2. 身份落差：不知道对方真实身份（总裁假装普通人）
3. 过去阴影：前任/家庭原因导致不信任
4. 性格差异：沟通方式不同导致争吵

冲突节奏：
- 第1次小冲突：心动后的第一次摩擦（考验感情基础）
- 第2次大冲突：感情升温时遭遇重大考验
- 第3次分离：误会爆发导致暂时分开
- 结局前和解：真相大白，两人坦诚相待
```

### 分离模板

```
分离类型：
1. 外部压力：家庭反对/异地/身份差距
2. 误会爆发：得知某个"真相"愤而离开
3. 自我牺牲：为了对方好选择离开
4. 事业冲突：追逐梦想导致暂时分开

分离期间：
- 双方各自反思/成长
- 有人挑拨或出现新追求者
- 一次意外重逢（不完整相遇）
- 最终相遇/和好
```

### 告白模板

```
告白时机：
1. 危机时刻：一方有危险，另一方爆发真情
2. 旅行时刻：异地/旅行中放下防备
3. 成就时刻：成功后想要与人分享
4. 夜晚时刻：星空/海边等浪漫场景

告白台词模板：
- 「我发现我好像离不开你了」
- 「我不想再错过了」
- 「让我照顾你」
- 「我等你，不管多久」
```

## 情感曲线模板

```
【典型言情曲线】

第一章：相遇（建立联系）
├─ 开端：意外相遇，留下深刻印象
├─ 发展：多次接触，关系逐渐熟悉
└─ 结尾：一个小事件让双方注意到彼此

第3-5章：心动（情感萌发）
├─ 开端：开始关注对方的细节
├─ 发展：单独相处，心跳加速
└─ 结尾：某个瞬间确认"我喜欢上他了"

第6-10章：波折（关系动摇）
├─ 开端：关系接近最甜点
├─ 发展：出现误会/挑战/竞争者
└─ 结尾：第一次分离/大冲突

第11-15章：转折（真相揭露）
├─ 开端：分别期间各自反思
├─ 发展：接近真相/成长改变
└─ 结尾：重逢/误解逐渐解开

第16-20章：高潮（感情确定）
├─ 开端：和好，感情更坚定
├─ 发展：新的挑战/更大考验
└─ 结尾：告白/确定关系

结局章：圆满
└─ 在一起后的甜蜜/婚礼/未来展望
```

## 章节钩子模板

```
【每章结尾钩子类型】

悬念钩子：
- 他突然消失，她说了一句话让他停住脚步……
- 手机屏幕亮起，显示一个陌生号码……

情感钩子：
- 他看着她，心里突然涌起一种奇怪的感觉……
- 她发现自己在期待下一次见面……

冲突钩子：
- 门被推开，进来的人让她愣住了……
- "你怎么会在这里？"她听到自己颤抖的声音……

伏笔钩子：
- 他注意到她手腕上的伤疤，若有所思……
- 她不知道，这只是更大阴谋的开始……
```

## 场景模板

### 甜场景

```
【英雄救美变体】
1. 她陷入困境（迷路/被欺负/遇到危险）
2. 他恰好出现，解救她
3. 事后她想道谢，他却说"不用谢"然后离开
4. 之后她发现他其实帮了她很多

【暗恋被发现变体】
1. 她偷偷观察他，被他发现
2. 她尴尬想逃，他拉住她
3. "你刚才在看什么？"他笑着问
4. 她的脸红透了，但心里却有些雀跃
```

### 虐场景

```
【误解型虐】
1. 她亲耳听到/看到某个"证据"
2. 她的心突然冷了
3. 她决定不再解释/不再见他
4. 真正的解释被第三人拦截

【牺牲型虐】
1. 她知道他为她放弃了什么
2. 她想阻止但无能为力
3. 她只能看着他在她面前倒下/消失
4. 分离时的承诺成为她的执念
```

## 质量检查清单

- [ ] 每章有明确的情感目标（这一章要让读者感受到什么）
- [ ] 心动瞬间要有具体细节，不要泛泛而写
- [ ] 冲突要有递进，不能每章都是小吵小闹
- [ ] 误会要在合理时机解开，不能拖太久
- [ ] 配角要有自己的动机，不是单纯作梗
- [ ] 结尾钩子要有吸引力，让读者想看下一章
- [ ] 甜蜜场景和虐心场景要交替出现
- [ ] 人物在感情中要有成长和改变
