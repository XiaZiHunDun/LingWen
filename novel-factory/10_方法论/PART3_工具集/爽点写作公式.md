# 爽点写作公式

> **版本**：v1.1
> **来源**：小说评审优化方法论_v1.1 §9.15 + 写小说流程文档
> **日期**：2026-05-19

## 核心公式：铺垫 → 压抑 → 爆发 → 反应

**通用原则**：
- 不要按比例拼积木，会扼杀自然流畅感
- 关注「动作-感受-反应」三环，而非百分比
- 质量比比例重要：铺垫够不够让读者期待？压抑是否真的让人紧张？爆发是否有力？

---

## 打脸爽写法（实操版）

```
1. 铺垫：用对手的具体动作/言语建立轻视
   - 示例：「对手冷笑一声，将一块石头丢在林夜面前」
   - 关键：对手行为要让读者感到「这人待会要被打脸」

2. 压抑：写主角的一次实质性受挫，并明确代价
   - 示例：「林夜挡下第三击，嘴角溢出血丝。若再硬接，内腑将受创」
   - 关键：不是简单的「陷入困境」，而是要让读者看到「代价是什么」

3. 转机：通过具体细节变化暗示突破
   - 示例：「林夜眼中的光芒突然变了——那是……」
   - 关键：一个具体的、可感知的变化，不是突然变强

4. 爆发：干脆有力的短句，避免长篇大招
   - 示例：「一拳。光芒炸裂。对手瞳孔骤缩。」
   - 关键：短句有力，情绪释放要快

5. 反应：重点写对手的震惊/后悔，这是爽点释放
   - 示例：「不可能！他的实力怎么会——」
   - 关键：读者想看的是对手的反应，而非主角的pose

6. 钩子：引出新的、更大的问题
```

## 逆袭爽写法（实操版）

```
1. 铺垫：建立「主角很弱/处于劣势」的认知
   - 用具体场景而非旁白：「他的法器碎了，他的灵力枯了」

2. 压抑：连续2-3次「似乎有希望但又失败」
   - 关键：每次压抑后要有小释放，让读者觉得「快成功了」再压下去

3. 转机：发现关键线索/爆发潜力/外援出现
   - 示例：「他突然想起师父说过的话……」
   - 关键：转机要合理，不能凭空出现

4. 爆发：逆转成功，情绪释放
   - 关键：逆转要干脆，不要拖沓

5. 钩子：新的威胁或更大的挑战
```

## 羁绊爽写法（实操版）

```
1. 铺垫：日常/并肩，建立关系基础
   - 关键：不要突然表白，用细节暗示感情

2. 危机：一方陷入危险，具体可感知的危险
   - 关键：危险要具体（「她被困在坍塌的矿区」），不要模糊（「她有危险」）

3. 牵挂：另一方的担忧/决心，要有具体动作
   - 示例：「林夜的手在发抖，但他还是冲了过去」
   - 关键：情绪通过动作展现，不是内心独白

4. 回应：羁绊被认可/双向奔赴
   - 关键：要有具体动作或对白，不是空泛的「她感动了」

5. 钩子：关系进一步或新考验
```

## 爽点质量检查

- 铺垫是否足够？（读者是否真的期待？不是自己觉得期待）
- 压抑是否到位？（情绪是否充分积累？不是压抑一下就爆发）
- 爆发是否有力？（情绪释放是否强烈？反应是否到位？）
- 常见失败：铺垫太长读者失去耐心 / 压抑太久没有小释放 / 爆发太短没感觉 / 忘记写对手反应

---

## 爽点穿透力检查

**爽点穿透力不足的信号**：
- 铺垫太短，读者还没进入状态就爆发了
- 压抑太轻，读者没有积累足够情绪
- 爆发太快，读者情绪还没释放就结束了
- 缺乏具体细节，只有「他赢了」「他变强了」的概括描述

**写完爽点后自问**：
- 读者读到这里，是否真的激动了？
- 还是感觉「就这？」、「不过如此」？
- 情绪释放是否充分？还是憋着没出来？

## 爽点与情感曲线对应

| 爽点类型 | 对应情感曲线 | 适用节点 |
|----------|--------------|----------|
| 打脸爽 | 压抑→爆发 | 节点8（写作期） |
| 逆袭爽 | 困境→转机→胜利 | 节点8（写作期） |
| 羁绊爽 | 日常→危机→和好 | 节点8（写作期） |
| 突破爽 | 积累→爆发→新高 | 节点8（写作期） |

## 情绪曲线与爽点映射

```
情感曲线：
├─ 开头（0-200字）：悬念/压力/好奇/紧张
├─ 发展（200-2000字）：期待↑/紧张↑/疑惑/希望
├─ 高潮（收尾前200字）：爽/哭/震惊/愤怒/感动
└─ 结尾：钩子

爽点放置：
├─ 3章一小爽（每3章有一次小情绪释放）
├─ 10章一大转折（每10章有一次重大情绪爆发）
└─ 全程埋伏笔，前后呼应
```
