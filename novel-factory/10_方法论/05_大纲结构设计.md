# 阶段3：大纲与结构设计

> **核心观点**：大纲是地图，能防止你迷路。先搭骨架，再添肉。

---

## 一、三幕式结构（新手推荐）

| 幕 | 占比 | 内容 |
|----|------|------|
| 第一幕 | 1-20% | 日常被打破，主角被迫上路 |
| 第二幕 | 20%-70% | 闯关+成长+阻碍升级+接近低谷 |
| 第三幕 | 70%-100% | 终极对决+收尾+人物蜕变 |

### 详细节点

```
第一幕（建置）
├── 日常：主角正常生活 → 展现性格+处境
├── 诱发事件：意外卷入 → 打破日常
└── 第一扇门：主角做出选择，踏入冲突（无法回头）

第二幕（对抗）
├── 尝试：主角第一次尝试解决问题 → 失败
├── 帮手+反派：遇到帮手 / 反派施压
├── 中点：重大转折（信息揭露/局势逆转）
├── 最低谷：一切崩溃，主角陷入绝境
└── 决心：主角触底反弹，决心反击

第三幕（解决）
├── 终极对决：主角直面最大阻碍
├── 关键选择：做出最重要的决定/牺牲
└── 结局：目标达成/遗憾，伏笔回收，蜕变闭环
```

## 二、8节点大纲（极简版）

适合快速梳理：

| 节点 | 内容 | 位置 |
|------|------|------|
| 1 | 主角日常 | 开端 |
| 2 | 突发意外，打破日常 | 开端 |
| 3 | 第一次尝试失败 | 发展 |
| 4 | 遇帮手，反派加压 | 发展 |
| 5 | 跌入最低谷 | 发展 |
| 6 | 醒悟，决心反击 | 转折 |
| 7 | 终极对决 | 高潮 |
| 8 | 结局收尾，人物蜕变 | 结局 |

## 三、15节拍表（进阶版）

来自经典三幕+节拍设计：

| 节拍 | 位置 | 内容 |
|------|------|------|
| 诱发事件 | ~10% | 打破日常平衡的事件 |
| 第一情节点 | ~25% | 主角主动踏入冲突 |
| 第一次答辩 | 25%-~ | 盟友帮助，应对第一考验 |
| 所有娱乐 | ~ | 相对平静期，角色关系发展 |
| 第二情节点 | ~50% | 中点，信息/局势重大转变 |
| 坏事来袭 | 50%-~ | 反派反击，局势恶化 |
| 第二答辩 | ~ | 主角重建计划，应对挫折 |
| 第三情节点 | ~75% | 跌入谷底，被逼入绝境 |
| 灵魂黑夜 | ~ | 最低点，主角几乎放弃 |
| 第四情节点 | ~90% | 找到答案，做出关键决定 |
| 高潮 | ~90% | 终极对决，核心冲突爆发 |
| 结局 | ~95% | 新平衡，变化后的人物 |

## 四、章纲写法

**每章一句话**（1-2句），只写"这章发生什么事"

| 要素 | 内容 |
|------|------|
| 本章目标 | 这章要推进什么？ |
| 出场人物 | 谁出现？ |
| 剧情走向 | 发生什么？ |
| 情绪落点 | 读者读这章的感受？ |
| 留钩子 | 章末如何让读者想翻下一页？ |

### 示例
```
第3章：本章目标=建立主角的初始能力；出场人物=主角+导师；
剧情走向=主角意外触发能力，被导师发现；情绪落点=兴奋+期待；
留钩子=导师说出"你和我见过的人都不一样"。
```

## 五、大纲检查清单

- [ ] 第一幕有没有快速建立主角+困境？（前3章决定读者去留）
- [ ] 开篇是否有抓人的钩子？（悬念/危机/反常）
- [ ] 主角的选择是主动的还是被动的？（主动更有张力）
- [ ] 每章结尾是否有"翻页钩子"？
- [ ] 伏笔有没有在后续回收？（不要埋而不收）
- [ ] 高潮是否足够"高"？（所有冲突在此爆发）
- [ ] 结局是否收拢所有主要线索？（不烂尾，不留坑）
