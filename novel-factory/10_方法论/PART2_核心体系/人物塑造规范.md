> **版本**：v1.0
> **来源**：整合写小说流程5份文档
> **日期**：2026-05-19

---

## 核心理念

**故事是人物推动的。不要只写标签，要探究内心。**

好的人物不是"完美的人"，而是"有缺陷的人做出了艰难的选择"。

---

## 一、主角设计

### 1.1 一句话主角

格式：**身份 + 性格 + 执念/短板**

示例：
- 退伍军人，创伤后应激障碍，极度害怕被抛弃
- 乡村医生，固执善良，只要认定救谁就死磕到底

### 1.2 主角必须具备

| 要素 | 说明 | 示例 |
|------|------|------|
| **欲望** | 主角最想得到什么 | 在三个月内赢得冠军 |
| **恐惧** | 主角最怕失去什么 | 再次被抛弃 |
| **缺陷** | 致命的性格缺陷或心理创伤 | 过于信任他人 |
| **弧光** | 故事结束时改变了什么 | 从懦弱到勇敢 |

### 1.3 主角设计公式

```
有缺陷的主角 + 清晰目标 + 层层阻碍 = 故事骨架
```

### 1.4 主角魅力检查（S7）

读者是否想看主角成功？检查：
- [ ] 主角的目标是否值得追求
- [ ] 主角的弱点是否让读者心疼
- [ ] 主角的努力是否让读者认可
- [ ] 主角的成长是否让读者满足

## 二、配角设计

### 2.1 配角类型

| 类型 | 功能 | 设计要点 |
|------|------|----------|
| **帮手型** | 提供帮助、推动成长 | 有自己的局限，不是全能 |
| **对手型** | 制造冲突、衬托主角 | 有合理动机，不是纯粹坏人 |
| **情报型** | 提供信息、解释背景 | 行动服务于信息传递 |
| **镜像型** | 反映主角另一种可能 | 与主角形成对比 |

### 2.2 配角设计原则

- 每人有独立动机
- 言行符合身份背景
- 与主角关系影响情节
- 暂时不要超过5个主要人物

## 三、反派设计

### 3.1 反派不是纯粹的"坏人"

反派需要有：
- **立场**：为什么这样做（合理的动机）
- **苦衷**：过去经历造成的背景
- **目标**：与主角目标冲突但不渺小

### 3.2 反派设计检查

| 检查项 | 说明 |
|--------|------|
| 动机合理 | 反派的行为有逻辑支撑 |
| 威胁真实 | 反派确实能对主角造成伤害 |
| 智慧相当 | 反派的智谋要与主角对等 |
| 不被低估 | 不要让反派显得愚蠢 |

---

## 四、人物关系网

### 4.1 关系类型

- 爱恨（亲情、友情、爱情）
- 敌对（对手、仇人、竞争者）
- 师徒（传授、引导、背叛）
- 上下级（命令、服从、反抗）

### 4.2 关系动态

人物关系不是静态的，而是动态变化的：
- 盟友可能变成敌人
- 敌人可能变成盟友
- 每一次变化都要有合理的契机

### 4.3 人物关系图谱

为每个主要人物记录：
```
角色A ←→ 角色B
├─ 关系类型：盟友/对手/爱慕/师徒...
├─ 当前状态：友好/中立/敌对
├─ 历史渊源：如何认识、发生过什么
└─ 未来可能：关系会如何发展
```

## 五、人物弧光设计

### 5.1 弧光类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **成长弧** | 从弱到强 | 懦弱少年成为英雄 |
| **堕落弧** | 从强到弱 | 英雄走向悲剧 |
| **救赎弧** | 从错到对 | 反派幡然悔悟 |
| **认知弧** | 从误解到理解 | 偏见消除 |

### 5.2 弧光设计检查

- [ ] 主角在故事结束时有什么改变
- [ ] 这种改变是否有前期铺垫
- [ ] 改变是否令人信服（不是突然顿悟）
- [ ] 改变是否与主题呼应

### 5.3 弧光与情节对应

每个重大情节转折应该对应人物的内在变化：
```
情节转折 → 人物选择 → 内在成长
```

## 六、人物小传

### 6.1 主角小传模板

```markdown
## [角色名]

### 基础信息
- 年龄：
- 职业：
- 身份：

### 性格特征
- 性格关键词（2-3个）：
- 口头禅：

### 背景故事
- 关键经历1：
- 关键经历2：

### 内心世界
- 欲望（表面想要）：
- 需求（真正需要）：
- 恐惧（最怕失去）：
- 秘密（不愿被人知道）：

### 人物弧光
- 故事开始时：
- 故事结束时：
- 关键转折点：
```

### 6.2 小传使用原则

- 小传是给自己看的，书里可能用不到
- 但创作者必须完全了解角色
- 小传帮助保持角色一致性

## 七、人物声音区分

### 7.1 声纹测试 [来源：剧本写作方法论]

**核心**：每个角色说话应该有独特的声音，遮住名字也能分辨是谁在说话。

**声纹四要素**：
- [ ] **用词习惯**（口头禅、常用词、行业用语）
- [ ] **说话风格**（简洁/啰嗦、直接/委婉、句式长短）
- [ ] **情绪表达**（激烈/平淡、高亢/低沉、喜怒哀乐的方式）
- [ ] **身份痕迹**（阶层用语、职业特征、地域口音）

**测试方法**：
1. 随机选取3段对话，遮住角色名
2. 仅通过语言风格判断是谁在说话
3. 如果无法判断，说明该角色声音不够鲜明

### 7.2 展示而非告知

| ❌ 告知 | ✅ 展示 |
|--------|--------|
| 他很紧张 | 他反复解开又扣上袖扣，咖啡洒了也没察觉 |
| 她恨他 | 她把他送的围巾剪碎，却留着碎布在抽屉里 |

## 八、与评审维度的对应

| 维度 | 人物相关检查点 |
|------|----------------|
| S1 剧情完整性 | 人物行为是否有明确目标驱动 |
| S2 逻辑自洽 | 人物行为是否符合性格设定 |
| S3 文笔风格 | 对话是否符合人物身份 |
| S4 情感共鸣 | 读者是否代入人物处境 |
| S7 主角魅力 | 读者是否想看主角成功 |
