# 评审维度体系

> **版本**：v1.1
> **来源**：小说评审优化方法论_v1.1

---

## 约束分层：Hard 与 Soft

### Hard（硬约束 - 一票否决）

违反即必须返工，与 `review_prompt.md` 对齐：

| 约束 | 说明 |
|------|------|
| **叙事主角唯一性** | 全书以林夜 + 苏琳为主线；无不合理的「接班人 / 新主角团 / 代际叙事夺舍」 |
| **禁止类情节** | 不出现「传承给新一代当主角」式 handoff |
| **修炼里程碑** | 与 `review_prompt.md` 所列章节节点一致 |
| **卷间关键衔接** | 如 ch120→121、ch240→241 等 |

Hard **不适用 1–5 分**；结论仅为 **通过 / 不通过**。

### Soft（软指标 - 可评分、可排序、可迭代）

| 代码 | 维度 | 核心问题 |
|------|------|----------|
| S1 | 剧情完整性 | 章内目标—障碍—结果是否清楚；卷内因果是否断档 |
| S2 | 逻辑自洽 | 时间线、战力、势力行为是否合理 |
| S3 | 文笔与风格 | 语言、对话、描写是否统一；角色声音区分度；世界观吸引力 |
| S4 | 情感共鸣 | 情绪曲线是否有效（至少一处读者可代入的波动） |
| S5 | 节奏控制 | 信息投放缓急、战斗/文戏比例是否服务于本章目标 |
| S6 | 可读性/吸引力 | 读者是否愿意一口气读完、是否期待下一章 |
| **S7** | **主角魅力/认同感** | 主角行为是否让读者认同；读者是否想看主角成功 |

**简单记忆**：
- S4 = 「你感动了吗？」
- S5 = 「读起来顺不顺？」
- S6 = 「你想继续读吗？」
- **S7 = 「你认同这个主角吗？你想看他成功吗？」**

## 软指标量表

每章对 **S1–S7** 打 **1–5 分**，并写一句证据（引用原文要点，勿空评）。

| 分档 | 含义 |
|------|------|
| 5 | 达到本书理想样本水平（可对齐 v1 高分章） |
| 4 | 可发，轻微润色即可 |
| 3 | 可发，但扩写/删水优先级中等 |
| 2 | 建议优先处理 |
| 1 | 结构性失败或严重出戏 |

**批次门槛（建议）**：同一卷内，随机抽样的章 **均分 ≥3.5** 且 **无两项同时 ≤2**；否则暂停扩写，先修风格指南或模板问题。

---

## 扩写基线不退（防注水硬规则）

对进入**扩写**流程的章节，须同时满足：

- 扩写后 **S1、S2 任一项不得低于扩写前该章基线分数**
- 若扩写后字数进入区间但 **S1 或 S2 低于基线**：视为 **不合格扩写**，须回退或改扩写策略

---

## 规模化重写占比预警

若落在**扩写池之外**、需 **整章级重写** 的章节占 **待处理子集**比例 **>20%**，应暂停规模化扩写，由主编评估是否切换以重写为主的子阶段。

**20% 为默认阈值**，可调整。

## 权重与优先级

| 维度 | 类型 | 权重 |
|------|------|------|
| 主角唯一性及相关禁令 | Hard | 一票否决 |
| 修炼与世界观里程碑 | Hard | 一票否决 |
| 剧情完整性 | Soft | 高（S1） |
| 逻辑自洽性 | Soft | 高（S2） |
| 可读性/吸引力 | Soft | 高（S6） |
| 文笔质量 | Soft | 中（S3） |
| 情感共鸣 | Soft | 中（S4） |
| 节奏控制 | Soft | 中（S5） |
| 主角魅力/认同感 | Soft | 高（S7） |

「高/中」用于**排期优先级**：同一阶段内先修 S1/S2/S6/S7，再打磨 S3–S5。
