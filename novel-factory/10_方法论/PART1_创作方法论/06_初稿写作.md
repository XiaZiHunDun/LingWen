# 阶段4：初稿写作

> **核心心法**：第一稿的使命是"存在"，不是"完美"。允许自己写得烂。

---

## 一、开篇写法（决定留人）

### 开篇前三原则

1. **快速出场主角** — 前3段让人知道谁是主角
2. **立刻抛出困境/危机** — 让读者知道主角面临什么
3. **直接给希望/转机** — 让读者知道主角有机会

### 开篇避雷

| ❌ 避免 | ✅ 推荐 |
|---------|--------|
| 大段背景介绍 | 通过情节带出设定 |
| 长篇景物描写 | 用景物衬托情绪 |
| 多人乱出场 | 聚焦主角，先不多线 |
| 节奏过慢 | 从变化和行动中开始 |

### 好开篇示例思路

> "他从噩梦中醒来，发现自己的手在发抖。十分钟后，他就要亲手毁掉自己经营了十年的一切。"

——一句话：主角 + 紧迫感 + 悬念 + 情绪

---

## 二、场景微观结构

每个场景都是一次"微型冲突"：

| 要素 | 问题 |
|------|------|
| **目标** | 角色进入场景想要什么？ |
| **阻碍** | 谁/什么阻止他？ |
| **转折/灾难** | 场景结束时，局面更好还是更糟？ |
| **余波** | 情绪/关系发生了什么变化？ |

**原则**：最好让角色"部分胜利但失去更多"或"彻底失败"
- 全胜 → 无张力
- 全败但有收获 → 推进情节，保持希望
- 惨败 → 设置绝境，后文反弹

### 晚进早出法则 [来源：剧本写作方法论]

**核心**：不要写角色"进门、打招呼、倒水"这些废话，直接切入冲突最激烈的那一刻；冲突解决或信息传递完，立刻切走。

| ❌ 错误示范 | ✅ 正确示范 |
|-------------|-------------|
| "他推开门，走进酒馆，找了个角落坐下，向酒保招了招手，点了杯酒。" | "酒馆里很吵。他一进门，就感觉到那道目光。" |
| "她回到家，打开灯，看到房间里一切如常，然后注意到桌上有一封信。" | "她拆开信封。里面只有一张照片——是她自己。" |
| "早上八点，他起床洗漱，穿好衣服，吃了早餐，然后出门上班。" | "他冲出家门。车钥匙呢？该死，昨晚她又把钥匙藏起来了。" |

**判断标准**：如果删掉这段文字后，读者仍然知道发生了什么，说明这段是"废话"，需要晚进或早出。

## 三、Show Don't Tell（展示而非讲述）

**核心**：不要直接说"他很愤怒"，而是写出愤怒的表现。

| ❌ 讲述 | ✅ 展示 |
|---------|--------|
| 他很紧张 | 他反复解开又扣上袖扣，咖啡洒了也没察觉 |
| 她恨他 | 她把他送的围巾剪碎，却把碎布留在抽屉里 |
| 他很有钱 | 他随手给了服务员一千块小费，眼睛都没眨一下 |

**技巧**：
- 多写**具体动作**、**感官细节**、**对话**
- 少用情绪化的形容词
- 用动作暗示心理，不用"他心想..."

## 四、叙事节奏

### 张弛结合原则

```
紧张场景（打斗/冲突）→ 呼吸空间（日常/铺垫）→ 再爆发
```

### 快节奏写法
- 短句
- 动作多
- 对话多
- 章节结尾留钩子

### 慢节奏写法
- 长句
- 描写多
- 内心独白
- 人物关系深化

**注意**：网文通常节奏偏快，不要连续3章以上没有实质性情节推进。

---

## 五、写作节奏管理

| 原则 | 说明 |
|------|------|
| **定量** | 设定每日字数目标（新手500-1000字/天） |
| **定时** | 固定时间写作，形成习惯 |
| **不回读修改** | 初稿期绝对不回头改，用`//TODO:`标记问题点 |
| **跳过写** | 卡住就跳，先写后面，空白处用`[此处待补]` |
| **按大纲写** | 别东一榔头西一棒子，按结构顺序推进 |

### 卡文解决法

1. 回看大纲，确定当前阶段目标
2. 给主角制造新麻烦/新敌人/意外变故
3. 让配角搞事，推动剧情
4. 暂停细写，先用`[场景：主角和反派在酒馆谈判]`标记，把骨架写完

---

## 六、初稿检查清单

- [ ] 开篇3章是否抓人？（前1万字决定读者去留）
- [ ] 主角目标是否清晰，读者知道主角要什么吗？
- [ ] 每个场景是否有冲突？（没有冲突的场景 = 水）
- [ ] 章节结尾是否有钩子？
- [ ] 是否按大纲顺序写，没有跑偏？
- [ ] 遇到写不下去的地方，有没有用`//TODO:`标记跳过？

---

## 七、初稿期禁止

1. ❌ 回读修改已经写过的内容
2. ❌ 停下来查资料（标记`[待查]`继续写）
3. ❌ 追求文笔（文笔是修改期的事）
4. ❌ 完美主义（先写完，再完美）
