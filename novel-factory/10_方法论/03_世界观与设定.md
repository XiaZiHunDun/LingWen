# 阶段2：世界观与设定

> **原则**：只做"够用"的设定，别堆世界观。新手最常犯的错误是一上来写几十页设定表，然后写崩。

---

## 一、极简设定表（新手照做）

### 世界/背景（只记关键）
| 维度 | 内容 |
|------|------|
| 时代 | 现代 / 古代 / 未来 / 架空 |
| 核心规则 | 有没有魔法/科技/异能/制度限制 |
| 主要场景 | 1-2个重点场景，不要超过3个 |
| 时间跨度 | 三天？三个月？三十年？ |

### 示例：都市异能小说
```
时代：现代都市
核心规则：存在"觉醒者"，能操控元素异能，但受到严格管控
主要场景：上海陆家嘴 / 城郊工业区
时间跨度：3个月内
```

### 示例：古代玄幻
```
时代：架空古代（三国-like）
核心规则：存在"灵气"，修士可修炼突破境界
主要场景：世俗皇城 + 宗门山门
时间跨度：2年
```

## 二、设定优先级

| 优先级 | 内容 | 说明 |
|--------|------|------|
| 必须有 | 核心规则 | 这个世界"能做什么/不能做什么" |
| 必须有 | 主要场景 | 故事主要发生在哪 |
| 最好有 | 势力格局 | 谁掌权、谁反抗、谁中立 |
| 可以有 | 历史背景 | 1-2段即可，不用写完整编年史 |
| 暂不需要 | 详细地理 | 需要时再补充 |
| 暂不需要 | 完整社会结构 | 先写人物，社会自然浮现 |

## 三、信息释放节奏

### 问题：新手容易"开篇堆设定"

**错误示范**：
> "这是一个魔法横行的世界，已经延续了三千年的索罗斯帝国统治着整片大陆……"（300字背景介绍）

**正确做法**：
- 设定随情节展开，不是开篇集中说明
- 通过角色行动/对话/冲突来"展示"设定
- 如果必须说明，放在场景里，让角色带出信息

### 示例：好设定展示 vs 坏设定展示

**❌ 坏**（讲述）：
> "他是一名火系觉醒者，能力评级为B，在觉醒者管理条例中需要登记。"

**✅ 好**（展示）：
> "管理员扫了一眼他的证件，手续停了。'B级？'她压低声音，'那你应该知道最近查得紧，别在市区用。'"

## 四、特殊规则设定（异能/魔法/科技）

如果是幻想题材，需要设定核心规则：

### 必要维度
1. **力量来源** — 魔法从哪里来？血统？修炼？科技？
2. **使用代价** — 能力越强，代价越大（这是防止主角开挂的根本）
3. **限制条件** — 什么情况下能力会失效/减弱？
4. **社会对它的态度** — 自由使用？严格管控？地下交易？

### 示例：火系异能设定
```
力量来源：觉醒者体内蕴含"火种"，通过情绪波动激发
使用代价：每次使用会消耗生命力，过度使用会"燃尽"
限制条件：在禁火区无法使用，特定道具可压制火种
社会态度：觉醒者需要登记，异能犯罪量刑加倍
```

## 五、设定检查清单

- [ ] 核心规则能一句话说清
- [ ] 设定不影响情节推进（设定是为故事服务的）
- [ ] 主要角色都知道主要规则（作者必须比角色更懂这个世界）
- [ ] 力量体系有代价（无代价的能力 = 无意义的故事）
- [ ] 不要堆砌设定（先写人物，让社会自然浮现）
