# 创作节奏SOP

> **版本**：v1.0
> **日期**：2026-05-19
> **目的**：标准化创作节奏管理，提高日更稳定性

---

## 核心理念

**稳定写作节奏比爆发式创作更有效。**

设定每日目标比"爆更5000然后弃文"更靠谱。

---

## 一、每日创作流程

### 1.1 写作前（5分钟）

1. **回顾大纲**：确定今天要写哪部分
2. **设定目标**：明确今天要完成的内容
3. **仪式感**：固定时间、固定地点、一杯特定的茶

### 1.2 写作中

```
写作节奏：
├─ 场景写作 → 场景写作 → 场景写作
├─ 保持流畅性，不追求完美
├─ 先完成，再完善
└─ 记录修改想法但不停笔
```

**关键原则**：
- 禁止回读修改
- 卡住就跳：用 `//TODO: 这里要改//` 标记
- 保持流畅，不要停下来

### 1.3 写作后（5分钟）

1. **记录进度**：今天写了多少字
2. **记录问题**：有什么需要明天处理的
3. **预览明天**：确定明天从哪开始

## 二、目标设定

### 2.1 日更目标建议

| 阶段 | 字数建议 | 说明 |
|------|----------|------|
| 新手 | 500-800字/天 | 养成习惯为主 |
| 进阶 | 1000-1500字/天 | 稳定输出 |
| 熟练 | 2000-3000字/天 | 高效产出 |

### 2.2 周更目标建议

| 类型 | 字数建议 |
|------|----------|
| 日更小说 | 7000-10000字/周 |
| 周更小说 | 5000-8000字/周 |

### 2.3 阶段目标

```
阶段目标：
├─ 30天：完成第一卷（3-5万字）
├─ 90天：完成第一本书（30-50万字）
└─ 持续：稳定输出，形成创作节奏
```

## 三、卡文应对

### 3.1 卡文诊断

| 症状 | 原因 | 解决方案 |
|------|------|----------|
| 不知道写什么 | 情节不明确 | 回看大纲，确定当前阶段目标 |
| 不知道怎么写 | 细节不清晰 | 给主角制造麻烦/敌人/意外 |
| 写不下去 | 情绪/体力 | 休息，不要硬撑 |
| 不想写 | 疲劳/倦怠 | 降低目标，保持手感 |

### 3.2 卡文处理流程

```
卡文 → 回看大纲 → 确定目标 → 制造麻烦 → 继续写
```

**具体方法**：
1. 回看大纲，确定当前阶段目标
2. 给主角制造麻烦、敌人、意外变故
3. 让配角搞事，推动剧情
4. 暂停细写，先把剧情骨架写完

### 3.3 卡文预防

```
预防措施：
├─ 提前一天想好明天要写的内容
├─ 保持情节有足够张力
├─ 设定合理的写作时间
└─ 避免过度疲劳
```

## 四、创作状态管理

### 4.1 状态判断

| 状态 | 表现 | 处理 |
|------|------|------|
| 最佳状态 | 思路清晰，写作流畅 | 趁热打铁，多写 |
| 正常状态 | 思路基本清晰 | 稳定输出，保持目标 |
| 低效状态 | 思路卡顿，效率低 | 降低目标，保持手感 |
| 疲劳状态 | 完全写不动 | 休息，不要硬撑 |

### 4.2 状态调整

```
状态调整：
├─ 最佳→保持：不要过度消耗
├─ 正常→最佳：调整节奏/环境
├─ 低效→正常：降低目标，专注写完
└─ 疲劳→恢复：停止写作，充分休息
```
