# 每日创作Review规范

> **版本**：v1.0
> **来源**：整合电影制作流程文档的"每日审片"制度
> **日期**：2026-05-19

---

## 核心理念

**每日复盘比事后补救更有效。**

借鉴电影制作的"每日审片"制度，在每日创作结束后进行自检，及时发现问题，避免问题积压到后期难以解决。

---

## 一、每日自检清单

### 1.1 创作完成自检

每次创作结束后，检查：

```markdown
## 每日自检

### 日期：YYYY-MM-DD
### 字数：____ / 目标 ____

□ 本章/本节目标是否达成
□ 是否有新问题需要记录
□ 章节是否留有钩子（章末悬念）
□ 字数是否符合当日目标
□ 是否有逻辑漏洞需要后续处理
□ 人物行为是否一致
□ 是否有新的伏笔需要记录

遗留问题：
-

下次创作重点：
-
```

### 1.2 问题分类

发现问题后，分类记录：

```markdown
## 问题记录

### 日期：YYYY-MM-DD

#### 逻辑问题（需后续修复）
- [问题描述]
- [影响范围：第X章]

#### 人物问题（需后续修复）
- [问题描述]
- [影响范围：第X章]

#### 节奏问题（需后续调整）
- [问题描述]
- [影响范围：第X章]

#### 灵感记录（可选追加）
- [灵感描述]
- [可使用位置]
```

## 三、周审机制

### 3.1 每周复盘（增强版）

每周结束前进行系统性复盘：

```markdown
## 周度复盘

### 周次：第X周（YYYY-MM-DD 至 YYYY-MM-DD）
### 周目标：____字
### 周完成：____字
### 达成率：____%

### 节奏分布统计
| 节奏类型 | 章节数 | 占比 |
|----------|--------|------|
| P1（快） | X | XX% |
| P2（中） | X | XX% |
| P3（慢） | X | XX% |

### 完成好的地方
-

### 需要改进的地方
-

### 连续性问题（重点关注节奏）
-

### 下周调整计划
-
```

### 3.2 周度检查清单（增强版）

```markdown
### 周度检查清单

□ 本周目标是否达成
□ 创作节奏是否稳定
□ 节奏分布是否合理（P1:P2:P3 ≈ 4:4:2）
□ 是否有连续3章以上P1（需插入P3呼吸）
□ 是否有连续2章以上P3（需插入P1激活）
□ 版本记录是否更新
□ 遗留问题是否积累过多

版本状态：vX.X.X
健康度：□ 健康 □ 需关注 □ 需干预
```

### 3.3 问题升级机制

如果连续两周出现同一问题，需要升级处理：

| 问题类型 | 升级条件 | 处理方式 |
|----------|----------|----------|
| 逻辑漏洞 | 同一类问题出现≥3次 | 暂停创作，专项修复 |
| 人物不一致 | 连续2周人物行为异常 | 重新审视人物设定 |
| 节奏失控 | 连续2周节奏过快/过慢 | 调整大纲或目标 |
| 卡文 | 连续3天无法正常创作 | 回看大纲，降低目标 |

## 四、版本更新规范

### 4.1 何时更新版本号

| 场景 | 版本变更 | 说明 |
|------|----------|------|
| 完成大纲锁定 | v1.0.0 | 进入初稿阶段 |
| 每完成一卷初稿 | +0.1.0 | 阶段内小版本 |
| 每日修订（积累5次以上） | +0.0.1 | 修订次数 |
| 完成初稿锁定 | v2.0.0 | 进入修改阶段 |
| 完成修改锁定 | v3.0.0 | 进入润色阶段 |
| 完成定稿 | v5.0.0 | 最终版本 |

## 五、Review与评审的衔接

```
每日Review → 发现问题 → 周审 → 周期性评估 → 评审前置检查 → 锁定
```

### 5.1 Review发现的问题流向

```
每日Review记录的问题
    ↓
周审评估严重程度
    ↓
严重问题 → 触发评审前置检查
轻微问题 → 纳入下次创作改进
```

### 5.2 进入评审前的准备

在进入评审前，必须确认：
- [ ] 每日Review记录完整
- [ ] 周度复盘已完成
- [ ] 问题已分类处理
- [ ] 版本记录已更新
- [ ] 达到锁定标准

## 七、习惯养成

### 7.1 固定Review时间

```
推荐节奏：
- 每日创作后10分钟：快速自检
- 每周结束时30分钟：系统性复盘
- 每月结束时1小时：阶段性评估
```

### 7.2 Review优先级

1. 先记录问题（不急于解决）
2. 再评估影响（判断严重程度）
3. 最后制定计划（安排处理时间）
