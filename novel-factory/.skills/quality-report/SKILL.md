# /quality-report - 质量报告

## 功能

运行所有质量检查器，生成汇总报告。

## 使用方式

```
/quality-report
/quality-report ch001-ch050
/quality-report all
```

## 执行流程

1. **解析参数**
   - 无参数：检查上次运行的待处理问题
   - 指定范围：只检查指定章节
   - all：检查所有章节

2. **运行检查器**
   - 命名一致性
   - 内容完整性
   - 人物状态
   - 时间线
   - 情感节奏
   - 等

3. **生成报告**
   - 按严重程度分类
   - 按章节分组
   - 提供修复建议

## 输出格式

```
=== 质量检查报告 ===

检查范围: ch001-ch050
检查时间: 2026-05-19

总问题数: 23
  P0 (严重): 2
  P1 (中等): 15
  P2 (轻微): 6

=== 按章节统计 ===
ch012: [P0] 人物状态矛盾 - 苏琳性别不一致
ch023: [P1] 命名不一致 - 文件名与标题不匹配
...

=== 建议 ===
1. 优先修复 P0 问题（2个）
2. 按章节批量修复 P1 问题
```
