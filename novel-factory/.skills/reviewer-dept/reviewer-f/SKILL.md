## 审核维度：S6 可读性

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 平均句长 | 15-25字 | 句子长度统计 |
| 生僻词比 | <5% | 生僻词统计 |
| 段落长度 | 每段≤150字 | 段落长度统计 |
| 过渡自然 | 无跳跃感 | 阅读体验评估 |
| 视角清晰 | 无视角混乱 | 视角一致性 |
| 章节钩子 | 能吸引继续阅读 | 开头效果评估 |

### 量化指标

```python
# S6评分计算
S6_SCORE = {
    "平均句长": 20,     # >35字=0, 15-25=20
    "生僻词比": 20,     # >10%=0, <5%=20
    "段落长度": 20,     # >300字/段=0, ≤150=20
    "过渡流畅": 20,     # 跳跃=0，自然=20
    "视角清晰": 20,     # 混乱=0，清晰=20
}
S6_THRESHOLD = 70  # 及格线
```
