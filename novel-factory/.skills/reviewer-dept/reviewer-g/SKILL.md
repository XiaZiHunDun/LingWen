## 审核维度：S7 主角魅力

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 性格鲜明 | 性格描述≥5特征 | 性格特征统计 |
| 出场覆盖 | 出场率≥70% | 场景统计 |
| 决策影响 | 决策影响≥3场景 | 决策事件统计 |
| 魅力指数 | 读者调研≥7/10 | 定性评估 |
| 缺点合理 | 缺点真实非功能性 | 合理性评估 |

### 量化指标

```python
# S7评分计算
S7_SCORE = {
    "性格鲜明": 20,     # <3特征=0, ≥5=20
    "出场覆盖": 20,     # <50%=0, ≥70%=20
    "决策影响": 20,     # <2=0, ≥3=20
    "魅力指数": 20,     # <5=0, ≥7=20
    "缺点合理": 20,     # 功能性=0，真实=20
}
S7_THRESHOLD = 75  # 及格线
```
