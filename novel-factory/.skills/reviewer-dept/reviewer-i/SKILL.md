## 核心能力

| 能力 | 说明 |
|------|------|
| 伏笔埋设识别 | 识别隐蔽的伏笔元素 |

## 审核维度：伏笔追踪

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 伏笔埋设 | 每章≥1个有效伏笔 | 伏笔识别 |
| 伏笔回收率 | ≥60% | 回收统计 |
| 逻辑关联 | 关联合理 | 因果链验证 |
| 伏笔密度 | 每10章5-8个 | 密度统计 |
| 隐蔽程度 | 读者无法直接预测 | 隐蔽性评估 |
| 揭示充分 | 回收时有足够信息 | 揭示效果评估 |

### 量化指标

```python
# 伏笔追踪评分
FORESIGHT_SCORE = {
    "埋设识别": 20,     # 无伏笔=0，≥1=20
    "回收率": 20,       # <40%=0, ≥60%=20
    "逻辑关联": 20,     # 无关=0，关联=20
    "密度评估": 20,     # 稀疏/过密=0，适中=20
    "隐蔽程度": 20,     # 过于明显=0，隐蔽=20
}
FORESIGHT_THRESHOLD = 75  # 及格线
```
