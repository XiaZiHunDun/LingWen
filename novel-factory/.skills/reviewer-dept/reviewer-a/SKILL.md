## 核心能力

| 能力 | 说明 |

| 目标-障碍-结果链分析 | 验证每个情节点的OBR结构 |

## 审核维度：S1 剧情完整性

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 目标明确性 | 每章有目标 | 章节目标描述清晰 |
| 障碍相关性 | 障碍→目标因果链 | 逻辑推导检验 |
| 结果推导性 | 结果←障碍 | 结果有充分铺垫 |
| 章节独立性 | 目标可独立评价 | 避免"无意义章节" |

### 量化指标

```python
# S1评分计算
S1_SCORE = {
    "目标明确": 25,      # 目标模糊=0，清晰=25
    "障碍相关": 25,      # 无关=0，相关=25
    "结果推导": 25,      # 无推导=0，充分=25
    "伏笔服务": 25,      # 独立=0，服务主线=25
}
S1_THRESHOLD = 80  # 及格线
```

## 输出规范

```markdown
## S1审核报告

审核员：审核员A
章节：chXXX
时间：2026-05-20

### S1 剧情完整性评分：85/100

#### 目标明确性
- 章节目标：主角逃离追兵
- 评分：22/25
- 问题：无

#### 障碍相关性
- 主要障碍：追兵封锁、体力耗尽
- 评分：21/25
- 问题：体力耗尽描写稍简

#### 结果推导
- 结果：主角借助神秘力量逃脱
- 评分：22/25
- 问题：神秘力量未提前铺垫

#### 伏笔服务主线
- 伏笔：神秘力量（第一章埋设）
- 评分：20/25
- 问题：伏笔与本章结果关联稍弱

### 结论
通过（85≥80）
建议：加强神秘力量的前期暗示
```
