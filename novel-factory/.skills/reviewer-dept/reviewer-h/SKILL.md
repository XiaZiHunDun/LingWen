## 审核维度：S8 人物弧光

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 弧光完整 | 四阶段完整 | 阶段识别 |
| 铺垫充分 | 转变前≥3铺垫 | 铺垫事件统计 |
| 转折有力 | 转折点有冲击力 | 事件强度评估 |
| 循序渐进 | 变化逻辑清晰 | 渐变vs突变 |
| 主题呼应 | 与主线主题关联 | 主题分析 |
| 回收验证 | 伏笔回收印证弧光 | 伏笔关联 |

### 量化指标

```python
# S8评分计算
S8_SCORE = {
    "弧光完整": 20,     # 缺失阶段=0，四阶段完整=20
    "铺垫充分": 20,     # <2=0, ≥3=20
    "转折有力": 20,     # 平淡=0，有力=20
    "循序渐进": 20,      # 突变=0，渐变=20
    "主题呼应": 20,      # 无关=0，关联=20
}
S8_THRESHOLD = 70  # 及格线
```

## 输出规范

```markdown
## S8审核报告

审核员：审核员H
章节：chXXX
时间：2026-05-20

### S8 人物弧光评分：76/100

#### 弧光阶段完整性
- 阶段：出现→发展→转变（进行中）
- 评分：14/20
- 问题：转变阶段尚未完成

#### 铺垫充分度
- 铺垫事件：3个（师父教诲/同伴牺牲/自我怀疑）
- 评分：16/20
- 问题：第2个铺垫稍显仓促

#### 转折有力度
- 转折点：决定与敌人同归于尽
- 评分：16/20
- 问题：无

#### 循序渐进
- 评估：从被动接受→主动选择
- 评分：15/20
- 问题：部分读者觉得变化太快

#### 主题呼应
- 主题：「牺牲与成长」
- 评分：15/20
- 问题：无

### 结论
通过（76≥70）
建议：增加角色内心挣扎的描写
```
