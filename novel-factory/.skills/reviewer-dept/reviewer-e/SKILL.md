## 审核维度：S5 节奏控制

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 信息密度 | 每千字≥50个有效信息点 | 信息点统计 |
| 冲突频率 | 每章≥1个有效冲突 | 冲突事件统计 |
| 高潮间距 | 3-5章一个中等高潮 | 高潮分布 |
| 张弛结合 | 紧张≥30%，缓冲≥20% | 场景类型分布 |

### 量化指标

```python
# S5评分计算
S5_SCORE = {
    "信息密度": 25,      # <30=0, ≥50=25
    "冲突频率": 25,      # 0=0, ≥2=25
    "高潮间距": 25,      # >6=0, 3-5=25
    "张弛结合": 25,      # 失衡=0，均衡=25
}
S5_THRESHOLD = 75  # 及格线
```

## 输出规范

```markdown
## S5审核报告

审核员：审核员E
章节：chXXX
时间：2026-05-20

### S5 节奏控制评分：80/100

#### 信息密度
- 信息点：58个/千字
- 评分：22/25
- 问题：部分过渡段信息偏少

#### 冲突频率
- 冲突数：1个（与反派对峙）
- 评分：21/25
- 问题：无

#### 高潮间距
- 距上次高潮：4章
- 评分：20/25
- 问题：略超4章

#### 张弛结合
- 紧张场景：35%
- 缓冲场景：25%
- 评分：17/25
- 问题：缓冲略少

### 结论
通过（80≥75）
建议：增加一场缓冲戏份
```
