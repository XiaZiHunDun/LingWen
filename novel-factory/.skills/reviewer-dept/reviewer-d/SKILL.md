## 审核维度：S4 情感共鸣

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 转折点数 | ≥1个/章 | 情绪变化点 |
| 情绪方差 | ≥0.05 | 波动幅度 |
| 高潮强度 | ≥7/10 | 情绪峰值 |
| 共鸣深度 | 读者反应 | 定性评估 |

### 量化指标

```python
# S4评分计算
S4_SCORE = {
    "转折点": 25,      # 0个=0, ≥2=25
    "情绪方差": 25,    # <0.03=0, ≥0.05=25
    "高潮强度": 25,    # <5=0, ≥7=25
    "共鸣深度": 25,    # 无代入=0，强代入=25
}
S4_THRESHOLD = 75  # 及格线
```
