## 审核维度：S3 文笔风格

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| Shannon指数 | ≥2.5 | 词频统计 |
| 句式种类 | ≥6种 | 句型分类 |
| 单一占比 | <40% | 主句型统计 |
| 描写均衡 | 动作/心理/环境 | 描写类型分布 |

### 量化指标

```python
# S3评分计算
S3_SCORE = {
    "Shannon指数": 25,      # <2.0=0, ≥2.5=25
    "句式变化": 25,         # <4种=0, ≥6种=25
    "单一占比": 25,         # >50%=0, <40%=25
    "描写均衡": 25,         # 严重失衡=0，均衡=25
}
S3_THRESHOLD = 80  # 及格线
```

## 输出规范

```markdown
## S3审核报告

审核员：审核员C
章节：chXXX
时间：2026-05-20

### S3 文笔风格评分：82/100

#### Shannon指数
- 计算值：2.7
- 评分：23/25
- 问题：部分段落词汇重复

#### 句式变化
- 句式种类：7种（陈述/疑问/感叹/短句/长句/被动/倒装）
- 评分：22/25
- 问题：无

#### 单一占比
- 主句型占比：38%（陈述句）
- 评分：20/25
- 问题：略超35%阈值

#### 描写均衡
- 动作描写：40%
- 心理描写：35%
- 环境描写：25%
- 评分：17/25
- 问题：环境描写偏少

### 结论
通过（82≥80）
建议：增加环境描写比重
```
