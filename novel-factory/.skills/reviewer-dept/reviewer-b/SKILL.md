## 审核维度：S2 逻辑自洽

### 审核标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 时间线一致 | 无时间悖论 | 事件顺序验证 |
| 空间一致 | 位置/方向正确 | 地图验证 |
| 能力边界 | 不超出设定 | 能力对比 |
| 因果完整 | 原因充分→结果 | 逻辑推导 |

### 量化指标

```python
# S2评分计算
S2_SCORE = {
    "时间线": 25,      # 有悖论=0，一致=25
    "空间感": 25,      # 混乱=0，清晰=25
    "能力边界": 25,    # 超出=0，合理=25
    "因果链": 25,      # 因果不足=0，充分=25
}
S2_THRESHOLD = 85  # 及格线（逻辑要求更严）
```
