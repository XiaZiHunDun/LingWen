# 结构设计 Skill

## 功能

设计小说的整体结构，使用节拍表模板将创意落地为可执行的故事蓝图。

## 核心原则

**"没有结构的灵感只是一盘散沙"**
**"绝不要没有大纲就直接写正文，否则必卡壳"**

---

## 方法论嵌入

使用 `小说方法论/tools/节拍表模板.md` 作为执行标准。

---

## 三幕剧结构概览

| 幕 | 占比 | 核心任务 | 关键节点 |
|----|------|---------|---------|
| **第一幕（建置）** | 1-25% | 建立世界、引入冲突 | 日常→催化剂→第一扇门 |
| **第二幕（对抗）** | 25-75% | 主角应对阻碍、成长 | 尝试→失败→中点→低谷 |
| **第三幕（结局）** | 75-100% | 终极对决、解决问题 | 高潮→解决→新平衡 |

---

## 15个节拍点

### 第一幕：建置（1-25%）

#### 节拍1：开场画面（Opening Image）

**任务**：展现主角"日常"的世界状态。

| 检查项 | 说明 |
|--------|------|
| 情绪基调 | 为全书定调（压抑/欢快/悬疑） |
| 主角状态 | 他现在是什么样的人？ |
| 世界呈现 | 让读者知道这是什么类型的故事 |

#### 节拍2：主题呈现（Theme Stated）

**任务**：通过一段对话或场景，隐晦点题。

**注意**：不是直接说出来，而是让观众感受到"这个故事在探讨什么"。

#### 节拍3：铺垫（Setup）

**任务**：介绍主角、他的缺陷、他的朋友/家人，以及他所缺失的东西。

| 要素 | 内容 |
|------|------|
| 主角日常 | 他现在的生活是什么样的？ |
| 核心缺陷 | 他的弱点是什么？ |
| 重要关系 | 谁在他身边？ |
| 所缺之物 | 他内心真正需要什么？ |

#### 节拍4：催化剂（Catalyst / 激励事件）

**任务**：一件突发事件，彻底打破主角的日常生活。

**关键**：将一种"不得不解决"的困境摆在主角面前。

#### 节拍5：第一幕终点（Break into Two / 第一扇门）

**任务**：主角做出一个不可撤销的决定，进入"新世界"，再也无法回头。

**关键**：这个决定必须是由主角主动做出的，不是被迫的。

---

### 第二幕：对抗（25-75%）

#### 节拍6：B故事（Band Story / 副线）

**任务**：通常是爱情、友情等情感线，承载主题，帮助主角成长。

#### 节拍7：游戏时间（Fun and Games / 玩乐时间）

**任务**：展示"前提的承诺"——观众最想看的有趣部分。

| 检查项 | 说明 |
|--------|------|
| 核心卖点 | 这个故事最吸引人的部分是什么？ |
| 冲突升级 | 阻碍越来越大，主角越来越主动 |
| 节奏感 | 不要一直紧张，要给读者喘息 |

#### 节拍8：中点（Midpoint）

**任务**：故事发生重大转折。

| 类型 | 说明 |
|------|------|
| 伪胜利 | 主角以为成功了，但其实代价惨重 |
| 假失败 | 主角以为彻底失败了，但发现还有希望 |
| 重大揭示 | 得知关键信息，改变全局认知 |

#### 节拍9：反派逼近（Bad Guys Close In）

**任务**：内部矛盾、外部压力同时收紧。

#### 节拍10：一无所有（All Is Lost / 灵魂黑夜）

**任务**：主角遭遇致命打击，失去最珍贵的东西（或人），彻底绝望。

#### 节拍11：黑夜黎明（Dark Night of the Soul）

**任务**：绝望中找到最后一丝希望或新的解决方案。

---

### 第三幕：结局（75-100%）

#### 节拍12：第三幕衔接点（Break into Three / 第二扇门）

**任务**：在绝望/认知转变后，主角做出最终决定，准备终极对决。

#### 节拍13：高潮（Finale / 高潮对决）

**任务**：主角与新/旧自我或反派进行终极对决，解决核心矛盾。

#### 节拍14：终场画面（Final Image）

**任务**：展现主角"新"的世界，与开场画面形成鲜明对比。

## 输出格式

### 节拍表

```markdown
# {卷/阶段} 节拍表

## 基本信息
- 章节范围：chXXX-chYYY
- 结构类型：三幕剧 / 救猫咪15节拍
- 字数目标：

## 节拍点检查清单

### 第一幕（建置，1-25%）
- [ ] 节拍1-开场画面：主角日常世界
- [ ] 节拍2-主题呈现：隐晦点题
- [ ] 节拍3-铺垫：缺陷+关系+所缺
- [ ] 节拍4-催化剂：打破日常的事件
- [ ] 节拍5-第一扇门：主角主动进入新世界

### 第二幕（对抗，25-75%）
- [ ] 节拍6-B故事：情感副线
- [ ] 节拍7-游戏时间：核心卖点展示
- [ ] 节拍8-中点：重大转折
- [ ] 节拍9-反派加压：内外部压力收紧
- [ ] 节拍10-一无所有：致命打击
- [ ] 节拍11-黑夜黎明：绝望中的希望

### 第三幕（结局，75-100%）
- [ ] 节拍12-第三幕衔接点：最终决定
- [ ] 节拍13-高潮：终极对决
- [ ] 节拍14-终场画面：新平衡，证明改变

## 关键节点量化

| 节拍 | 章节位置 | 预期字数 | 核心功能 |
|------|---------|---------|---------|
| 催化剂 | chXX（5%处） | ~3000字 | 建立核心冲突 |
| 第一扇门 | chXX（25%处） | ~3000字 | 进入新世界 |
| 中点 | chXX（50%处） | ~3000字 | 重大转折 |
| 一无所有 | chXX（75%处） | ~3000字 | 情绪最低点 |
| 高潮 | chXX（95%处） | ~3000字 | 终极对决 |
```

## 常见结构问题

| 问题 | 症状 | 解决方案 |
|------|------|---------|
| **第二幕塌陷** | 中段拖沓、重复 | 增加中点转折、升级阻碍 |
| **高潮无力** | 对决太短、缺乏张力 | 延长高潮、增加代价 |
| **弧光断裂** | 主角没变化或突变 | 逐节点检查，补充中间变化 |
| **反派太弱** | 最终对决缺乏悬念 | 给反派增加筹码或智慧 |
| **B故事脱节** | 副线与主线无关 | 让副线承载主题论证 |
