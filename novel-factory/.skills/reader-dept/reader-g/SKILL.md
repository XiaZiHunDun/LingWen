## 阅读维度：伏笔感知

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 伏笔发现 | 能发现多少伏笔 | 伏笔统计 |
| 回收预期 | 能否预测回收方向 | 读者反馈 |
| 伏笔密度 | 是否过于密集 | 密度分析 |
| 隐蔽效果 | 是否足够隐蔽 | 隐蔽性评估 |

### 量化指标

```python
# 伏笔感知评分
FORESIGHT_SCORE = {
    "伏笔发现": 25,     # 0=0, ≥3=25
    "回收预期": 25,     # 无法预测=0，能预测=25
    "伏笔密度": 25,     # 过密/稀疏=0，适中=25
    "隐蔽效果": 25,     # 过于明显=0，隐蔽=25
}
FORESIGHT_THRESHOLD = 75
```
