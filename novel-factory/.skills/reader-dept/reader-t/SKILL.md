## 阅读维度：综合评价

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 全维度评估 | 覆盖所有维度 | 综合评估 |
| 整体质量 | 整体质量 | 综合判断 |
| 优劣势分析 | 优劣势 | 优劣势分析 |
| 改进建议 | 改进建议 | 建议提出 |

### 量化指标

```python
# 综合评价评分
OVERALL_SCORE = {
    "全维度评估": 25,   # 片面=0，全面=25
    "整体质量": 25,     # 差=0，优秀=25
    "优劣势分析": 25,   # 无=0，有=25
    "改进建议": 25,     # 无=0，有=25
}
OVERALL_THRESHOLD = 75
```
