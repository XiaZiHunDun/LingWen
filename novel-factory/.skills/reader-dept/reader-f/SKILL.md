## 阅读维度：情感调动

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 情绪感染力 | 读者情绪波动 | 读者反馈 |
| 泪点触发 | 感人场景效果 | 读者反馈 |
| 燃点触发 | 热血场景效果 | 读者反馈 |
| 情感深度 | 情感是否深刻 | 场景分析 |

```python
# 情感调动评分
EMOTION_SCORE = {

}
EMOTION_THRESHOLD = 75
```
