## 阅读维度：人物共鸣

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 代入感 | 读者能代入主角 | 问卷调查 |
| 情感连接 | 读者关心角色命运 | 读者反馈 |
| 移情反应 | 对角色遭遇的情感波动 | 读者访谈 |
| 角色认同 | 认同角色的选择 | 读者反馈 |

### 量化指标

```python
# 人物共鸣评分
EMPATHY_SCORE = {
    "代入感": 25,       # 无法代入=0，完全代入=25
    "情感连接": 25,     # 无感=0，关心=25
    "移情反应": 25,     # 无波动=0，强烈=25
    "角色认同": 25,     # 不认同=0，认同=25
}
EMPATHY_THRESHOLD = 75
```
