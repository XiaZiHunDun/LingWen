## 阅读维度：专业度

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 行业准确性 | 行业描写是否准确 | 专业分析 |
| 专业细节 | 细节是否可信 | 细节评估 |
| 细节真实感 | 细节是否真实 | 真实感评估 |
| 合理夸张 | 夸张是否合理 | 夸张评估 |

### 量化指标

```python
# 专业度评分
EXPERTISE_SCORE = {
    "行业准确性": 25,   # 不准=0，准确=25
    "专业细节": 25,     # 虚假=0，可信=25
    "细节真实感": 25,   # 虚假=0，真实=25
    "合理夸张": 25,      # 荒谬=0，合理=25
}
EXPERTISE_THRESHOLD = 75
```
