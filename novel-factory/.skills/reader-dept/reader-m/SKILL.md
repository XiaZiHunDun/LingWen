## 阅读维度：期待感

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 追读动力 | 追读意愿强度 | 读者反馈 |
| 下一章期待 | 期待下一章程度 | 读者反馈 |
| 悬念留存 | 悬念让人想继续 | 悬念分析 |
| 整体吸引力 | 整体吸引力 | 综合评估 |

### 量化指标

```python
# 期待感评分
ANTICIPATION_SCORE = {
    "追读动力": 25,     # 无=0，强=25
    "下一章期待": 25,   # 无=0，强=25
    "悬念留存": 25,     # 无=0，强=25
    "整体吸引力": 25,   # 弱=0，强=25
}
ANTICIPATION_THRESHOLD = 75
```
