## 阅读维度：世界观

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 设定可信度 | 设定符合逻辑 | 逻辑分析 |
| 沉浸感 | 读者沉浸程度 | 读者反馈 |
| 细节丰富度 | 细节是否充分 | 细节统计 |
| 逻辑自洽 | 世界规则是否一致 | 规则检查 |

### 量化指标

```python
# 世界观评分
WORLD_SCORE = {
    "设定可信度": 25,   # 不可信=0，可信=25
    "沉浸感": 25,       # 出戏=0，沉浸=25
    "细节丰富度": 25,   # 空洞=0，丰富=25
    "逻辑自洽": 25,     # 矛盾=0，自洽=25
}
WORLD_THRESHOLD = 75
```

- 反馈：无明显矛盾

通过（82≥75）
