## 阅读维度：节奏感知

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 张弛结合 | 紧张≥30%，缓冲≥20% | 场景分布 |
| 高潮间距 | 3-5章一个中等高潮 | 高潮统计 |
| 章节末钩子 | 吸引继续阅读 | 读者反馈 |
| 整体节奏 | 批次节奏是否流畅 | 整体评估 |

### 量化指标

```python
# 节奏感知评分
PACE_SCORE = {
    "张弛结合": 25,     # 失衡=0，均衡=25
    "高潮间距": 25,     # >6章=0，3-5章=25
    "章节末钩子": 25,   # 无吸引力=0，强=25
    "整体节奏": 25,     # 拖沓=0，流畅=25
}
PACE_THRESHOLD = 75
```

```markdown
- 评估：流畅
- 评分：18/25
- 反馈：无

### 结论
通过（80≥75）
建议：无严重问题
```
