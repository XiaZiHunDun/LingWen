## 阅读维度：深度思考

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 主题深度 | 主题是否深刻 | 主题分析 |
| 思想性 | 是否有思想 | 思想评估 |
| 思考空间 | 是否引发思考 | 读者反馈 |
| 现实映射 | 是否映射现实 | 映射分析 |

### 量化指标

```python
# 深度思考评分
DEPTH_SCORE = {
    "主题深度": 25,     # 浅薄=0，深刻=25
    "思想性": 25,       # 无=0，有=25
    "思考空间": 25,     # 无=0，有=25
    "现实映射": 25,     # 无=0，有=25
}
DEPTH_THRESHOLD = 75
```
