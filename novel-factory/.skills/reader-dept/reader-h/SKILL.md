## 阅读维度：主线追踪

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 主线识别 | 主线是否清晰 | 读者反馈 |
| 分支关联 | 分支是否服务主线 | 分析评估 |
| 进度感知 | 是否知道当前位置 | 读者反馈 |
| 方向明确 | 故事方向是否清晰 | 读者反馈 |

### 量化指标

```python
# 主线追踪评分
PLOT_SCORE = {
    "主线识别": 25,     # 模糊=0，清晰=25
    "分支关联": 25,     # 独立=0，服务主线=25
    "进度感知": 25,     # 迷茫=0，清晰=25
    "方向明确": 25,     # 迷失=0，明确=25
}
PLOT_THRESHOLD = 75
```
