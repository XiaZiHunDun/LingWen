## 阅读维度：逻辑通顺

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 情节逻辑 | 情节是否合理 | 逻辑分析 |
| 因果关联 | 因果是否清晰 | 因果分析 |
| 合理性 | 情节是否可信 | 合理性评估 |
| 矛盾检测 | 是否有矛盾 | 矛盾排查 |

### 量化指标

```python
# 逻辑通顺评分
LOGIC_SCORE = {
    "情节逻辑": 25,     # 混乱=0，合理=25
    "因果关联": 25,     # 无关=0，关联=25
    "合理性": 25,       # 不合理=0，合理=25
    "矛盾检测": 25,     # 有矛盾=0，无矛盾=25
}
LOGIC_THRESHOLD = 80
```
