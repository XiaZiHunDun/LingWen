## 阅读维度：悬念铺设

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 悬念强度 | 每章≥1个悬念 | 悬念点统计 |
| 钩子效果 | 章节末钩子吸引力 | 读者反馈 |
| 期待维持 | 追读意愿≥8/10 | 问卷调查 |
| 谜题设计 | 谜题清晰度/揭示节奏 | 谜题分析 |

### 量化指标

```python
# 悬念铺设评分
SUSPENSE_SCORE = {
    "悬念强度": 25,     # 0=0, ≥2=25
    "钩子效果": 25,     # 无吸引力=0，强=25
    "期待维持": 25,     # <6=0, ≥8=25
    "谜题设计": 25,     # 混乱=0，清晰=25
}
SUSPENSE_THRESHOLD = 75
```
