## 阅读维度：场景描写

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 画面感 | 画面是否清晰 | 读者反馈 |
| 场景构建 | 场景是否立体 | 场景分析 |
| 感官体验 | 感官描写是否丰富 | 感官统计 |
| 沉浸感 | 是否沉浸于场景 | 读者反馈 |

### 量化指标

```python
# 场景描写评分
SCENE_SCORE = {
    "画面感": 25,       # 模糊=0，清晰=25
    "场景构建": 25,     # 空洞=0，立体=25
    "感官体验": 25,     # 单一=0，丰富=25
    "沉浸感": 25,       # 出戏=0，沉浸=25
}
SCENE_THRESHOLD = 75
```
