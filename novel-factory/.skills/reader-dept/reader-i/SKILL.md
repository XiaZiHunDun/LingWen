## 阅读维度：文笔评价

### 评估标准

| 检查项 | 阈值 | 方法 |
|--------|------|------|
| 文字优美度 | 优美/一般/差 | 主观评价 |
| 表达精准 | 用词精准度 | 用词分析 |
| 风格统一 | 风格是否一致 | 风格检查 |
| 创新性 | 是否有创新 | 创新评估 |

### 量化指标

```python
# 文笔评价评分
PROSE_SCORE = {
    "文字优美度": 25,   # 差=0，优美=25
    "表达精准": 25,     # 不准=0，精准=25
    "风格统一": 25,     # 不统一=0，统一=25
    "创新性": 25,       # 陈旧=0，创新=25
}
PROSE_THRESHOLD = 75
```
