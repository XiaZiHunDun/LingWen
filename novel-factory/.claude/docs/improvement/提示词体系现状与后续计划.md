## 一、项目背景

灵文小说工厂各部门在调用LLM时，提示词普遍偏短（平均约400 tokens），无法有效利用LLM的深度能力。本次改进旨在建立各部门的提示词体系，提升至1000-3000 tokens范围。

### 2.3 自动化工具

#### 提示词自动生成器
**文件**：`.claude/scripts/prompt_generator.py`

```bash
# 列出所有模板
python3 .claude/scripts/prompt_generator.py list-templates

# 生成章节写作prompt
python3 .claude/scripts/prompt_generator.py chapter-writer --chapter 25

# 生成质量检查prompt
python3 .claude/scripts/prompt_generator.py quality-check --chapter 25
```

**已实现功能**：
- [x] 从 `基础层.yaml` 读取作品信息（名称、类型、风格、目标读者）
- [x] 从 `深度层.md` 提取世界观、势力数据
- [x] 自动获取前2章摘要作为前情
- [x] 自动判断章节类型（开篇/发展/高潮/收尾/过渡）
- [x] 自动判断卷/阶段
- [x] 生成完整prompt输出

#### 反馈记录工具
**文件**：`.claude/scripts/feedback_recorder.py`

```bash
# 记录反馈
python3 .claude/scripts/feedback_recorder.py record \
  --template chapter-writing-prompt --chapter 25 \
  --completeness 4 --executability 4 \
  --pros "..." --cons "..." --suggestions "..."

# 列出最近反馈
python3 .claude/scripts/feedback_recorder.py list

# 查看统计
python3 .claude/scripts/feedback_recorder.py stats
```

## 五、使用指南

### 5.1 快速开始

```bash
# 1. 生成章节写作prompt
python3 .claude/scripts/prompt_generator.py chapter-writer --chapter 25 -o /tmp/prompt.md

# 2. 使用prompt进行创作（复制到LLM调用中）

# 3. 记录反馈
python3 .claude/scripts/feedback_recorder.py record \
  --template chapter-writing-prompt --chapter 25 \
  --completeness 4 --executability 4 \
  --pros "..." --cons "..."
```
