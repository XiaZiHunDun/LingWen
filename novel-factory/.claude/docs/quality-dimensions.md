# 质量检查维度详解

## 10 维质量检查

| 维度 | 类型 | 说明 |
|------|------|------|
| 命名一致性 | 规则 | 文件名chXXX.md与章节内标题一致 |
| 内容完整性 | 规则 | **本章完**标记、字数>=500 |
| 章节重复 | 规则 | 跨章节相似度>80%预警 |
| 人物状态 | 规则 | 性别/生死/形态前后矛盾 |
| 时间线 | 规则 | "年前"与"瞬间"同时出现等错误 |
| 情节关联度 | 规则 | 每段落与前后章节的关联程度 |
| 伏笔回收率 | 规则 | 首次出现元素在后续N章内回收 |
| 场景逻辑 | 规则 | 场景转换合理性、孤岛章节检测 |
| 情感节奏 | 规则 | 情绪波动是否合理 |
| 对话风格 | 规则 | 角色对话字数异常检测 |
| 人物弧光 | **LLM** | 角色弧光完整性（通过Agent调用） |

## 情感审核专项（5维）

| 专项 | 检查内容 |
|------|---------|
| 情感弧线检查 | 主角/女主情感转折、婚后日常、情感锚点 |
| 配角价值验证 | 非战力价值、独立支线、道具情感承载 |
| 反派动机深度 | 反派前身、行动逻辑、验证者vs毁灭者 |
| 节奏健康度 | 阶段变速、核心冲突、信息密度 |
| 转折合理性 | 隐瞒必要性、理解铺垫、说服场景 |

## 质量等级

| 等级 | 条件 | 处理 |
|------|------|------|
| S级 | 5项全部通过 | 直接进入下一环节 |
| A级 | 4项通过 + 1项警告 | 修复后复查 |
| B级 | 3项通过 + 2项警告 或 有不通过 | 返回修改 |
| 不合格 | >=3项不通过 | 打回重做 |
