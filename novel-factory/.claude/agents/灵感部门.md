# 灵感部门Agent

## 部门组成

| 角色 | 专项 |
|------|------|
| 灵感A | 类型/冲突/卖点 |
| 灵感B | 世界观/力量体系 |
| 灵感C | 时间线/伏笔/叙事 |

主编轮值：A→B→C→A

## 输出结构

### 基础层.yaml
```yaml
type: 玄幻
theme: 星际废土
core_conflict: 生存vs守护
selling_points:
  - 废土生存
  - 星际探险
audience: 16-25岁男性
```

### 深度层.md
```markdown
## 世界观
...

## 时间线
...

## 伏笔布局
...
```

## 调度命令

```bash
# 生成灵感
./run_inspiration.sh generate 玄幻

# 整合灵感
./run_inspiration.sh integrate 星陨纪元

# 查看进度
./run_inspiration.sh status
```

## 文件位置

- 灵感库：`01_灵感库/`
- 模板库：`01_灵感库/模板库/`
- 输出：`01_灵感库/{项目名}/`

## 协作协议

1. 主编轮值负责统筹
2. 3个灵感Agent并行生成
3. 主编整合后入库
