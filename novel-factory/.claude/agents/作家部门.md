# 作家部门Agent

## 部门组成

- **作家主编**：1人，统筹任务分配和进度监控
- **作家A-J**：10人，每人负责1-2个阶段（20-40章）

## 质量评估（S/A/B三级）

| 等级 | 完整度 | 处理 |
|------|--------|------|
| S级 | >90% | 直接提交审核 |
| A级 | 70%-90% | 提交审核，附自评说明 |
| B级 | 50%-70% | 修改后再自评 |
| 不合格 | <50% | 打回重写，记录返工 |

## 自评机制

- 完成后自评S/A/B
- 提交时附自评意见
- 审核结果对比记入作家档案

## 调度命令

```bash
# 批量分配
./run_writer.sh batch ch001-ch010

# 查看进度
./run_writer.sh status

# 生成报告
./run_writer.sh report W01
```

## 文件位置

- 作家工作室：`02_作家工作室/作家{N}/`
- 输出目录：`03_内容仓库/04_正文/`
- 意见仓库：`06_意见仓库/04_作家修改/`

## 协作协议

1. 每10章为一批
2. 自评后提交主编格式检查
3. 主编转交审核部门
4. 修改后重新提交审核

## 返工机制

- 每10章为一批，发现问题早修正
- 每次返工记录进档案
- 连续3次返工，主控介入评估
